# Глава 6. Проценты и пропорции

> **Единый сборник задач по математике, 5-6 класс**

## Описание главы

Глава посвящена одной из ключевых тем курса 6-го класса — процентам и пропорциям. Учащиеся осваивают понятие процента, учатся решать три основных типа задач на проценты, изучают пропорции и их свойства, прямую и обратную пропорциональность, масштаб, деление числа в данном отношении.

## Структура

| Файл | Содержание |
