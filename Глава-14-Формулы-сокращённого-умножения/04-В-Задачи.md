## Разложение на множители комбинированными способами (14.101 — 14.112)

**14.101.** Разложите на множители:
а) $a^4 - 2a^3 + a^2 - 1$;
б) $b^4 - b^2 - 2b - 1$;
в) $c^8 - c^4 - 2c^2 - 1$.

**14.102.** Разложите на множители:
а) $a(a+2) - (b+1)(b-1)$;
б) $(a+3b)(a+b+2) - (a+b)(a+b+2)$.

**14.103.** Разложите на множители:
а) $x^3 - xy^2 - 6y^2 + 6x^2$;
б) $a^3 - 2a^2 + 18 - 9a$.

**14.104.** Разложите на множители:
а) $30x - 9x^2 - 70y + 49y^2$;
б) $20a^2 - 45b^2 + 30b - 5$.

**14.105.** Разложите на множители:
а) $x^4 + 4x^2 + 3$; *(подсказка: введите замену $t = x^2$)*
б) $x^4 + 8x^2 + 15$.

**14.106.** Разложите на множители:
а) $a^2 - 3ab + 2b^2$;
б) $a^2 + 4ab + 3b^2$.

**14.107.** Разложите на множители:
а) $2x^5 + 5x^4 - 2x^2 - 5x$;
б) $x^7 + 9x^6 - x^2 - 9x$.

**14.108.** Разложите на множители:
а) $3a - b + (a - b)^2$;
б) $(a + b)^2 + 2a + 2b$.

**14.109.** Разложите на множители:
а) $(a^2 + 4)^2 - 16a^2$;
б) $(4a + 1)^2(4a - 1)^2 - (16a^2 + 1)^2 + 64a^2$.

**14.110.** Разложите на множители:
а) $x^2 - 2xy + y - x + yz - xz$;
б) $a^3 - ab - a^2b + b^2$.

**14.111.** Разложите на множители:
а) $162b^3 - 2a^2b$;
б) $24y^5 - 3y^2$;
в) $3xy^2 - 147x^3$.

**14.112.** Разложите на множители:
а) $a^2 - x^2 - 6x - 9$;
б) $a^2 - x^2 + 4x - 4$.

## Доказательства тождеств (14.113 — 14.120)

**14.113.** Докажите тождество:
$$(a + b)^2 + (a - b)^2 = 2(a^2 + b^2).$$

**14.114.** Докажите тождество:
$$(a + b)^2 - (a - b)^2 = 4ab.$$

**14.115.** Докажите тождество:
$$(a + b + c)^2 = a^2 + b^2 + c^2 + 2ab + 2bc + 2ac.$$

**14.116.** Докажите тождество:
$$(a + b)^2 + (a + b - c)^2 + (a - b + c)^2 + (b - c)^2 = (2a + b - c)^2 + b^2.$$

*(Подсказка: раскройте левую часть и упростите.)*

**14.117.** Докажите тождество:
$$x^5 + 1 = (x + 1)(x^4 - x^3 + x^2 - x + 1).$$

**14.118.** Докажите тождество:
$$(a - b)^2 + (b - c)^2 + (c - a)^2 = (a + b - 2c)^2 + (b + c - 2a)^2 + (c + a - 2b)^2.$$

**14.119.** Докажите, что при $a + b + c = 0$ верно равенство:
$$a^3 + b^3 + c^3 = 3abc.$$

**14.120.** Докажите тождество:
$$(a^2 + 4)^2 - 16a^2 = (a+2)^2(a-2)^2.$$

## Задачи на делимость (14.121 — 14.127)

**14.121.** Докажите, что при любом натуральном $n$ значение выражения $n^3 - n$ кратно $6$.

**14.122.** Докажите, что при любом натуральном $n$ значение выражения $(n^2 + n)(n + 2)$ кратно $3$.

**14.123.** Докажите, что при любом натуральном $n$ значение выражения $8n^3 - 2n$ кратно $3$.

**14.124.** Докажите, что при любом натуральном $n$ выражение $5^n - 1$ кратно $4$.

*(Подсказка: $5^n - 1 = (5-1)(5^{n-1} + 5^{n-2} + \ldots + 1)$ или рассмотрите $5 = 4 + 1$ и примените формулу для $a^n - b^n$.)*

**14.125.** Докажите, что выражение $8^5 + 2^{13}$ делится на $10$.

**14.126.** Докажите, что при целом $n$ выражение $n^2 - n$ чётно.

**14.127.** Докажите, что при любом натуральном $n$ выражение $10^n - 1$ кратно $9$.

## Нахождение наибольшего и наименьшего значения (14.128 — 14.132)

**14.128.** Найдите наименьшее значение выражения $x^2 - 10x + 29$. При каком значении $x$ оно достигается?

*(Подсказка: запишите в виде $(x - 5)^2 + \ldots$)*

**14.129.** Найдите наибольшее значение выражения $-y^2 + 4y - 5$. При каком значении $y$ оно достигается?

**14.130.** Найдите наименьшее значение выражения $2a^2 - 2ab + b^2 - 2a + 2$.

**14.131.** Найдите наибольшее значение выражения $4b(5a - b) - (5a - 2)(5a + 2)$.

**14.132.** Докажите, что при любых значениях $x$ и $y$ выражение $4x^2 - 20xy + 25y^2$ неотрицательно. При каких $x$ и $y$ оно равно нулю?

## Олимпиадные задачи (14.133 — 14.135)

**14.133.** Известно, что $a + b + c = 12$ и $ab + bc + ca = -15$. Найдите $a^2 + b^2 + c^2$.

**14.134.** Упростите выражение:
$$(a + b)(a^2 + b^2)(a^4 + b^4)(a^8 + b^8)(a^{16} + b^{16})(a^{32} + b^{32})$$
при условии, что $a = b + 1$.

*(Подсказка: умножьте на $(a - b)$ и применяйте разность квадратов последовательно.)*

**14.135.** Докажите, что из равенства $x^2 + y^2 + z^2 = xy + yz + zx$ следует $x = y = z$.

*(Подсказка: умножьте обе части на $2$ и перепишите левую часть как сумму трёх квадратов разностей.)*
