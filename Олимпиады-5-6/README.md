## Об этом разделе

Собраны задачи из ведущих международных математических соревнований, адаптированные для учащихся 5–6 классов:

- **AMC 8** (American Mathematics Competition 8) — США, для учащихся 6–8 классов
- **Math Kangaroo** (Кенгуру) — международное соревнование, группы 5–6 класс

Задачи отсортированы по уровню сложности внутри каждого соревнования.
