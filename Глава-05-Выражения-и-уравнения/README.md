# Глава 5. Выражения и уравнения

## Обзор

Пятая глава единого сборника задач по математике для 5-6 классов посвящена выражениям и уравнениям — фундаменту алгебры. Учащиеся осваивают переход от арифметики к работе с буквенными выражениями, учатся упрощать, преобразовывать и решать уравнения, а также составлять математические модели по условию текстовых задач.

- **Диапазон:** 5.001 – 5.150
- **Всего задач:** 150

| Группа | Задачи | Кол-во | Уровень |
|--------|--------|--------|---------|

## Тематическое содержание

1. **Числовые выражения** — порядок действий, вычисление значений
2. **Буквенные выражения** — подстановка значений переменных
3. **Свойства действий** — переместительное, сочетательное, распределительное
4. **Упрощение выражений** — раскрытие скобок, приведение подобных слагаемых
5. **Уравнения** — нахождение неизвестных компонентов, составные уравнения
6. **Задачи на составление уравнений** — движение, возраст, покупки
7. **Формулы** — пути, площади, периметра
8. **Уравнения с модулем** — для 6 класса
9. **Системы уравнений** — пропедевтика, решение подбором
10. **Олимпиадные задачи** — функциональные уравнения, закономерности

## Рекомендации по использованию

- **5 класс:** Группа А полностью, Группа Б — задачи 5.061–5.090
- **6 класс:** Группа А для повторения, Группы Б и В полностью
- **Олимпиадная подготовка:** Группа В (задачи 5.131–5.150)
