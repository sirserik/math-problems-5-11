## Как пользоваться сборником

1. **Прочитай теорию** — файл `00-Теория.md` содержит все определения и свойства.
2. **Разбери примеры** — в файле `01-Примеры.md` показано, как оформлять решения.
3. **Начни с группы А** — убедись, что базовые навыки освоены.
4. **Переходи к группе Б** — здесь нужны рациональные приёмы и смекалка.
5. **Группа В** — для тех, кто готовится к олимпиадам.
