# Последовательности, факториалы, математическая индукция (10.379–10.395)

> **Класс:** 10–11 | **Темы:** монотонность, сокращение дробей, индукция, делимость

---

## Последовательности (10.379–10.380)

**32.379.** Доказать, что последовательность, заданная формулой $y_n = \dfrac{10n + 7}{2n}$, убывает.

**32.380.** Установить, является ли заданная последовательность убывающей или возрастающей:

а) $x_n = 3n^2 - n$;
б) $x_n = n^2 - 3n$;
в) $x_n = 7n - n^2$;
г) $y_n = \lg\left(\dfrac{3}{4}\right)^n.$

## Сократить дроби (10.381–10.384)

**32.381.** $\dfrac{n!}{(n + 1)! - n!}.$

**32.382.** $\dfrac{(n + 2)! + (n + 1)!}{(n + 2)! - (n + 1)!}.$

**32.383.** $\dfrac{(n + 2)\,n!}{(n + 1)!}.$

**32.384.** $\dfrac{((n + 2)! + n!)\,(n + 1)}{(n + 2)!\,(n^2 + 3n + 3)}.$

## Доказать неравенство (10.385–10.386)

**32.385.** Доказать, что для всякого $n \in \mathbb{N}$ выполняется неравенство $\dfrac{1}{1!} + \dfrac{1}{2!} + \dfrac{1}{3!} + \ldots + \dfrac{1}{n!} < 2.$

**32.386.** Как использовать тождество $\dfrac{1}{k-1} - \dfrac{1}{k} = \dfrac{1}{k(k-1)}$ для доказательства справедливости неравенства $\dfrac{1}{2^2} + \dfrac{1}{3^2} + \dfrac{1}{4^2} + \ldots + \dfrac{1}{n^2} < \dfrac{n - 1}{n}$?

## Делимость (10.387–10.389)

**32.387.** Доказать, что если $p$ и $q$ — простые числа, бо́льшие 3, то $p^2 - q^2$ делится на 24.

**32.388.** Доказать, что если $n$ — простое число, бо́льшее 3, то $\dfrac{n^2 - 1}{24}$ — целое число.

**32.389.** Доказать, что $n^7 - n$ делится на 42, если $n$ — любое целое число.

## Суммы и формулы (доказать методом математической индукции) (10.390–10.395)

**32.390.** Доказать, что сумма кубов $n$ нечётных чисел равна $n^2(2n^2 - 1)$ при любом натуральном значении $n$.

**32.391.** Доказать, что $1 + 3 + 6 + \ldots + \dfrac{n(n+1)}{2} = \dfrac{n(n+1)(n+2)}{6}.$

**32.392.** Доказать, что $1^2 + 2^2 + 3^2 + \ldots + n^2 = \dfrac{n(n+1)(2n+1)}{6}.$

**32.393.** Доказать, что неравенство $3^n > n + 2$ верно, если $n$ — натуральное число.

**32.394.** Используя метод математической индукции, доказать справедливость неравенства $(1 + a)^n \geqslant 1 + na$ ($n$ — натуральное число, $n \geqslant 2$ и $a > -1$).

**32.395.** Доказать, что $|\alpha_1 + \alpha_2| \leqslant |\alpha_1| + |\alpha_2|$. На основании этого методом математической индукции доказать, что $|\alpha_1 + \alpha_2 + \ldots + \alpha_n| \leqslant |\alpha_1| + |\alpha_2| + \ldots + |\alpha_n|.$
