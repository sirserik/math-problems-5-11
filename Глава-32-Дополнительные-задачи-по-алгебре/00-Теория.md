## Обзор тематики

Глава содержит задачи, объединяющие несколько тем школьной алгебры. Они требуют комплексного подхода и умения комбинировать методы из разных разделов.

| Тема | Типы задач |

## Ключевые методы

### 1. Модуль — раскрытие по определению
$$|a| = \begin{cases} a, & a \geqslant 0, \\ -a, & a < 0. \end{cases}$$

Для уравнений/неравенств с модулем: разбить на случаи по знаку подмодульного выражения.

### 2. Графики с модулем
- $y = |f(x)|$ — отразить отрицательную часть графика $f(x)$ вверх
- $y = f(|x|)$ — оставить правую часть ($x \geqslant 0$) и отразить её влево

### 3. Исследование функций без производной
- **Чётность/нечётность**: $f(-x) = f(x)$ или $f(-x) = -f(x)$
- **Область значений**: через дискриминант (для рациональных функций)
- **Монотонность**: через разность $f(x_1) - f(x_2)$

### 4. Математическая индукция
1. **База**: проверить для $n = 1$ (или начального значения)
2. **Шаг**: предположить верность для $n = k$, доказать для $n = k + 1$

## Типичные ошибки

| Ошибка | Правильно |
|--------|-----------|
| Забыть проверку ОДЗ в уравнениях с модулем | Всегда проверять допустимость корней |
| Неправильно строить $y = f(\|x\|)$ | Сначала построить $f(x)$ при $x \geqslant 0$, затем отразить |
| В индукции — забыть базу | База обязательна, без неё доказательство неполно |
| Путать $\lg x^2$ и $(\lg x)^2$ | $\lg x^2 = 2\lg|x|$, $(\lg x)^2 = \lg^2 x$ |
