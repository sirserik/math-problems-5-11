## Типы задач

- **Модуль** — уравнения, неравенства, графики с $|f(x)|$ и $f(|x|)$
- **Графики** — построение, исследование, пересечения, области
- **Уравнения с параметрами** — иррациональные, показательные, логарифмические
- **Неравенства** — алгебраические, логарифмические, показательные
- **Тригонометрия** — вычисления, тождества, доказательства, треугольник
- **Логарифмы** — вычисления, знаки, свойства, иррациональность
- **Прогрессии** — AP, GP, суммы, монотонность
- **Теория чисел** — делимость, представления, иррациональность
- **Математическая индукция** — суммы, неравенства, делимость
- **Доказательства** — неравенства средних, свойства функций
