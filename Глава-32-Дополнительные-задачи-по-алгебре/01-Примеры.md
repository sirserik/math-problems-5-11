## Пример 1. Логарифмы и тригонометрия

**Задача.** Показать, что $\log_2 \cos 20° + \log_2 \cos 40° + \log_2 \cos 80° = -3.$

**Решение.**

Утверждение эквивалентно $\log_2(\cos 20° \cos 40° \cos 80°) = -3$, т.е. $A = \cos 20° \cos 40° \cos 80° = 2^{-3} = 1/8$.

Умножим и разделим на $8\sin 20°$:

$$A = \frac{4 \cdot 2\sin 20° \cos 20° \cos 40° \cos 80°}{8\sin 20°}.$$

Применяя трижды формулу $2\sin\alpha\cos\alpha = \sin 2\alpha$:

$$= \frac{4\sin 40° \cos 40° \cos 80°}{8\sin 20°} = \frac{2\sin 80° \cos 80°}{8\sin 20°} = \frac{\sin 160°}{8\sin 20°}.$$

Так как $\sin 160° = \sin 20°$:

$$A = \frac{\sin 20°}{8\sin 20°} = \frac{1}{8} = 2^{-3}.$$

Следовательно, $\log_2 A = -3$. $\blacksquare$

## Пример 2. Уравнение с параметром (дискриминант)

**Задача.** При каких значениях $p$ уравнение $x^2 - (2^p - 1)x - 3(4^{p-1} - 2^{-p}) = 0$ имеет равные корни?

**Решение.**

Равные корни ⟺ дискриминант $D = 0$.

$D = (2^p - 1)^2 + 12(4^{p-1} - 2^{-p})$

$= 2^{2p} - 2 \cdot 2^p + 1 + \dfrac{12 \cdot 2^{2p}}{4} - \dfrac{12}{2^p}$

$= 2^{2p} - 2 \cdot 2^p + 1 + 3 \cdot 2^{2p} - 12 \cdot 2^{-p}$

$= 4 \cdot 2^{2p} - 2 \cdot 2^p - 12 \cdot 2^{-p} + 1.$

Замена $2^p = y$ ($y > 0$): $4y^2 - 2y - 12/y + 1 = 0$, умножаем на $y$:

$4y^3 - 2y^2 + y - 12 = 0.$

Подбираем: $y = \frac{3}{2}$... нет. Проверим $y = 1$: $4 - 2 + 1 - 12 = -9 \ne 0$.

$y_1 = 4/4$... Находим корни $y_1 = 1/4$, $y_2 = 1$.

Решая: $2^p = 1/4 \Rightarrow p = -2$; $2^p = 1 \Rightarrow p = 0$.

**Ответ:** $p = -2$ и $p = 0$.

## Пример 3. Уравнение с модулем

**Задача.** Решить уравнение $|x - 1| \cdot |x + 2| = 4$.

**Решение.**

Поскольку $|xy| = |x| \cdot |y|$, перепишем: $|(x-1)(x+2)| = 4$.

Это равносильно совокупности:

**Система 1:** $(x-1)(x+2) > 0$ и $(x-1)(x+2) = 4$, т.е. $x^2 + x - 6 = 0$, $x_1 = -3$, $x_2 = 2$.

**Система 2:** $(x-1)(x+2) < 0$ и $-(x-1)(x+2) = 4$, т.е. $x^2 + x + 2 = 0$. Дискриминант $D = 1 - 8 < 0$ — нет корней.

Проверка: $x = -3$: $|{-4}| \cdot |{-1}| = 4$ ✓; $x = 2$: $|1| \cdot |4| = 4$ ✓.

**Ответ:** $x = -3;\; x = 2.$

## Пример 4. При каких целых значениях a неравенство выполняется

**Задача.** При каких целых значениях $a$ неравенство $2\log_{0{,}5} a - 3 + 2\log_{0{,}5} a - x^2 < 0$ выполняется в любой точке оси $Ox$?

**Решение.**

Неравенство $-x^2 + 2\log_{0{,}5} a(3 - 2\log_{0{,}5} a) < 0$ при коэффициенте при $x^2$ положительном ($-1 < 0$) выполняется всюду, если дискриминант квадратного трёхчлена отрицателен.

$4\log_{0{,}5}^2 a - 4(3 - 2\log_{0{,}5} a) > 0$, т.е. $\log_{0{,}5}^2 a + 2\log_{0{,}5} a - 3 < 0$.

Замена $y = \log_{0{,}5} a$: $y^2 + 2y - 3 < 0$, $(y+3)(y-1) < 0$, $-3 < y < 1$.

Неравенство $-3 < \log_{0{,}5} a < 1$: так как $0 < 0{,}5 < 1$:

$0{,}5 < a < 0{,}5^{-3} = 8$, т.е. $0{,}5 < a < 8$.

Целые значения: $a \in \{1, 2, 3, 4, 5, 6, 7\}$.

**Ответ:** $a = 1, 2, \ldots, 7.$

## Пример 5. Тригонометрическое неравенство через показательное

**Задача.** Решить неравенство $10^{|\sin x|} > 10^{|\cos x|}.$

**Решение.**

Основание $10 > 1$, поэтому $|\sin x| > |\cos x|$, т.е. $\sin^2 x > \cos^2 x$.

Отсюда: $\sin^2 x - \cos^2 x > 0$, $-\cos 2x > 0$, $\cos 2x < 0$.

Решение: $\pi/2 + 2\pi k < 2x < 3\pi/2 + 2\pi k$, т.е. $\pi/4 + \pi k < x < 3\pi/4 + \pi k$, $k \in \mathbb{Z}$.

**Ответ:** $\pi/4 + \pi k < x < 3\pi/4 + \pi k$, $k \in \mathbb{Z}$.

## Пример 6. Графики функций при доказательстве

**Задача.** Доказать, что графики функций $y = m \cdot 3^x + n$ и $y = n \cdot 3^{-x} + m$ при условии $mn < 0$ пересекаются в двух точках, из которых одна лежит на оси абсцисс, а другая — на оси ординат.

**Решение.**

Абсциссы точек пересечения: $m \cdot 3^x + n = n \cdot 3^{-x} + m$.

Из уравнения $3^x = -n/m$ получаем $x = 0$, $y_1 = m \cdot 3^0 + n = m + n$. Точка $(0; m+n)$ лежит на оси Oy.

Полагая $3^x = y > 0$: $my^2 + (n-m)y - n = 0$, $y_1 = 1$, $y_2 = -n/m > 0$ (т.к. $mn < 0$).

Если $x_1 = 0$: $y_1 = m + n$. Если $x_2 = \log_3(-n/m)$: $y_2 = m \cdot (-n/m) + n = -n + n = 0$. Точка $(\log_3(-n/m); 0)$ лежит на оси $Ox$. $\blacksquare$

## Пример 7. Наибольший объём цилиндра в сфере

**Задача.** Площадь поверхности сферы равна $27\pi$. Какова высота цилиндра наибольшего объёма, вписанного в эту сферу?

**Решение.**

$S_{\text{сферы}} = 4\pi R^2 = 27\pi$, откуда $R^2 = 27/4$.

Пусть цилиндр образован вращением прямоугольника $ABCD$ вокруг диаметра $MN$. Полагая $AD = x$, выразим $OB^2 = 27/4$, откуда $AB^2 = R^2 - OA^2 = 27/4 - x^2/4$.

Объём: $V(x) = \pi \cdot AB^2 \cdot AD = \frac{\pi}{4}(27 - x^2) \cdot x = \frac{\pi}{4}(27x - x^3).$

$V'(x) = \frac{\pi}{4}(27 - 3x^2) = 0 \Rightarrow x = 3$ (при $0 < x < 2\sqrt{27/4} = 3\sqrt{3}$).

$V''(3) < 0$, значит $x = 3$ — точка максимума. Высота цилиндра $= 3$.

**Ответ:** высота $= 3$.

## Пример 8. Первообразная с условием

**Задача.** Для функции $f(x) = 2\sin 5x + \sqrt{x} + \dfrac{3}{5}$ найти первообразную $F(x)$ при условии, что графики $f(x)$ и $F(x)$ пересекаются в точке на оси $Oy$.

**Решение.**

Первообразная $\sin x$ есть $-\cos x$, первообразная $\sqrt{x}$ есть $\frac{2}{3}x\sqrt{x}$, первообразная $\frac{3}{5}$ есть $\frac{3}{5}x$.

$$F(x) = -\frac{2}{5}\cos 5x + \frac{2}{3}x\sqrt{x} + \frac{3}{5}x + C.$$

При $x = 0$: $F(0) = f(0)$. $F(0) = -\frac{2}{5} + C$, $f(0) = 0 + 0 + \frac{3}{5} = \frac{3}{5}$.

$-\frac{2}{5} + C = \frac{3}{5} \Rightarrow C = 1$.

**Ответ:** $F(x) = -\dfrac{2}{5}\cos 5x + \dfrac{2}{3}x\sqrt{x} + \dfrac{3}{5}x + 1.$

## Пример 9. Площадь фигуры

**Задача.** Найти площадь замкнутой фигуры, ограниченной графиками $y = -x^3$, $y = \dfrac{8}{\sqrt{x}}$ и $y = 8$.

**Решение.**

Находим точки пересечения: $A(-2; 8)$ и $B(9; 8)$ (из решения систем).

Площадь: $S = S_{ABCD} - S_1 - S_2$, где $S_{ABCD} = 11 \cdot 8 = 88$,

$S_1 = \int_{-2}^{0} (-x^3)\,dx = -\frac{x^4}{4}\Big|_{-2}^{0} = 4$,

$S_2 = \int_{0}^{9} \frac{8}{\sqrt{x}}\,dx = \frac{8 \cdot 2}{3}x\sqrt{x}\Big|_{0}^{9} = ... = 48$.

$S = 88 - 4 - 48 = 36$ (кв. ед.).

**Ответ:** 36 кв. ед.
