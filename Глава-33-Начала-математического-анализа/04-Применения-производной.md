# Применения производной (33.060–33.082)

> **Класс:** 10–11 | **Темы:** вторая производная, знак производной, область значений, монотонность, доказательства

---

## Вторая производная и исследование (33.060–33.066)

**33.060.** Найти вторую производную функции $f(x)$ и вычислить её значение при указанном значении $x$:

а) $f(x) = x^2\ln x + \cos 2x$; $\;f''(1) = ?$ $\;f''(\pi) = ?$

б) $f(x) = \sin\dfrac{x}{3} + x\ln x^2$; $\;f''(3) = ?$ $\;f''(\pi/2) = ?$

**33.061.** Выяснить знак производной функции $y = \sqrt{4x + 9}\,(x^2 - 16)$ в точке $x = 0$.

**33.062.** Дана функция $f(x) = \sqrt[4]{x^3}$. Как изменяется её производная с возрастанием $x$ от $1/16$ до $81$?

**33.063.** Дана функция $f(x) = 2\cos^2(4x - 1)$. Найти область значений $f'(x)$.

**33.064.** Доказать, что функция $f(x) = \dfrac{x + 3}{x - 5}$ убывает на всех интервалах области определения.

**33.065.** Дана функция $f(x) = x\ln x - x$. Как изменяется её производная с возрастанием $x$ от 1 до 9?

**33.066.** Найти область определения функции $f(x) = \sqrt{4 + 3x - x^2}$ и область определения её производной.

## Исследование функций (33.067–33.082)

**33.067.** Дано: $f(x) = \dfrac{1}{2}\sqrt{x^3 + 1}$ и $g(x) = xe^{-x}$. Показать, что $f'(2)$ является корнем уравнения $g'(x) = 0$.

**33.068.** Функция задана формулой $f(x) = e^{ax^2 + bx + 1}$. Найти значения постоянных $a$ и $b$, если $f(1) = f'(0)$.

**33.069.** Функция задана формулой $f(x) = 5\sin x + 3\cos x$. Решить уравнение $f'(0) = f'(x)$.

**33.070.** Функция задана формулой $f(x) = e^{-x}(x^2 + 3x + 1)$. Решить уравнение $f'(x) = 2f(x)$.

**33.071.** Построить раздельно графики функций $f(x) = x$, $\varphi(x) = |x|$ и $g(x) = x\,|x|$ в окрестности точки $x = 0$. Несмотря на то что $f(x)$ дифференцируема при $x = 0$, а $\varphi(x)$ — нет, их произведение $g(x) = x\,|x|$ имеет производную в точке $x = 0$. Обосновать правильность этих утверждений и найти $g'(0)$.

**33.072.** Доказать, что функция $f(x) = x + \sin x$ не убывает в каждой точке оси $Ox$.

**33.073.** Показать, что при любых значениях постоянных $p$ и $q$ ($p \ne q$) функция, заданная формулой

$$f(x) = \begin{cases} p\cos x + q\sin x & \text{при } x \geqslant 0, \\ px + q + 1 & \text{при } x < 0, \end{cases}$$

не дифференцируема в точке $x = 0$.

**33.074.** Дана функция $f(x) = 0{,}5\,(x^2 - \cos x)$. Пользуясь соображениями непрерывности, выяснить, имеют ли уравнения $f(x) = 7{,}8$ и $f'(x) = 7{,}8$ хотя бы по одному корню на промежутке $[2\pi, 3\pi]$.

**33.075.** Найти все значения постоянной $a$, при которых производная функции, заданной формулой $y = e^{ax^3 + 3x^2 + x}$, принимает только положительные значения на всей области определения данной функции.

**33.076.** Найти сумму $x + x^2 + \ldots + x^n$, а затем сумму $1 + 2x + 3x^2 + \ldots + nx^{n-1}$.

**33.077.** Функция задана формулой $f(x) = \dfrac{\sin x}{\sin\left(x + \dfrac{\pi}{3}\right)}$. Показать, что эта функция возрастает в любой точке, принадлежащей её области определения.

**33.078.** Функция задана формулой $y = \sqrt{ax^3 - 6x^2 + 3x}$. Найти все значения постоянной величины $a$, при которых данная функция определена и монотонно возрастает при всех $x > 0$.

**33.079.** Показать, что функция $y = x^3 + 4x$ возрастает на всей числовой оси.

**33.080.** При каком значении $p$ функция $f(x) = \cos x - px + q$ убывает на всей числовой оси?

**33.081.** Доказать, что функция $y = 2x + \sin x$ возрастает на всей числовой оси.

**33.082.** Доказать, что функция $y = x + \dfrac{1}{1 + x^2}$ возрастает на всей числовой оси.
