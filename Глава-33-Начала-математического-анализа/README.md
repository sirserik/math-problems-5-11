## Типы задач

- **Пределы** — вычисление, неопределённости $0/0$, сравнение пределов
- **Производные** — таблица, правила, цепное правило, значения в точках
- **Уравнения/неравенства с производными** — решение через $f'(x)$
- **Монотонность** — промежутки возрастания/убывания, сравнение значений
- **Экстремумы на отрезках** — наибольшее и наименьшее значения
- **Оптимизация** — геометрические и прикладные задачи на max/min
- **Касательные** — уравнение, угол наклона, точки касания, площади
- **Исследование функций** — экстремумы, промежутки, эскизы графиков
- **Кинематика** — скорость, ускорение, прямолинейное движение
- **Первообразные** — нахождение $F(x)$ по $f(x)$ с начальными условиями
- **Интегралы** — формула Ньютона—Лейбница, тригонометрические
- **Площади** — криволинейные трапеции, фигуры между кривыми
- **Дифференциальные уравнения** — гармонические колебания
