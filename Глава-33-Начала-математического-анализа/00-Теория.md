# Теория: Начала математического анализа

> **Класс:** 10–11 | **Темы:** пределы, производные, экстремумы, первообразные, интегралы, площади

---

## 1. Предел функции

**Предел функции** $f(x)$ при $x \to a$ — это число $L$, к которому стремится $f(x)$, когда $x$ стремится к $a$:

$$\lim_{x \to a} f(x) = L.$$

### Основные приёмы вычисления пределов

| Неопределённость | Приём |
|---|---|
| $\dfrac{0}{0}$ (многочлены) | Разложить на множители, сократить |
| $\dfrac{0}{0}$ (корни) | Умножить на сопряжённое |
| $\dfrac{\infty}{\infty}$ | Разделить на старшую степень |

> **Как запомнить:** неопределённость $\frac{0}{0}$ — сигнал, что числитель и знаменатель имеют общий множитель $(x - a)$, который нужно найти и сократить.

## 2. Производная

**Производная** функции $f(x)$ — это скорость изменения функции:

$$f'(x) = \lim_{\Delta x \to 0} \frac{f(x + \Delta x) - f(x)}{\Delta x}.$$

### Таблица производных

| Функция $f(x)$ | Производная $f'(x)$ |
|---|---|
| $c$ (const) | $0$ |
| $x^n$ | $nx^{n-1}$ |
| $\sqrt{x}$ | $\dfrac{1}{2\sqrt{x}}$ |
| $\dfrac{1}{x}$ | $-\dfrac{1}{x^2}$ |
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $\operatorname{tg} x$ | $\dfrac{1}{\cos^2 x}$ |
| $\operatorname{ctg} x$ | $-\dfrac{1}{\sin^2 x}$ |
| $e^x$ | $e^x$ |
| $a^x$ | $a^x \ln a$ |
| $\ln x$ | $\dfrac{1}{x}$ |
| $\log_a x$ | $\dfrac{1}{x \ln a}$ |
| $\arcsin x$ | $\dfrac{1}{\sqrt{1 - x^2}}$ |
| $\arccos x$ | $-\dfrac{1}{\sqrt{1 - x^2}}$ |
| $\operatorname{arctg} x$ | $\dfrac{1}{1 + x^2}$ |

### Правила дифференцирования

1. **Линейность:** $(af + bg)' = af' + bg'$
2. **Произведение:** $(fg)' = f'g + fg'$
3. **Частное:** $\left(\dfrac{f}{g}\right)' = \dfrac{f'g - fg'}{g^2}$
4. **Цепное правило (сложная функция):** $(g(f(x)))' = g'(f(x)) \cdot f'(x)$

> **Как запомнить правило произведения:** «производная первого на второе плюс первое на производную второго».

> **Цепное правило:** «производная внешней на производную внутренней».

## 3. Касательная к графику

Уравнение касательной к графику $y = f(x)$ в точке $(x_0, y_0)$:

$$y - y_0 = f'(x_0)(x - x_0),$$

где $(x_0, y_0)$ — точка касания, $f'(x_0)$ — угловой коэффициент касательной.

> **Геометрический смысл производной:** $f'(x_0)$ — тангенс угла наклона касательной к оси $Ox$.

---

## 4. Исследование функции с помощью производной

### Монотонность
- $f'(x) > 0$ на $(a, b)$ $\Rightarrow$ $f$ **возрастает** на $(a, b)$
- $f'(x) < 0$ на $(a, b)$ $\Rightarrow$ $f$ **убывает** на $(a, b)$

### Экстремумы
- Если $f'(x)$ меняет знак с $+$ на $-$ в точке $x_0$ — **максимум**
- Если $f'(x)$ меняет знак с $-$ на $+$ в точке $x_0$ — **минимум**

### Наибольшее и наименьшее на отрезке $[a, b]$

1. Найти $f'(x) = 0$ — критические точки на $[a, b]$
2. Вычислить $f$ в критических точках и на концах
3. Выбрать наибольшее и наименьшее значения

> **Важно:** наибольшее/наименьшее значение может достигаться на концах отрезка!

## 5. Первообразная

**Первообразная** функции $f(x)$ — это функция $F(x)$ такая, что $F'(x) = f(x)$.

### Правила нахождения первообразных

1. Если $F$ — первообразная для $f$, а $G$ — для $g$, то $F + G$ — первообразная для $f + g$
2. Если $F$ — первообразная для $f$, а $k$ — постоянная, то $kF$ — первообразная для $kf$
3. Если $F(x)$ — первообразная для $f(x)$, а $k \ne 0$ и $b$ — постоянные, то $\dfrac{1}{k}F(kx + b)$ — первообразная для $f(kx + b)$

### Таблица первообразных

| Функция $f(x)$ | Первообразная $F(x)$ |
|---|---|
| $x^n$ ($n \ne -1$) | $\dfrac{x^{n+1}}{n+1}$ |
| $\dfrac{1}{x}$ | $\ln|x|$ |
| $e^x$ | $e^x$ |
| $a^x$ | $\dfrac{a^x}{\ln a}$ |
| $\sin x$ | $-\cos x$ |
| $\cos x$ | $\sin x$ |
| $\dfrac{1}{\cos^2 x}$ | $\operatorname{tg} x$ |
| $\dfrac{1}{\sin^2 x}$ | $-\operatorname{ctg} x$ |

## 6. Определённый интеграл

### Формула Ньютона—Лейбница

$$\int_a^b f(x)\,dx = F(b) - F(a),$$

где $F$ — любая первообразная функции $f$.

### Площадь криволинейной трапеции

Площадь фигуры, ограниченной графиком неотрицательной функции $y = f(x)$, осью $Ox$, прямыми $x = a$ и $x = b$:

$$S = \int_a^b f(x)\,dx.$$

Если $f(x)$ меняет знак, площадь вычисляется как сумма модулей интегралов по промежуткам знакопостоянства.

> **Как запомнить:** интеграл «считает площадь со знаком» — над осью $Ox$ положительно, под осью — отрицательно.
