**33.265.** $S = \int_{-\pi/4}^{\pi/4} \cos x\,dx = 2\sin\dfrac{\pi}{4} = \sqrt{2}$

**33.266.** $S = \int_0^4 \sqrt{x}\,dx + \int_4^9 (2 - \sqrt{x})...$; зависит от области

**33.267.** $S = \int_0^1 (\sqrt{x} - x^3)\,dx = \dfrac{2}{3} - \dfrac{1}{4} = \dfrac{5}{12}$

**33.268.** $S = \int_{1/2}^{3/2}\left(2x - x^2 - \dfrac{3}{4}\right)\,dx = \dfrac{1}{12}$; пересечение: $2x - x^2 = 3/4$; $x^2 - 2x + 3/4 = 0$; $x = \dfrac{1}{2}$ и $x = \dfrac{3}{2}$; $S = \dfrac{1}{3}$

**33.269.** $S = \int_0^1 (x - x^4)\,dx = \dfrac{1}{2} - \dfrac{1}{5} = \dfrac{3}{10}$

**33.270.** $S = \int_{0.5}^{2.5} \dfrac{1}{x^2}\,dx = -\dfrac{1}{x}\Big|_{0.5}^{2.5} = -\dfrac{2}{5} + 2 = \dfrac{8}{5}$

**33.271.** Пересечение $5/x = 6 - x$: $x^2 - 6x + 5 = 0$; $x = 1$ и $x = 5$; $S = \int_1^5 \left(6 - x - \dfrac{5}{x}\right)\,dx = [6x - x^2/2 - 5\ln x]_1^5 = (30 - 12{,}5 - 5\ln 5) - (6 - 0{,}5 - 0) = 12 - 5\ln 5$

**33.272.** $y = 0$ при $x = 0$ и $x = 2$; $S = \int_0^2 |x^3 - 4x|\,dx = \int_0^2 (4x - x^3)\,dx = [2x^2 - x^4/4]_0^2 = 8 - 4 = 4$

**33.273.** Касательная при $x = 2$: $y'(2) = 8 - 2 = 6$; $y(2) = 5$; $y = 6x - 7$; $S$ определяется интегрированием

**33.274.** а) $y'' = -4y$; б) $y'' = -0{,}36y$

**33.275.** а) $y' = -36y$: $y = Ce^{-36x}$; два решения: $y = e^{-36x}$ и $y = -e^{-36x}$; б) $y'' = -36y$: $y = A\sin 6x + B\cos 6x$; два решения: $y = \sin 6x$ и $y = \cos 6x$
