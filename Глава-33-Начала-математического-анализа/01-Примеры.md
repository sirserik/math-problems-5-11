## Пример 1. Вычисление предела

**Задача.** Найти $\displaystyle\lim_{x \to 2} \frac{x^3 - 8}{2x - 4}.$

**Решение.**

Функция $f(x) = \dfrac{x^3 - 8}{2x - 4}$ не определена при $x = 2$. Разложим числитель на множители по формуле разности кубов, а знаменатель вынесем 2:

$$f(x) = \frac{(x - 2)(x^2 + 2x + 4)}{2(x - 2)}.$$

В области определения ($x \ne 2$) можно сократить на $(x - 2)$:

$$\lim_{x \to 2} \frac{x^3 - 8}{2x - 4} = \lim_{x \to 2} \frac{x^2 + 2x + 4}{2} = \frac{4 + 4 + 4}{2} = 6. \quad\blacksquare$$

## Пример 2. Предел с иррациональностью

**Задача.** Найти $\displaystyle\lim_{x \to 4} \frac{\sqrt{x^2 - 7} - 3}{x - 4}.$

**Решение.**

Умножим числитель и знаменатель на сопряжённое $\sqrt{x^2 - 7} + 3$:

$$\lim_{x \to 4} \frac{x^2 - 7 - 9}{(x - 4)(\sqrt{x^2 - 7} + 3)} = \lim_{x \to 4} \frac{(x - 4)(x + 4)}{(x - 4)(\sqrt{x^2 - 7} + 3)}.$$

Сокращаем $(x - 4)$:

$$= \lim_{x \to 4} \frac{x + 4}{\sqrt{x^2 - 7} + 3} = \frac{8}{3 + 3} = \frac{4}{3}. \quad\blacksquare$$

## Пример 3. Уравнение касательной

**Задача.** Составить уравнение касательной к графику функции $f(x) = \dfrac{x^2 + 4}{x - 2}$ в точке его пересечения с осью ординат.

**Решение.**

Абсциса $x_0$ точки пересечения графика с осью $Oy$ равна 0, ордината $y_0 = f(0) = \dfrac{4}{-2} = -2$. Значит, точка касания — $(0; -2)$.

Найдём производную по правилу частного:

$$f'(x) = \frac{2x(x - 2) - (x^2 + 4) \cdot 1}{(x - 2)^2} = \frac{x^2 - 4x - 4}{(x - 2)^2}.$$

Подставим $x = 0$:

$$f'(0) = \frac{0 - 0 - 4}{4} = -1.$$

Уравнение касательной: $y - (-2) = -1 \cdot (x - 0)$, т. е. $y = -x - 2$. $\blacksquare$

## Пример 4. Промежутки монотонности и экстремумы

**Задача.** Найти промежутки возрастания и убывания и точки экстремума функции $f(x) = \dfrac{(x - 2)^2}{x^2}.$

**Решение.**

Область определения — вся числовая ось, кроме $x = 0$. Найдём производную:

$$f'(x) = \frac{2(x - 2) \cdot x^2 - (x - 2)^2 \cdot 2x}{x^4} = \frac{2x(x - 2)[x - (x - 2)]}{x^4} = \frac{4(x - 2)}{x^3}.$$

$f'(x) = 0$ при $x = 2$. Составим таблицу знаков:

| Интервал | $(-\infty, 0)$ | $(0, 2)$ | $(2, +\infty)$ |
|---|:---:|:---:|:---:|
| $f'(x)$ | $+$ | $-$ | $+$ |
| $f(x)$ | $\nearrow$ | $\searrow$ | $\nearrow$ |

$x = 2$ — точка минимума, $f(2) = 0$.

Функция возрастает на $(-\infty, 0)$ и $(2, +\infty)$, убывает на $(0, 2)$. $\blacksquare$

## Пример 5. Наибольшее и наименьшее на отрезке

**Задача.** Найти наименьшее и наибольшее значения функции $y = f(x) = 2\sin x + \sin 2x$ на отрезке $[0, 3\pi/2]$.

**Решение.**

Значения на концах: $f(0) = 0$, $f(3\pi/2) = -2$.

Находим критические точки:

$$f'(x) = 2\cos x + 2\cos 2x = 0 \implies \cos x + \cos 2x = 0.$$

Используя $\cos 2x = 2\cos^2 x - 1$:

$$2\cos^2 x + \cos x - 1 = 0 \implies \cos x = \frac{1}{2} \text{ или } \cos x = -1.$$

Из $\cos x = \frac{1}{2}$: $x = \frac{\pi}{3}$ и $x = \frac{5\pi}{3}$ (не входит в отрезок). Из $\cos x = -1$: $x = \pi$.

Значения: $f(\pi/3) = 2 \cdot \frac{\sqrt{3}}{2} + \sin\frac{2\pi}{3} = \frac{3\sqrt{3}}{2}$, $f(\pi) = 0$.

Сравнивая: $y_{\min} = -2$ при $x = 3\pi/2$, $y_{\max} = \frac{3\sqrt{3}}{2}$ при $x = \pi/3$. $\blacksquare$

## Пример 6. Арифметическая прогрессия — оптимизация

**Задача.** В арифметической прогрессии шестой член равен 3, разность больше 0,5. При каком значении разности произведение первого, четвёртого и пятого её членов наибольшее?

**Решение.**

$a_6 = a_1 + 5d = 3$, откуда $a_1 = 3 - 5d$. Обозначим произведение:

$$y = a_1 \cdot a_4 \cdot a_5 = (3 - 5d)(3 - 5d + 3d)(3 - 5d + 4d) = (3 - 5d)(3 - 2d)(3 - d).$$

Раскрывая: $y = -10d^3 + 51d^2 - 72d + 27$.

$$y' = -30d^2 + 102d - 72 = -6(5d^2 - 17d + 12) = 0.$$

Корни: $d_1 = 1$, $d_2 = 2{,}4$. По условию $d > 0{,}5$.

Исследуем на $(0{,}5; +\infty)$: $d = 2{,}4$ — точка максимума. $\blacksquare$

## Пример 7. Оптимизация — цилиндр в сфере

**Задача.** Площадь поверхности сферы равна $27\pi$. Какова высота цилиндра наибольшего объёма, вписанного в эту сферу?

**Решение.**

$4\pi R^2 = 27\pi$, откуда $R^2 = 27/4$. Пусть $AD = x$ — диаметр основания цилиндра. Из прямоугольного треугольника $AOB$:

$$AB^2 = OB^2 - OA^2 = \frac{27}{4} - \frac{x^2}{4}.$$

Высота цилиндра $h = 2 \cdot AB$, объём:

$$V(x) = \pi \cdot \frac{x^2}{4} \cdot 2\sqrt{\frac{27 - x^2}{4}} = \frac{\pi}{4}(27x - x^3).$$

По смыслу $0 < x < 2R$, т. е. $0 < x < 3\sqrt{3}$.

$$V'(x) = \frac{\pi}{4}(27 - 3x^2) = 0 \implies x = 3.$$

$x = 3$ — точка максимума. Высота $h = 2\sqrt{\frac{27 - 9}{4}} = 2 \cdot \frac{3\sqrt{2}}{2} = 3\sqrt{2}$. $\blacksquare$

## Пример 8. Первообразная с условием

**Задача.** Для функции $f(x) = 2\sin 5x + \sqrt{x} + \dfrac{3}{5}$ найти первообразную $F(x)$ при условии, что графики $f(x)$ и $F(x)$ пересекаются в точке на оси $Oy$.

**Решение.**

Первообразная $\sin x$ есть $-\cos x$, поэтому по правилам п. 3°:

$$F(x) = -\frac{2}{5}\cos 5x + \frac{2}{3}x\sqrt{x} + \frac{3}{5}x + C.$$

Графики пересекаются на оси $Oy$, значит при $x = 0$: $F(0) = f(0)$.

$F(0) = -\frac{2}{5} + C$, а $f(0) = \frac{3}{5}$. Откуда $C = 1$.

Итого: $F(x) = -\dfrac{2}{5}\cos 5x + \dfrac{2}{3}x\sqrt{x} + \dfrac{3}{5}x + 1$. $\blacksquare$

## Пример 9. Площадь фигуры

**Задача.** Найти площадь замкнутой фигуры, ограниченной графиками функций $y = -x^3$, $y = \dfrac{8}{\sqrt{x}}$ и $y = 8$.

**Решение.**

Найдём координаты точек пересечения. Из $\begin{cases} y = -x^3 \\ y = 8 \end{cases}$ получим $A(-2; 8)$. Из $\begin{cases} y = \frac{8}{\sqrt{x}} \\ y = 8 \end{cases}$ получим $B(1; 8)$. Далее: $C(9; 0)$, $D(-2; 0)$.

Площадь — разность площади прямоугольника $ABCD$ ($S_{ABCD} = 11 \cdot 8 = 88$) и двух криволинейных треугольников $OAD$ и $OBC$:

$$S_1 = \int_{-2}^{0} (-x^3)\,dx = -\frac{x^4}{4}\Big|_{-2}^{0} = 4, \quad S_2 = \int_1^9 \frac{8}{\sqrt{x}}\,dx = \frac{8 \cdot 2}{3}x\sqrt{x}\Big|_1^9 = 48 - \frac{16}{3}.$$

Но проще: $S_2 = 16\sqrt{x}\Big|_1^9 = 48 - 16 = 48$ (пересчитаем точно по $8 \cdot 2\sqrt{x}$).

Итого: $S = S_{ABCD} - S_1 - S_2 = 88 - 4 - 48 = 36$ (кв. ед.). $\blacksquare$
