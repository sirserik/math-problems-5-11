# Пределы (11.001–11.018)

> **Класс:** 10–11 | **Темы:** вычисление пределов, неопределённости, проверка утверждений

---

## Вычислить (11.001–11.010)

**33.001.** $\displaystyle\lim_{x \to -2} \frac{x^3 + 8}{x^2 - 4}.$

**33.002.** $\displaystyle\lim_{x \to 0{,}5} \frac{4x^2 - 8x + 3}{2x^2 - 7x + 3}.$

**33.003.** $\displaystyle\lim_{x \to -1} \frac{x^3 + x^2 - x - 1}{x^3 + x^2 + x + 1}.$

**33.004.** $\displaystyle\lim_{x \to 2/5} \frac{5x^3 - 2x^2 + 5x - 2}{5x^4 - 2x^3 - 5x^2 + 2x}.$

**33.005.** $\displaystyle\lim_{x \to 4} \frac{x + \sqrt{x} - 6}{x - 5\sqrt{x} + 6}.$

**33.006.** $\displaystyle\lim_{x \to 0} \frac{\sqrt{1 + 3x^4} - \sqrt{1 - 2x}}{x + x^2 + 2x^3}.$

**33.007.** $\displaystyle\lim_{x \to 1} \frac{\sqrt[3]{\vphantom{1}x} - 1}{\sqrt[3]{x^2} + 2\,\sqrt[3]{x} - 3}.$

**33.008.** $\displaystyle\lim_{x \to 0} \frac{1 - \sqrt{2x + 1}}{x}.$

**33.009.** $\displaystyle\lim_{x \to 2} \frac{x - 2}{\sqrt{4x + 1} - 3}.$

**33.010.** $\displaystyle\lim_{x \to -1} \frac{\sqrt{2x + 3} - 1}{\sqrt{5 + x} - 2}.$

## Вычислить пределы и подтвердить или опровергнуть данные утверждения (11.011–11.018)

**33.011.** $\displaystyle\lim_{x \to 3} \frac{x^2 + x - 12}{3x - 9} = 5 + \lim_{x \to 2} \frac{x^4 - 16}{8 - x^3}.$

**33.012.** $\displaystyle\lim_{x \to -1} \frac{\sqrt{x^2 + 3} + 2x}{x + 1} > \lim_{x \to -0{,}5} \frac{2x^2 - 5x - 3}{4x^2 - 18x - 10}.$

**33.013.** $\displaystyle\lim_{x \to 0} \frac{\sqrt{1 + x + x^2} - 1}{x} < \lim_{x \to 1} \frac{x^4 - 2x^2 + 1}{x^2 - 1}.$

**33.014.** $\displaystyle\lim_{x \to 2} \frac{\sqrt{3x - 5} - 1}{x - 2} - \lim_{x \to 0} \frac{\sqrt[3]{x + 1} - 1}{x} < \cos\frac{\pi}{10}.$

**33.015.** $\displaystyle\lim_{x \to 1} \frac{x^3 - 1}{x^2 + 5x - 6} > \lim_{x \to 5} \frac{5 - x}{\sqrt{5} - \sqrt{x}}.$

**33.016.** $\displaystyle\lim_{x \to 1} \frac{x^3 + 3x^2 - 4}{x^2 - 1} \cdot \lim_{x \to 3} \frac{x - 3}{\sqrt[3]{x^2 - 1} - 2} < 0.$

**33.017.** $\displaystyle\lim_{x \to -1} \frac{3x^2 + 2x - 1}{9x^3 + 9x^2 - x - 1} = \lim_{x \to 0} \frac{x - \sqrt{x}}{2\sqrt{x + x}}.$

**33.018.** $\displaystyle\lim_{x \to 2} \frac{x^3 - 8}{x(x^2 - 4)} - \lim_{x \to 2} \frac{2^{x+2} - 16}{4^x - 4^2} = 1.$
