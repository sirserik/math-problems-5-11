## Найти производные функций (11.019–11.033)

**33.022.** а) $y = \sqrt[3]{4x^3 - 7x^2 + 1}$; б) $y = (\sin^2 x + 1)\,e^x.$

## Решить уравнения и неравенства с производными (11.034–11.037)

**33.034.** Решить уравнение $f'(x) - \dfrac{2}{x}f(x) = 0$, если $f(x) = x^3\ln x.$

**33.035.** Решить неравенство $f'(x) < g'(x)$, если $f(x) = \dfrac{x^3 + 1}{x}$, $g(x) = 5x + \dfrac{1}{x}.$

**33.036.** Решить неравенство $f'(x) + \varphi'(x) \leqslant 0$, если $f(x) = 2x^3 + 12x^2$, $\varphi(x) = 9x^2 + 72x.$

**33.037.** Решить уравнение $1 + 5f(x) + 6f'(x) = 0$, если $f(x) = \dfrac{1}{1 - x}.$
