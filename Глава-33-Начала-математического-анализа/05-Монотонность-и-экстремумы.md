## Найти промежутки возрастания и убывания функций (11.083–11.089)

**33.083.** $y = \sqrt{3}\sin x - \cos x.$

**33.084.** $y = \dfrac{1}{x^2 + x + 1}.$

**33.085.** $f(x) = -x\,(x - 3)^2.$

**33.086.** $f(x) = \dfrac{1}{x} + \dfrac{2}{x^2} - \dfrac{4}{x^3}.$

**33.087.** $f(x) = (2^x - 1)(2^x - 4)^2.$

**33.088.** Найти промежутки возрастания и убывания функции $f(x) = x + \dfrac{1}{x}$ и установить, в какой из точек $x_1 = \log_5 4$, $x_2 = \log_5 3$ заданная функция принимает большее значение.

**33.089.** Найти промежутки возрастания и убывания функции $f(x) = \dfrac{1 + x}{\sqrt{x}}$ и установить, в какой из точек $x_1 = e^{-1}$ или $x_2 = e^{-2}$ функция принимает большее значение.

## Найти наименьшее и наибольшее значения функций на заданных промежутках (11.090–11.111)

**33.090.** $y = x^3 - 3x^2 + 3x + 2$; $\;[-2, 2].$

**33.091.** $f(x) = 3x^4 + 4x^3 + 1$; $\;[-2, 1].$

**33.092.** $y = x^5 - x^3 + x + 2$; $\;[-1, 1].$

**33.093.** $y = \dfrac{x}{3} + \dfrac{3}{x}$; $\;[-5, -1].$

**33.094.** $y = \dfrac{x}{8} + \dfrac{2}{x}$; $\;[1, 6].$

**33.095.** $f(x) = \dfrac{\sin 2x}{\sin\left(\dfrac{\pi}{4} + x\right)}$; $\;\left[0, \dfrac{\pi}{2}\right].$

**33.096.** $f(x) = \cos^2\dfrac{x}{2}\sin x$; $\;[0, \pi].$

**33.097.** $y(x) = \dfrac{4}{\sqrt{x^2 + 16}}$;

а) $[-3, 3]$; б) $[2\sqrt{5}, 8].$

**33.098.** $f(x) = x + \cos^2 x$; $\;[0, \pi/2].$

**33.099.** $f(x) = \operatorname{tg} x + \operatorname{ctg} 2x$; $\;[\pi/6, \pi/3].$

**33.100.** $f(x) = \dfrac{1}{2}\cos 2x + \sin x$; $\;[0, \pi/2].$

**33.101.** $f(x) = \dfrac{x}{2} - \dfrac{1}{4}\sin 2x + \dfrac{1}{3}\cos^3 x - \cos x$; $\;[-\pi/2, \pi/2].$

**33.102.** $f(x) = \cos^2 x + \sin x$;

а) $[0, \pi/4]$; б) $[\pi/3, \pi].$

**33.103.** $f(x) = \sqrt[3]{\dfrac{x^2}{2x - 1}}$;

а) $[3/4, 2]$; б) $[3/2, 3].$

**33.105.** $f(x) = (5 - x)\,2^{-x}$;

**33.106.** $f(x) = 2^{\sqrt{x^2}}$;

**33.107.** $y = 3\,\sqrt[3]{(x - 1)^2} + x$;

а) $[-7, 0]$; б) $[1, 2].$

**33.108.** $f(x) = 2x^2 - \ln x$; $\;[1, e].$

**33.109.** Найти наибольшее значение функции $f(x) = \cos x\sqrt{\sin x}$ в промежутке $[0, \pi/2]$.

**33.110.** Найти наименьшее значение функции

$$f(x) = \left(\frac{2 + \cos x}{\sin x}\right)^2$$

в промежутке $(0, \pi)$.

**33.111.** Найти наименьшее значение функции

$$f(x) = \frac{1}{\sqrt[n]{1 + x}} + \frac{1}{\sqrt[n]{1 - x}}$$

в промежутке $[0, 1)$, $n$ — натуральное.
