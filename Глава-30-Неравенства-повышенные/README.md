## Типы задач

- **Алгебраические неравенства** — метод интервалов, дробно-рациональные, с модулем
- **Иррациональные неравенства** — возведение в квадрат с учётом знаков, системы
- **Показательные неравенства** — сравнение степеней, замена переменной
- **Логарифмические неравенства** — учёт основания (> 1 или < 1), ОДЗ, переменное основание
- **Тригонометрические неравенства** — графический метод, единичная окружность
- **Неравенства с параметрами** — исследование на выполнимость для всех x
- **Доказательства** — AM–GM, метод разности, полные квадраты
- **Области определения** — системы неравенств из условий существования
- **Системы неравенств** — пересечение множеств решений
