## Неравенства с модулем и алгебраические (8.115–8.134)

**30.115.** $\dfrac{|3x + 1|}{|x - 3|} < 3.$

**30.116.** $\dfrac{|x + 2| - |x|}{\sqrt{4 - x^3}} > 0.$

**30.117.** $\left(\dfrac{x^2}{8} + \dfrac{3x}{4} + \dfrac{3}{2} + \dfrac{1}{x}\right)\left(1 - x - \dfrac{(x-2)^2\,(1-x)}{(x+2)^2}\right) > 0.$

**30.118.** $\dfrac{|x - 3|}{x^2 - 5x + 6} \geqslant 2.$

**30.119.** $\dfrac{m^2 x + 1}{2} \leqslant \dfrac{m^2 x + 3}{3} \leqslant \dfrac{m + 9x}{6}.$

**30.120.** $\left|\dfrac{x^2 - 5x + 4}{x^2 - 4}\right| \leqslant 1.$

**30.121.** $\dfrac{x^4 + 3x^3 + 4x^2 - 8}{x^{0{,}5}} < 0.$

**30.122.** $\dfrac{3}{6x^2 - x - 12} < \dfrac{25x - 47}{10x - 15} + \dfrac{3}{3x + 4}.$

**30.123.** $\dfrac{2 - x}{x^3 + x^2} > \dfrac{1 - 2x}{x^3 - 3x^2}.$

**30.124.** $\dfrac{1}{x + 1} - \dfrac{2}{x^2 - x + 1} + \dfrac{1 - 2x}{x^3 + 1} \leqslant 0.$

**30.125.** $\dfrac{1}{x^2 - 4} + \dfrac{4}{2x^2 + 7x + 6} \leqslant \dfrac{1}{2x + 3} + \dfrac{4}{2x^3 + 3x^2 - 8x - 12}.$

**30.126.** $\dfrac{10\,(5 - x)}{3\,(x - 4)} - \dfrac{11}{3} \geqslant \dfrac{6 - x}{x - 4} + \dfrac{5\,(6 - x)}{x - 2}.$

**30.127.** $-9 < x^4 - 10x^2 < 56.$

**30.128.** $216x^6 + 19x^3 < 1.$

**30.129.** $|x - 6| > |x^2 - 5x + 9|.$

**30.130.** $x^2\,(x^4 + 36) - 6\sqrt{3}\,(x^4 + 4) < 0.$

**30.131.** $\dfrac{x^3 + 3x^2 - x - 3}{x^2 + 3x - 10} < 0.$

**30.132.** $\dfrac{x^3 - 2x^2 - 5x + 6}{x - 2} > 0.$

**30.133.** $x^2\,(x + 3\sqrt{5}) + 5\,(3x + \sqrt{5}) > 0.$

**30.134.** $(x^2 + 4x + 10)^2 - 7\,(x^2 + 4x + 11) + 7 < 0.$

## Иррациональные неравенства (8.135–8.142)

**30.136.** $\sqrt{x + 3} < \sqrt{x - 1} + \sqrt{x - 2}.$

**30.138.** $\dfrac{1}{\sqrt{x^2}} - \dfrac{3}{4} < \dfrac{1}{x} - \dfrac{1}{2}.$

**30.140.** $\sqrt{x + 3} + \sqrt{x - 2} - \sqrt{2x + 4} > 0.$

## Показательные неравенства (8.143–8.152)

**30.143.** $0{,}5^{\sqrt{3}} < 0{,}5^{1 - \cos 2x} < 0{,}5.$

**30.144.** $0{,}3^{\frac{1}{2} + \frac{1}{4} + \frac{1}{8} + \ldots} < 3\sqrt{0{,}3^{3x^2 + 5x}} < 1.$

**30.146.** $0{,}2^{-\log_4 x} > \sqrt[3]{0{,}008^{\log_4 x^{-1}}}.$

**30.147.** $2{,}25^{\log_2(x^2 - 3x - 10)} > \left(\dfrac{2}{3}\right)^{\log_{1/2}(x^2 + 4x + 4)}.$

**30.148.** $\left(\dfrac{3}{5}\right)^{13x^2} \leqslant \left(\dfrac{3}{5}\right)^{x^4 + 36} < \left(\dfrac{3}{5}\right)^{12x^2}.$

**30.149.** $0{,}5^{2\sqrt{x}} + 2 > 3 \cdot 0{,}5^{\sqrt{x}}.$

**30.150.** $9^{\log_2(x-1)-1} - 8 \cdot 5^{\log_2(x-1)-2} > 9^{\log_2(x-1)} - 16 \cdot 5^{\log_2(x-1)-1}.$

**30.152.** $25 \cdot 2^x - 10^x + 5^x > 25.$

**30.154.** $\log_{4/3}(\sqrt{x + 3} - x) > 0.$

**30.155.** $\log_{0{,}5}(x + 3) < \log_{0{,}25}(x + 15).$

**30.158.** $\dfrac{\log_2(\sqrt{4x + 5} - 1)}{\log_2(\sqrt{4x + 5} + 11)} > \dfrac{1}{2}.$
