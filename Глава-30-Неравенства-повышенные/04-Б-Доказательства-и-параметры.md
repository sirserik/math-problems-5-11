# Группа Б. Доказательства, параметры и оценки (8.095–8.114)

> **Класс:** 10–11 | **Сложность:** повышенная

---

## Доказать неравенства (8.095–8.100)

**30.095.** Доказать, что произведение суммы трёх положительных чисел на сумму их обратных не меньше 9.

**30.096.** Доказать, что если $a$ — любое действительное число, то верно неравенство $\dfrac{a^2 + a + 2}{\sqrt{a^2 + a + 1}} \geqslant 2.$

**30.097.** Доказать, что при условии $2y + 5x = 10$ выполняется неравенство $3xy - x^2 - y^2 < 7.$

**30.098.** Доказать, что если $4b + a = 1$, то выполняется неравенство $a^2 + 4b^2 \geqslant 1/5.$

**30.099.** Доказать, что многочлен $m^6 - m^5 + m^4 + m^2 - m + 1$ принимает положительные значения при всех действительных значениях $m$.

**30.100.** Показать, что при любых действительных значениях $x$ функция $y = \dfrac{x^2 + x + 1}{x^2 + 1}$ не может принимать значений, бо́льших $3/2$ и ме́ньших $1/2$.

## Параметры и оценки (8.101–8.114)

**30.101.** Найти все значения $a$, при которых выражение $\sqrt{(a+1)\,x^2 - 2\,(a-1)\,x + 3a - 3}$ имеет смысл для любых $x \in \mathbb{R}$.

**30.102.** При каких значениях $p$ оба корня квадратного трёхчлена $x^2 + 2(p+1)x + 9p - 5$ отрицательны?

**30.103.** При каких значениях $n$ оба корня уравнения $(n - 2)\,x^2 - 2nx + n + 3 = 0$ положительны?

**30.104.** При каких значениях $m$ корни уравнения $x^2 - (3m + 1)\,x - m - 2 = 0$ заключены в промежутке между $-1$ и $2$?

**30.105.** При каких значениях $a$ квадратный трёхчлен $ax^2 - 7x + 4a$ принимает отрицательные значения для любых действительных значений $x$?

**30.106.** Найти целые числа $x$, удовлетворяющие неравенству $\dfrac{2}{|x - 13|} > \dfrac{8}{9}.$

**30.107.** Найти область определения функции $f(x) = \sqrt{9 - \left(\dfrac{4x - 22}{x - 5}\right)^2}.$

**30.108.** Найти целые неотрицательные значения $x$, удовлетворяющие неравенству $\dfrac{x + 3}{x^2 - 4} + \dfrac{1}{x + 2} < \dfrac{2x}{2x - x^2}.$

**30.109.** При каких значениях $a$ неравенство $\dfrac{ax}{x^2 + 4} < 1{,}5$ выполняется для любых значений $x \in \mathbb{R}$?

**30.110.** Найти те значения $m$, при которых неравенство $\dfrac{x^2 - 8x + 20}{mx^2 + 2(m + 1)\,x + 9m + 4} < 0$ выполняется для любых действительных значений $x$.

**30.111.** При каких значениях $x$ разность $\dfrac{11x^2 - 5x + 6}{x^2 + 5x + 6} - x$ принимает только отрицательные значения?

**30.112.** При каких значениях $m$ неравенство $\dfrac{x^2 + mx - 1}{2x^2 - 2x + 3} < 1$ выполняется для любых $x$?

**30.113.** При каких значениях $m$ неравенство $\dfrac{x^2 - mx - 2}{x^2 - 3x + 4} > -1$ выполняется для любых $x$?

**30.114.** При каких значениях $a$ сумма $a + \dfrac{-1 + 9a + 4a^2}{a^2 - 3a - 10}$ принимает только положительные значения?
