## Дробные рациональные уравнения (базовые)

**18.039.** Решите уравнение:
а) $\dfrac{x^2 - 6}{x - 3} = \dfrac{x}{x - 3}$;
б) $\dfrac{x^2 + 2x}{x + 4} = \dfrac{8}{x + 4}$;
в) $\dfrac{20}{x} = 9 - x$;
г) $\dfrac{10}{x} = 7 - x$.

**18.040.** Решите уравнение:
а) $\dfrac{x - 4}{x} = \dfrac{2x + 10}{x + 4}$;
б) $\dfrac{x + 3}{x} = \dfrac{2x + 10}{x - 3}$;
в) $\dfrac{2x - 5}{x + 5} + \dfrac{3x + 4}{x + 2} = 1$;
г) $\dfrac{3x + 1}{x - 3} - \dfrac{2x - 3}{x + 2} = -7\dfrac{1}{11}$.

**18.041.** Решите уравнение:
а) $\dfrac{x^2 - 2x}{x - 1} - \dfrac{2x - 1}{1 - x} = 3$;
б) $\dfrac{x^2 - 2x + 1}{x - 3} + \dfrac{x + 1}{3 - x} = 4$;
в) $\dfrac{x^2 + 2x - 1}{x^2 - 12x} - \dfrac{3}{x - 12} = \dfrac{3}{x}$;
г) $\dfrac{36}{x^2 - 12x} - \dfrac{3}{x - 12} = 3$.

**18.042.** Решите уравнение:
а) $\dfrac{2x - 5}{x + 5} + \dfrac{3x + 4}{x + 2} = 1$;
б) $\dfrac{4 - 3x}{x + 1} + \dfrac{x + 1}{4 - 3x} = \dfrac{50}{7}$;
в) $\dfrac{x}{x - 5} + \dfrac{x}{x + 5} = 2\dfrac{2}{3}$.

**18.043.** Решите уравнение:

**18.044.** Найдите корни уравнения:
