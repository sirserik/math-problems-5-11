## Тематическое распределение

### Группа А (18.001 — 18.055)
- Неполные квадратные уравнения вида $ax^2 + c = 0$ (18.001 — 18.010)
- Неполные квадратные уравнения вида $ax^2 + bx = 0$ (18.011 — 18.018)
- Полные квадратные уравнения: формула дискриминанта (18.019 — 18.030)
- Теорема Виета: устный подбор корней (18.031 — 18.040)
- Составление квадратных уравнений по корням (18.041 — 18.048)
- Дробные рациональные уравнения (базовые) (18.049 — 18.055)

### Группа Б (18.056 — 18.100)
- Полные квадратные уравнения различных типов (18.056 — 18.065)
- Теорема Виета: выражения через корни (18.066 — 18.075)
- Исследование квадратного уравнения (18.076 — 18.083)
- Дробные рациональные уравнения (18.084 — 18.092)
- Текстовые задачи, сводящиеся к квадратным уравнениям (18.093 — 18.100)

### Группа В (18.101 — 18.135)
- Уравнения, сводящиеся к квадратным (замена переменной) (18.101 — 18.110)
- Параметрические и исследовательские задачи (18.111 — 18.120)
- Сложные дробные уравнения (18.121 — 18.127)
- Нестандартные задачи на теорему Виета (18.128 — 18.135)

## Как пользоваться сборником

1. **Прочитай теорию** — файл `00-Теория.md` содержит все определения и формулы.
2. **Разбери примеры** — в файле `01-Примеры.md` показаны 10 типовых решений.
3. **Начни с группы А** — неполные уравнения и базовое применение формулы корней.
4. **Переходи к группе Б** — теорема Виета, исследование, текстовые задачи.
5. **Группа В** — для тех, кто хочет решать олимпиадные и конкурсные задачи.
