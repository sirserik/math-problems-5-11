# Теория. Квадратные уравнения

---

## 1. Квадратное уравнение. Основные понятия

**Квадратным уравнением** называется уравнение вида

$$ax^2 + bx + c = 0,$$

где $a$, $b$, $c$ — данные числа, $a \neq 0$, а $x$ — неизвестное.

- $a$ — **старший коэффициент** (при $x^2$)
- $b$ — **второй коэффициент** (при $x$)
- $c$ — **свободный член**

Квадратное уравнение называется **приведённым**, если $a = 1$, то есть имеет вид $x^2 + px + q = 0$.

> **Запомни:** Обязательное условие — $a \neq 0$. Если $a = 0$, уравнение становится линейным.

---

## 2. Неполные квадратные уравнения

**Неполным** называется квадратное уравнение, в котором $b = 0$ или $c = 0$ (или оба сразу).

### Случай 1: $b = 0$, уравнение вида $ax^2 + c = 0$

Решение:

$$ax^2 = -c \quad \Rightarrow \quad x^2 = -\frac{c}{a}$$

- Если $-\dfrac{c}{a} > 0$: два корня $x_{1,2} = \pm\sqrt{-\dfrac{c}{a}}$
- Если $-\dfrac{c}{a} = 0$: один корень $x = 0$
- Если $-\dfrac{c}{a} < 0$: корней нет

**Пример:** $3x^2 - 12 = 0 \;\Rightarrow\; x^2 = 4 \;\Rightarrow\; x = \pm 2$.

### Случай 2: $c = 0$, уравнение вида $ax^2 + bx = 0$

Выносим $x$ за скобку:

$$x(ax + b) = 0$$

По свойству произведения: $x = 0$ или $ax + b = 0$, откуда $x = -\dfrac{b}{a}$.

**Всегда два корня:** $x_1 = 0$, $x_2 = -\dfrac{b}{a}$.

**Пример:** $5x^2 - 15x = 0 \;\Rightarrow\; x(5x - 15) = 0 \;\Rightarrow\; x_1 = 0,\; x_2 = 3$.

### Случай 3: $b = 0$ и $c = 0$, уравнение вида $ax^2 = 0$

Единственный корень: $x = 0$.

> **Важно:** Неполные уравнения решаются **без формулы дискриминанта** — методом разложения на множители или извлечением корня.

---

## 3. Дискриминант. Формула корней квадратного уравнения

Для **полного** квадратного уравнения $ax^2 + bx + c = 0$ ($a \neq 0$) введём **дискриминант**:

$$\boxed{D = b^2 - 4ac}$$

**Формула корней:**

$$\boxed{x_{1,2} = \frac{-b \pm \sqrt{D}}{2a}}$$

### Анализ дискриминанта

| Значение $D$ | Количество корней | Корни |
|:---:|:---:|---|
| $D > 0$ | два различных корня | $x_1 = \dfrac{-b + \sqrt{D}}{2a}$, $\quad x_2 = \dfrac{-b - \sqrt{D}}{2a}$ |
| $D = 0$ | один корень (двойной) | $x_1 = x_2 = \dfrac{-b}{2a}$ |
| $D < 0$ | нет действительных корней | — |

### Формула при чётном $b$

Если второй коэффициент чётный: $b = 2k$, удобно использовать сокращённую формулу.

Введём $D' = k^2 - ac$ (половинный дискриминант), тогда:

$$x_{1,2} = \frac{-k \pm \sqrt{D'}}{a}$$

**Пример:** $x^2 - 6x + 8 = 0$. Здесь $a=1$, $b=-6$, $c=8$.

$$D = (-6)^2 - 4 \cdot 1 \cdot 8 = 36 - 32 = 4 > 0$$
$$x_{1,2} = \frac{6 \pm 2}{2} \;\Rightarrow\; x_1 = 4,\; x_2 = 2.$$

## 4. Теорема Виета

Для **приведённого** квадратного уравнения $x^2 + px + q = 0$ с корнями $x_1$ и $x_2$:

$$\boxed{x_1 + x_2 = -p, \qquad x_1 \cdot x_2 = q}$$

Для **общего** уравнения $ax^2 + bx + c = 0$:

$$\boxed{x_1 + x_2 = -\frac{b}{a}, \qquad x_1 \cdot x_2 = \frac{c}{a}}$$

### Обратная теорема Виета

Если числа $x_1$ и $x_2$ таковы, что $x_1 + x_2 = -p$ и $x_1 \cdot x_2 = q$, то они являются корнями уравнения $x^2 + px + q = 0$.

**Применение:** Для уравнения $x^2 - 5x + 6 = 0$ ищем два числа с суммой $5$ и произведением $6$. Это $2$ и $3$: $\;x_1 = 2,\; x_2 = 3$.

### Полезные выражения через корни

Пусть $x_1, x_2$ — корни уравнения $ax^2 + bx + c = 0$. Обозначим $S = x_1 + x_2 = -\dfrac{b}{a}$, $P = x_1 x_2 = \dfrac{c}{a}$.

| Выражение | Формула |
|-----------|---------|
| $x_1^2 + x_2^2$ | $S^2 - 2P$ |
| $(x_1 - x_2)^2$ | $S^2 - 4P$ |
| $x_1^3 + x_2^3$ | $S^3 - 3PS$ |
| $\dfrac{1}{x_1} + \dfrac{1}{x_2}$ | $\dfrac{S}{P}$ (при $P \neq 0$) |
| $x_1^2 - x_2^2$ | $(x_1 - x_2)(x_1 + x_2) = S\sqrt{S^2 - 4P}$ |

---

## 5. Составление квадратного уравнения по корням

Если известны корни $x_1$ и $x_2$, уравнение составляется по формуле:

$$(x - x_1)(x - x_2) = 0$$

или в развёрнутой форме:

$$x^2 - (x_1 + x_2)x + x_1 x_2 = 0$$

**Пример:** Корни $x_1 = -3$, $x_2 = 5$.

$$x^2 - (-3 + 5)x + (-3) \cdot 5 = 0 \;\Rightarrow\; x^2 - 2x - 15 = 0.$$

---

## 6. Исследование квадратного уравнения

Квадратное уравнение $ax^2 + bx + c = 0$ исследуется через знак дискриминанта $D = b^2 - 4ac$ и расположение корней относительно нуля.

### Знаки коэффициентов и корней

По теореме Виета:

- Если $x_1 > 0$ и $x_2 > 0$: $\;S = x_1 + x_2 > 0$ и $P = x_1 x_2 > 0$, значит $-\dfrac{b}{a} > 0$ и $\dfrac{c}{a} > 0$.
- Если $x_1 < 0$ и $x_2 < 0$: $\;S < 0$ и $P > 0$.
- Если корни разных знаков: $P = x_1 x_2 < 0$, то есть $\dfrac{c}{a} < 0$.

### Особые условия

| Условие | Следствие |
|---------|-----------|
| Один корень равен нулю | $c = 0$ |
| Оба корня равны нулю | $b = 0$ и $c = 0$ |
| Корни равны по модулю, противоположны по знаку | $b = 0$ |
| Сумма корней равна нулю | $b = 0$ |
| Произведение корней равно нулю | $c = 0$ |

## 7. Дробные рациональные уравнения

**Дробным рациональным уравнением** называется уравнение, содержащее переменную в знаменателе.

### Метод решения

1. Найти **область допустимых значений** (ОДЗ) — все $x$, при которых знаменатели не равны нулю.
2. Умножить обе части уравнения на **наименьший общий знаменатель** (НОЗ).
3. Решить полученное целое уравнение.
4. **Проверить:** исключить из ответа значения, не входящие в ОДЗ (посторонние корни).

**Пример:**

$$\frac{x+1}{x-2} = \frac{3}{x-2} + 1$$

ОДЗ: $x \neq 2$. Умножаем на $(x-2)$:

$$x + 1 = 3 + (x - 2) \;\Rightarrow\; x + 1 = x + 1.$$

Тождество — уравнение не имеет решений (все $x \neq 2$ подходят? нет — тождество означает бесконечно много решений, но следует проверить ОДЗ: $x = 2$ исключается). Ответ: $x \in \mathbb{R},\; x \neq 2$.

> **Главная ошибка:** Не забывай проверять посторонние корни! Значение $x$, обращающее знаменатель в нуль, не является корнем.

---

## 8. Уравнения, сводящиеся к квадратным

### Метод замены переменной

Уравнения вида $f(g(x))^2 + p \cdot f(g(x)) + q = 0$ решают заменой $t = f(g(x))$:

$$t^2 + pt + q = 0.$$

Затем из найденных значений $t$ находят $x$.

**Пример:** $x^4 - 5x^2 + 4 = 0$. Замена $t = x^2$:

$$t^2 - 5t + 4 = 0 \;\Rightarrow\; t_1 = 1,\; t_2 = 4.$$

Из $x^2 = 1$: $x = \pm 1$. Из $x^2 = 4$: $x = \pm 2$.

### Типичные замены

| Тип уравнения | Замена |
|---------------|--------|
| $ax^4 + bx^2 + c = 0$ | $t = x^2$ |
| $a(f(x))^2 + b \cdot f(x) + c = 0$ | $t = f(x)$ |
| $ax^2 + \dfrac{b}{x^2} + c = 0$ | $t = x + \dfrac{k}{x}$ или $t = x^2 + \dfrac{k}{x^2}$ |

## Краткая шпаргалка

| Тема | Ключевая формула |
|------|-----------------|
| Дискриминант | $D = b^2 - 4ac$ |
| Формула корней | $x_{1,2} = \dfrac{-b \pm \sqrt{D}}{2a}$ |
| Двойной корень ($D=0$) | $x_0 = \dfrac{-b}{2a}$ |
| Теорема Виета (общая) | $x_1 + x_2 = -\dfrac{b}{a}$, $\quad x_1 x_2 = \dfrac{c}{a}$ |
| Теорема Виета (приведённая) | $x_1 + x_2 = -p$, $\quad x_1 x_2 = q$ |
| Уравнение по корням | $x^2 - (x_1 + x_2)x + x_1 x_2 = 0$ |
| Сумма квадратов корней | $x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1 x_2$ |
