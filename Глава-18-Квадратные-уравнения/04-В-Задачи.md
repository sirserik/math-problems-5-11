## Уравнения, сводящиеся к квадратным (замена переменной)

**18.101.** Решите уравнение:
а) $x^4 - 5|x^2| + 4 = 0$;
б) $x^4 - (9a^2 + 4)x^2 + 36a^2 = 0$;
в) $4x^4 - (b + 36)x^2 + 9b = 0$;
г) $9x^4 - (b - 18)x^2 - 2b = 0$.

**18.102.** Решите уравнение:
а) $(x+3)^4 - 13(x+3)^2 + 36 = 0$;
б) $(2x-1)^4 - (2x-1)^2 - 12 = 0$;
в) $(x-1)^4 - x^2 - 2^2 - 73 = 0$;
г) $(x+2)^4 + 2x^2 + 8x - 16 = 0$.

**18.103.** Решите уравнение:
а) $(x+1)^2(x^2 + 2x) = 12$;
б) $(x-2)^2(x^2 - 4x) + 3 = 0$;
в) $(x^2 + 3x + 1)(x^2 + 3x + 3) + 1 = 0$;
г) $(x^2 - 5x + 2)(2x^2 - 5x - 1) = 28$.

**18.104.** Решите уравнение:
а) $x^4 - (a^2 + 9)x^2 + 9a^2 = 0$;
б) $4x^4 - (b + 36)x^2 + 9b = 0$;
в) $(x^2 + 2x - 1)^2 = 2(x^2 + 2x - 1) + 3$;
г) $(x^2 - 3x)^2 - 2(x^2 - 3x) - 8 = 0$.

**18.105.** Решите уравнение:
а) $x^2 - 2x - 3\sqrt{x^2 - 2x + 2} + 4 = 0$;
б) $x^2 - 4x - \sqrt{x^2 - 4x + 3} + 1 = 0$.

**18.106.** Решите уравнение:
а) $\dfrac{x-2}{x+2} - 10\sqrt{\dfrac{x-2}{x+2}} + 9 = 0$;
б) $\dfrac{x^2 - 2x}{4x - 3} = 2x - x^2$;
в) $\dfrac{8x - 4x^2}{1 - x^2} = \dfrac{3 - 4x}{x + 1}$;
г) $\dfrac{x^2 - 2x - 2}{x - 3} = \dfrac{2x - 4}{x^2 - 3x}$.

**18.107.** Решите уравнение:
а) $x^2 - 7|x| + 6 = 0$;
б) $x^2 - 4|x| - 21 = 0$;
в) $(x-2)^2 - 8|x-2| + 15 = 0$;
г) $9x^2 - 24|x| + 16 = 0$.

**18.108.** Решите уравнение:
а) $x^2 - 5\sqrt{x^2} + 4 = 0$;
б) $x^2 + \sqrt{x^2} - 2 = 0$;
в) $x^2 - 5x - \dfrac{6|x|}{x} = 0$;
г) $x|x| + 7x - 12 = 0$.

**18.109.** Найдите сумму квадратов корней уравнения:
а) $x^2 + 2|x| - 1 = 0$;
б) $x^2 - 4|x| + 1 = 0$.

**18.110.** Решите уравнение $f(x) = f\!\left(\dfrac{1}{x}\right)$, где $f(x) = \dfrac{x+1}{x^2}$.

## Параметрические и исследовательские задачи

**18.111.** При каких значениях параметра $a$ уравнение $(a^2 - 6a + 8)x^2 + (a^2 - 4)x + (10 - 3a - a^2) = 0$ имеет более двух корней?

**18.112.** При каком значении $a$ уравнение $ax^2 - (a+1)x + 2a - 1 = 0$ имеет один корень?

**18.113.** При каком значении $a$ уравнение $(a+2)x^2 + 2(a+2)x + 2 = 0$ имеет один корень?

**18.114.** При каких значениях $a$ уравнение $(a^2 - 5a + 3)x^2 + (3a - 1)x + 2 = 0$ имеет два корня?

**18.115.** При каком значении параметра $a$ уравнение $(a + 4x - x^2 - 1)(a + 1 - |x - 2|) = 0$ имеет три корня?

**18.116.** Известно, что корни уравнения $x^2 - 5x + a = 0$ на $1$ меньше корней уравнения $x^2 - 7x + 3a - 6 = 0$. Найдите $a$ и корни каждого из уравнений.

**18.117.** Известно, что корни уравнения $x^2 - 13x + b = 0$ соответственно равны квадратам корней уравнения $x^2 + ax + 6 = 0$. Найдите $a$ и $b$ и корни каждого из уравнений.

**18.118.** При каких значениях $p$ и $q$ корни уравнения $x^2 + px + q = 0$ равны $2p$ и $\dfrac{q}{2}$?

**18.119.** При каких значениях параметра $a$ один из корней квадратного уравнения $(a^2 - 5a + 3)x^2 + (3a - 1)x + 2 = 0$ в два раза больше другого?

**18.120.** Докажите, что корни уравнения $x^2 + px + q = 0$, где $p$ и $q$ — нечётные числа, иррациональны.

## Сложные дробные уравнения

**18.121.** Решите уравнение:
а) $\dfrac{x^2 - 2x + 4}{x^2 - 2x + 4x - 8} + \dfrac{x^2 + 2x + 4}{x^2 + 2x^2 + 4x + 8} = \dfrac{2x + 2}{x^2 - 4}$;
б) $\dfrac{38}{x^4 + 20x - 100} + \dfrac{x + 10}{x^2 - x + 10} = \dfrac{x^2 + x}{x^2 + x - 10}$;
в) $\dfrac{4x}{8x^3 + 1} + \dfrac{1}{16x^4 - 4x^2 + 1} = \dfrac{2}{4x^2 - 2x + 1}$.

**18.122.** Найдите координаты точек пересечения с осью абсцисс графика функции:
а) $y = \dfrac{x^3 - 3x^2 + 2x}{x - 1}$;
б) $y = \dfrac{x^3 - x^2 - 2x}{x - 2}$.

**18.123.** Путь от $A$ до $B$, равный $20$ км, турист должен был пройти за определённое время. Однако он был задержан с выходом из $A$ на $1$ час, поэтому ему пришлось увеличить скорость на $1$ км/ч, чтобы ликвидировать опоздание. С какой скоростью должен был идти турист?

**18.124.** На участке пути длиной $300$ км поезд увеличил скорость на $10$ км/ч, в результате чего прибыл на конечную станцию на $1$ час раньше, чем планировалось по расписанию. С какой скоростью должен был идти поезд по расписанию?

**18.125.** При каком значении $y$ равны значения выражений:
а) $\dfrac{3y + 1}{9y^2 + 3y + 1} - \dfrac{1}{3y - 1}$ и $\dfrac{3y^2 - 13y + 1}{27y^3 - 1}$?
б) $\dfrac{2y - 1}{4y^2 - 2y + 1} + \dfrac{1}{2y + 1}$ и $\dfrac{6y^2 + y + 2}{8y^3 + 1}$?

**18.126.** Решите уравнение:
а) $\dfrac{3}{a + 2} + 1 = \dfrac{4}{a^2 + 4a + 4}$;
б) $\dfrac{2}{a - 3} + 1 = \dfrac{15}{a^2 - 6a + 9}$;
в) $\dfrac{1}{a^2 - 4a + 4} - \dfrac{4}{a^2 - 4} = \dfrac{1}{a + 2}$;
г) $\dfrac{4}{a^2 - 4} - \dfrac{1}{a^2 + 4a + 4} = \dfrac{1}{a - 2}$.

**18.127.** Решите уравнение:
а) $\dfrac{x^2 - 12}{x^2 - 4} + \dfrac{x}{x - 2} = 1$;
б) $\dfrac{x^2 - 3}{x^2 - 1} + \dfrac{x}{x - 1} = 1$;
в) $\dfrac{x}{x + 1} - \dfrac{1}{x} = \dfrac{1}{x^2 + x}$;
г) $\dfrac{x}{x - 2} - \dfrac{1}{x} = \dfrac{4}{x^2 - 2x}$.

## Нестандартные задачи на теорему Виета

**18.128.** Пусть $x_1$ и $x_2$ — корни уравнения $2x^2 - 7x - 3 = 0$. Составьте квадратное уравнение, корнями которого являются числа:
а) $x_1 - 2$ и $x_2 - 2$;
б) $2x_1 + 3$ и $2x_2 + 3$;
в) $\dfrac{1}{x_1}$ и $\dfrac{1}{x_2}$;
г) $x_1 + \dfrac{1}{x_2}$ и $x_2 + \dfrac{1}{x_1}$.

**18.129.** Пусть $x_1$ и $x_2$ — корни уравнения $4x^2 - 6x - 1 = 0$. Не вычисляя корней, найдите:
а) $x_1^3 + x_2^3$;
б) $\dfrac{x_1}{x_2^2} + \dfrac{x_2}{x_1^2}$;
в) $x_1^4 + x_2^4$.

**18.130.** Дано соотношение $a^2 - 3ab - 4b^2 = 0$. Выразите $a$ через $b$.

**18.131.** Дано соотношение $21a^2 - 4ab - b^2 = 0$. Выразите $a$ через $b$.

**18.132.** Дано соотношение $\left(\dfrac{a+2b}{a-b}\right)^2 - 2\left(\dfrac{a+2b}{a-b}\right) = 3$. Выразите $a$ через $b$.

**18.133.** Найдите отношение двух чисел, если отношение произведения этих чисел к сумме их квадратов равно $0{,}3$.

**18.134.** Найдите отношение двух чисел, если квадрат суммы этих чисел в $3$ раза больше неполного квадрата разности этих чисел.

**18.135.** Найдите отношение двух положительных чисел, если отношение их среднего геометрического к среднему арифметическому равно $0{,}6$.
