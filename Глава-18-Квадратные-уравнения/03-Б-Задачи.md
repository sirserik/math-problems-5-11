## Полные квадратные уравнения различных типов

**18.056.** Решите уравнение:
а) $x^2 + 2x - 63 = 0$;
б) $x^2 + 18x + 65 = 0$;
в) $0{,}9x - 3x^2 = 0$;
г) $0{,}6x + 2x^2 = 0$.

**18.057.** Решите уравнение:
а) $2x^2 - 5x + 2 = 0$;
б) $2x^2 - 3x - 2 = 0$;
в) $x^2 - 2x - 6 = 0$;
г) $x^2 + 2x - 4 = 0$.

**18.058.** Определите значения $y$, при которых верно равенство:
а) $y^2 + 6y - \dfrac{2y + 3}{2} = 12$;
б) $\dfrac{y^2 + 10y}{10} - \dfrac{2y + 5}{2} = 20$.

**18.059.** Решите уравнение:
а) $x^2 + x = 90$;
б) $x^2 - x = 110$;
в) $-4x = 7x^2$;
г) $-3x^2 = 11x$.

**18.060.** Решите уравнение:
а) $\dfrac{1}{5}x^2 + x - 10 = 0$;
б) $\dfrac{1}{4}x^2 - x - 3 = 0$;
в) $x^2 + 4x + 5 = 0$ (исследуйте $D$);
г) $x^2 - 2x + 3 = 0$ (исследуйте $D$).

**18.061.** Определите значения $x$, при которых верно равенство:
а) $\dfrac{(x-3)^2}{16} - \dfrac{(x-2)^2}{4} = \dfrac{1-x}{2}$;
б) $\dfrac{(x+1)^2}{12} - \dfrac{(x-1)^2}{3} = \dfrac{2x-1}{4}$.

**18.062.** Решите уравнение:
а) $x^2 - 3\sqrt{2}\,x + 4 = 0$;
б) $x^2 + 2(1 + \sqrt{8})\,x + 8\sqrt{2} = 0$;
в) $x^2 - 3x - 5 - \sqrt{7} = 0$;
г) $x^2 + 4x - \sqrt{3} + 1 = 0$.

**18.063.** Разность корней уравнения $2x^2 - 5x + c = 0$ равна $1{,}5$. Найдите $c$.

**18.064.** Разность корней уравнения $2x^2 - 3x + c = 0$ равна $2{,}5$. Найдите $c$.

**18.065.** Один из корней уравнения $x^2 + x - a = 0$ равен $4$. Найдите второй корень и число $a$.

## Теорема Виета: выражения через корни

а) $x_1^2 + x_2^2$;
б) $x_1 x_2^2 + x_2 x_1^2$;
в) $\dfrac{x_1}{x_2} + \dfrac{x_2}{x_1}$;
г) $x_1^4 + x_2^4$.

**18.067.** Не вычисляя корней уравнения $2x^2 - 5x - 4 = 0$, найдите:
а) $\dfrac{1}{x_1} + \dfrac{1}{x_2}$;
б) $x_1 x_2^2 + x_2 x_1^2$;
в) $\dfrac{x_1}{x_2^2} + \dfrac{x_2}{x_1^2}$;
г) $x_1^4 + x_2^4$.

**18.068.** Не вычисляя корней уравнения $2x^2 - 5x + 1 = 0$, найдите разность квадратов его корней.

**18.069.** Пусть $x_1$ и $x_2$ — корни уравнения $2x^2 - 7x - 3 = 0$. Составьте квадратное уравнение, корнями которого являются числа:
а) $x_1 - 2$ и $x_2 - 2$;
б) $2x_1 + 3$ и $2x_2 + 3$;
в) $\dfrac{1}{x_1}$ и $\dfrac{1}{x_2}$;
г) $x_1 + \dfrac{1}{x_2}$ и $x_2 + \dfrac{1}{x_1}$.

**18.070.** Пусть $x_1$ и $x_2$ — корни уравнения $4x^2 - 6x - 1 = 0$. Составьте квадратное уравнение, корнями которого являются числа:
а) $x_1^2 x_2$ и $x_1 x_2^2$;
б) $\dfrac{1}{x_1^2}$ и $\dfrac{1}{x_2^2}$.

**18.071.** При каких значениях $k$ произведение корней квадратного уравнения $x^2 + 3x + (k^2 - 7k + 12) = 0$ равно нулю?

**18.072.** При каких значениях $k$ сумма корней квадратного уравнения $x^2 + (k^2 + 4k - 5)x - k = 0$ равна нулю?

**18.073.** В уравнении $x^2 - 4x + a = 0$ сумма квадратов корней равна $16$. Найдите $a$.

**18.074.** В уравнении $x^2 - 2x + a = 0$ квадрат разности корней равен $16$. Найдите $a$.

## Дробные рациональные уравнения

**18.084.** Решите уравнение:
а) $\dfrac{x^2 - 12}{x - 3} = \dfrac{x}{3 - x}$;
б) $\dfrac{x^2 - 8x}{5 - x} = \dfrac{15}{x - 5}$;
в) $\dfrac{2x^2 - 5x + 2}{x - 2} = 4x + 1$;
г) $\dfrac{2x^2 - x - 1}{x + 1} = 3x + 1$.

**18.085.** Решите уравнение:
а) $\dfrac{2x - 3}{x} - \dfrac{1}{x + 2} = \dfrac{4x - 6}{x^2 + 2x}$;
б) $\dfrac{3x + 1}{x} + \dfrac{5}{x - 2} = \dfrac{6x - 2}{x^2 - 2x}$;
в) $\dfrac{x + 2}{x - 1} + \dfrac{x}{x + 1} = \dfrac{6}{x^2 - 1}$;
г) $\dfrac{x - 1}{x + 2} + \dfrac{x}{x - 2} = \dfrac{8}{x^2 - 4}$.

**18.086.** Решите уравнение:
а) $\dfrac{7}{x + 1} - \dfrac{x + 4}{2 - 2x} = \dfrac{3x^2 - 38}{x^2 - 1}$;
б) $\dfrac{x + 0{,}5}{9x + 3} + \dfrac{8x^2 + 3}{9x^2 - 1} = \dfrac{x + 2}{3x - 1}$;
в) $\dfrac{x + 3}{4x^2 - 9} - \dfrac{3 - x}{4x^2 + 12x + 9} = \dfrac{2}{2x - 3}$;
г) $\dfrac{1 - 2x}{14x^2 - 7x} + \dfrac{2x + 1}{14x^2 - 7x} = \dfrac{8}{12x^2 - 3}$.

**18.087.** Решите уравнение:
а) $\dfrac{30}{x^2 - 1} + \dfrac{7 - 18x}{x^2 - x + 1} = \dfrac{13}{x^2 - x + 1}$;
б) $\dfrac{1}{1 - x} - \dfrac{2}{x^2 + x + 1} = \dfrac{2x + 1}{1 - x^3}$;
в) $\dfrac{65}{1 - x^3} + \dfrac{17x - 10}{x^2 + x + 1} = \dfrac{25}{x - 1}$;
г) $\dfrac{x^2 + x + 16}{x^2 - x + 1} - \dfrac{36 - x}{x^3 + 1} = \dfrac{x - 6}{x + 1}$.

**18.088.** Решите уравнение:
а) $\dfrac{2x - 7}{x^2 - 9x + 14} = \dfrac{1}{x^2 - 3x + 2} - \dfrac{1}{x - 1}$;
б) $\dfrac{2x + 7}{x^2 + 5x - 6} + \dfrac{3}{x^2 + 9x + 18} = \dfrac{1}{x + 3}$;
в) $\dfrac{25}{4x^2 - 1} - \dfrac{8x + 29}{16x^3 - 1} = \dfrac{18x + 5}{8x^3 + 4x^2 + 2x + 1}$;
г) $\dfrac{x - 1}{x^3 + 3x^2 - x + 3} + \dfrac{1}{x^3 - 1} = \dfrac{x + 2}{x^3 + 3x^2 - x - 3}$.

**18.089.** Решите уравнение:
а) $\dfrac{6}{x^2 - 7x^2 - 7x + 1} - \dfrac{8}{x^3 - 8x^2 + x} = \dfrac{1}{x}$;
б) $\dfrac{x^2 - 2x + 4}{x^2 - 2x + 4x - 8} + \dfrac{x^2 + 2x + 4}{x^2 + 2x^2 + 4x + 8} = \dfrac{2x + 2}{x^2 - 4}$;
в) $\dfrac{38}{x^4 - 20x - 100} + \dfrac{x + 10}{x^2 - x + 10} = \dfrac{x^2 + x}{x^2 + x - 10}$.

**18.090.** Решите уравнение:
а) $\dfrac{x^2 + (3 - a)x - 3a}{x^2 - x - 12} = 0$;
б) $\dfrac{x^2 - (a+1)x + 2a - 2}{3x^2 - 7x + 2} = 0$;
в) $\dfrac{x^2 - (3b - 1)x + 2b^2 - 2b}{x^2 - 7x + 6} = 0$;
г) $\dfrac{x^2 + (1 - 4b)x + 3b^2 - b}{2x^2 + 3x - 5} = 0$.

## Текстовые задачи, сводящиеся к квадратным уравнениям

**18.091.** Моторная лодка прошла $60$ км по течению реки и $36$ км по озеру, затратив на весь путь $5$ часов. Найдите собственную скорость лодки, если скорость течения реки равна $2$ км/ч.

**18.092.** Расстояние между пристанями равно $112$ км. Двигаясь по течению, катер прошёл это расстояние на $1$ час быстрее, чем обратный путь. Найдите собственную скорость катера, если скорость течения реки равна $1$ км/ч.

**18.093.** Катер прошёл $18$ км по течению реки, а затем $20$ км против течения, затратив на весь путь $2$ ч. Найдите скорость течения реки, если собственная скорость катера $20$ км/ч.

**18.094.** Расстояние между станциями $A$ и $B$ равно $120$ км. В полночь из $A$ в $B$ отправляется поезд. В $3$ ч той же ночи из $B$ в $A$ отправляется другой поезд, проходящий в час на $10$ км больше первого. Второй поезд прибывает в $B$ на $2$ ч позже первого. В котором часу второй поезд прибыл в $B$?

**18.095.** Два пешехода выходят навстречу друг другу из пунктов, расстояние между которыми $28$ км. Если бы первый пешеход не задержался на $1$ ч на расстоянии $9$ км от места своего отправления, то встреча пешеходов произошла бы на середине пути. После остановки первый пешеход увеличил свою скорость на $1$ км/ч, и они встретились на расстоянии $4$ км от места его остановки. Найдите скорость каждого пешехода.

**18.096.** Экскаватор роет котлованы ёмкостью по $20$ м$^3$. После того как был вырыт первый котлован, производительность экскаватора уменьшилась на $1$ м$^3$/ч. Известно, что через $6{,}5$ ч после начала работы было вырыто полтора котлована. Найдите первоначальную производительность экскаватора.

**18.097.** Два двигателя начали работу одновременно. Первый из них, прекратив работу на $2$ ч позже второго, израсходовал $300$ т топлива. Второй двигатель израсходовал $192$ т. Сколько топлива расходует в час первый двигатель, если известно, что эта его характеристика на $6$ г превышает соответствующую характеристику второго двигателя?

**18.098.** Бак ёмкостью $2400$ м$^3$ наполняется топливом. При опорожнении этого бака производительность насоса на $10$ м$^3$/мин выше, чем при заполнении. В результате время опорожнения бака на $8$ мин меньше времени заполнения. Определите производительность насоса при заполнении.

**18.099.** В колхозе два поля засеяли пшеницей: с первого поля собрали $1080$ ц зерна, а со второго — $750$ ц. Площадь второго поля на $10$ га больше площади первого. Если бы с $1$ га первого поля собирали столько же, сколько с $1$ га второго поля, а с $1$ га второго поля столько же, сколько с $1$ га первого поля, то с обоих полей собрали бы одинаковое количество зерна. Сколько центнеров зерна собрали с $1$ га каждого поля?

**18.100.** Имеется два одинаковых бака. При совместной работе двух насосов один бак наполняется водой за $3$ ч $36$ мин. За сколько времени наполняется каждый бак, если к нему подведён только один насос и с помощью второго насоса бак наполняется на $3$ ч быстрее, чем с помощью первого?
