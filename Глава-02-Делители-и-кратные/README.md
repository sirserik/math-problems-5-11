# Глава 2. Делители, кратные, признаки делимости

## О главе

**Класс:** 5–6
**Количество задач:** 150 (нумерация 2.001–2.150)
**Тема:** Делимость натуральных чисел — фундамент всей арифметики и алгебры.

## Структура главы

| Файл | Содержание |
|------|-----------|
| `00-Теория.md` | Теоретический справочник: определения, признаки делимости, НОД, НОК, простые числа |
| `01-Примеры.md` | 8 разобранных примеров с подробными решениями |
| `02-А-Задачи.md` | Группа А — базовый уровень (задачи 2.001–2.060, 60 задач) |
| `03-Б-Задачи.md` | Группа Б — продвинутый уровень (задачи 2.061–2.110, 50 задач) |
| `04-В-Задачи.md` | Группа В — олимпиадный уровень (задачи 2.111–2.150, 40 задач) |

## Распределение по темам

### Группа А (базовый уровень) — 60 задач
- **2.001–2.015** — Делители и кратные числа (перечислить, найти)
- **2.016–2.030** — Признаки делимости (определить без деления)
- **2.031–2.040** — Простые и составные числа
- **2.041–2.050** — Разложение на простые множители
- **2.051–2.060** — НОД и НОК (прямые вычисления)

### Группа Б (продвинутый уровень) — 50 задач
- **2.061–2.075** — Применение признаков делимости (вставить цифру, комбинации)
- **2.076–2.085** — Текстовые задачи на НОД и НОК
- **2.086–2.095** — Взаимно простые числа и свойства
- **2.096–2.110** — Комбинированные задачи

### Группа В (олимпиадный уровень) — 40 задач
- **2.111–2.120** — Доказательства делимости
- **2.121–2.130** — Экзотические признаки делимости (на 7, 11, 13)
- **2.131–2.140** — Алгоритм Евклида для больших чисел
- **2.141–2.150** — Олимпиадные задачи на делимость

## Рекомендации

- **5 класс:** начните с группы А, затем переходите к группе Б (задачи 2.061–2.085).
- **6 класс:** группы А и Б полностью, из группы В — по силам.
- **Олимпиадная подготовка:** группа В целиком, с обязательным разбором доказательств.
