## Задачи на графики (29.277–29.278)

**29.277.** Найти точку пересечения графиков функций $y = \log_2(x + 14)$ и $y = 6 - \log_2(x + 2).$

**29.278.** Найти точки пересечения графика функции $f(x) = x^{\lg x} - 100\,000x^4$ с осью абсцисс.

## Системы уравнений (29.279–29.314)

**29.279.** $\begin{cases}2 - \log_2 y = 2\log_2(x + y),\\ \log_2(x + y) + \log_2(x^2 - xy + y^2) = 1.\end{cases}$

**29.280.** $\begin{cases}3 \cdot \left(\dfrac{2}{3}\right)^{2x-y} + 7 \cdot \left(\dfrac{2}{3}\right)^{\frac{x+y}{2}} - 6 = 0,\\ \lg(3x - y) + \lg(y + x) - 4\lg 2 = 0.\end{cases}$

**29.281.** $\begin{cases}(0{,}48^{x^2+2})^{2x-y} = 1,\\ \lg(x + y) - 1 = \lg 6 - \lg(x + 2y).\end{cases}$

**29.282.** $\begin{cases}\log_2(x - y) = 5 - \log_2(x + y),\\ \dfrac{\lg x - \lg 4}{\lg y - \lg 3} = -1.\end{cases}$

**29.283.** $\begin{cases}4^{y/x} = 32,\\ \log_3(x - y) = 1 - \log_3(x + y).\end{cases}$

**29.284.** $\begin{cases}y^{5x^3-51x+10} = 1,\\ xy = 15.\end{cases}$

**29.285.** $\begin{cases}\log_x y = 2,\\ \log_{x+1}(y + 23) = 3.\end{cases}$

**29.286.** $\begin{cases}(x^2 + y)\,2^{y-x^3} = 1,\\ 9\,(x^2 + y) = 6^{x^3-y}.\end{cases}$

**29.287.** $\begin{cases}y - \log_3 x = 1,\\ x^y = 3^{12}.\end{cases}$

**29.288.** $\begin{cases}9^{4\sqrt{xy^2}} - 27 \cdot 3^{\sqrt{y}} = 0,\\ \dfrac{1}{4}\lg x + \dfrac{1}{2}\lg y = \lg(4 - 2\sqrt{x}).\end{cases}$

**29.289.** $\begin{cases}3^{-x} \cdot 2^y = 1152,\\ \log_{\sqrt{5}}(x + y) = 2.\end{cases}$

**29.290.** $\begin{cases}\lg(x^2 + y^2) = 1 + \lg 8,\\ \lg(x + y) - \lg(x - y) = \lg 3.\end{cases}$

**29.291.** $\begin{cases}3^x \cdot 2^y = 972,\\ \log_{\sqrt{3}}(x - y) = 2.\end{cases}$

**29.292.** $\begin{cases}3^{1+2\log_3(y-x)} = 48,\\ 2\log_5(2y - x - 12) - \log_5(y - x) = \log_5(y + x).\end{cases}$

**29.293.** $\log_9(x^2 + y^3) = \log_3(x^2 - y^2) = \log_5(x + y).$

**29.294.** $\begin{cases}(\log_a x + \log_a y - 2)\log_{18} a = 1,\\ 2x + y - 20a = 0.\end{cases}$

**29.295.** $\begin{cases}(x + y)\,3^{y-x} = \dfrac{5}{27},\\ 3\log_5(x + y) = x - y.\end{cases}$

**29.296.** $\begin{cases}2^{\sqrt{x}-2} + 4^{\sqrt{x}-1} = 5,\\ 3\,(x + y) + \dfrac{5\,(x - y)}{x - y} = 8.\end{cases}$

**29.297.** $\begin{cases}x^y = 2,\\ (2x)^{y^4} = 64\ (x > 0).\end{cases}$

**29.298.** $\begin{cases}\dfrac{x^2}{y} + \dfrac{y^2}{x} = 12,\\ 2^{-\log_2 x^2} + 5^{\frac{\log_5 1}{y}} = 1/3.\end{cases}$

**29.299.** $\begin{cases}x^{y^2-7y+10} = 1,\\ x + y = 8\ (x > 0).\end{cases}$

**29.300.** $\begin{cases}2\left(\log_{1/y} x - 2\log_{x/y} y\right) + 5 = 0,\\ xy^2 = 32.\end{cases}$

**29.301.** $\begin{cases}yx^{\log_y x} = x^{2{,}5},\\ \log_5 y \cdot \log_y(y - 2x) = 1.\end{cases}$

**29.302.** $\begin{cases}\lg(x - 3) - \lg(5 - y) = 0,\\ 4^{-1}\sqrt[4]{4^x} - 8\;\sqrt[4]{\sqrt{8^y}} = 0.\end{cases}$

**29.303.** $\begin{cases}\log_a(3x + 2y) = 2,\\ \log_b(2x + 3y) = 2.\end{cases}$

**29.304.** $\begin{cases}x + y = 12,\\ 2\,(2\log_3 x - \log_{1/y} y) = 5.\end{cases}$

**29.305.** $\begin{cases}x^{x^2-y^2-16} = 1,\\ x - y = 2\ (x > 0).\end{cases}$

**29.306.** $\begin{cases}\lg\sqrt{(x + y)^2} = 1,\\ \lg y - \lg|x| = \lg 2.\end{cases}$

**29.307.** $\begin{cases}4^x - 7 \cdot 2^{\frac{x-y}{2}} = 2^{3-y},\\ y - x = 3.\end{cases}$

**29.308.** $\begin{cases}5^{3\sqrt{x}} \cdot 2^{2\sqrt{y}} = 200,\\ 5^{2\sqrt[3]{x}} + 2^{2\sqrt{y}} = 689.\end{cases}$

**29.309.** $\begin{cases}10^{\frac{\lg^2(x^2+y^2)+1{,}5}{2}} = 100\sqrt{10},\\ \dfrac{\sqrt{x^2 + 10y}}{3} = \dfrac{6}{2\sqrt{x^2 + 10y} - 9}.\end{cases}$

**29.310.** $\begin{cases}y^x = 1{,}5 + y^{-x},\\ y^{2{,}5+x} = 64\ (y > 0).\end{cases}$

**29.311.** $\begin{cases}\lg(x + y) - \lg 5 = \lg x + \lg y - \lg 6,\\ \dfrac{\lg x}{\lg(y + 6) - (\lg y + \lg 6)} = -1.\end{cases}$

**29.312.** $\begin{cases}\log_y\dfrac{y}{x} - \log_x^2 x = 1,\\ \log_2(y - x) = 1.\end{cases}$

**29.313.** $\begin{cases}(x + y)^x = (x - y)^y,\\ \log_2 x - \log_2 y = 1.\end{cases}$

**29.314.** $\begin{cases}x^{x-2y} = 36,\\ 4\,(x - 2y) + \log_6 x = 9\end{cases}$ (найти только целочисленные решения).
