## 1. Показательная функция $y = a^x$

| Свойство | Описание |
|----------|----------|
| ООпр | $\mathbb{R}$ (все действительные числа) |
| Область значений | $\mathbb{R}_+$ (все положительные числа): $a^x > 0$ |
| Монотонность | При $a > 1$ возрастает; при $0 < a < 1$ убывает |
| Ключевое свойство | $a^{x_1} = a^{x_2} \Rightarrow x_1 = x_2$ |

---

## 2. Логарифмическая функция $y = \log_a x$

| Свойство | Описание |
|----------|----------|
| ООпр | $\mathbb{R}_+$ (все положительные числа) |
| Область значений | $\mathbb{R}$ |
| Монотонность | При $a > 1$ возрастает; при $0 < a < 1$ убывает |

---

## 3. Свойства логарифмов

### Основные формулы

| № | Формула | Название |
|---|---------|----------|
| (7.1) | $x = a^{\log_a x}$ | Основное логарифмическое тождество |
| (7.2) | $\log_a a = 1$ | Логарифм основания |
| (7.3) | $\log_a 1 = 0$ | Логарифм единицы |
| (7.4) | $\log_a(x_1 x_2) = \log_a x_1 + \log_a x_2$ | Логарифм произведения |
| (7.5) | $\log_a\dfrac{x_1}{x_2} = \log_a x_1 - \log_a x_2$ | Логарифм частного |
| (7.6) | $\log_a x^p = p\log_a x$ | Логарифм степени |
| (7.7) | $\log_a x = \dfrac{\log_b x}{\log_b a}$ | Переход к другому основанию |
| (7.8) | $\log_a b = \dfrac{1}{\log_b a}$ | Связь взаимных логарифмов |
| (7.9) | $\log_a b = \log_{a^p} b^p$ ($p \ne 0$) | Одновременное возведение |

> **Условия:** $a > 0$, $a \ne 1$, $x > 0$, $x_1 > 0$, $x_2 > 0$.

### Важные следствия

- $\log_a b \cdot \log_b a = 1$
- $a^{\log_a x} = x$ (при $x > 0$)
- $\log_{a^k} x = \dfrac{1}{k}\log_a x$

### Формулы с модулем (для чётных степеней)

- $\log_a(f(x) \cdot g(x)) = \log_a|f(x)| + \log_a|g(x)|$ (формула (7.24))
- $\log_a\dfrac{f(x)}{g(x)} = \log_a|f(x)| - \log_a|g(x)|$ (формула (7.25))
- $\log_a(f(x))^p = p\log|f(x)|$, $p$ — чётное число (формула (7.26))

> **Зачем:** формулы (7.24)–(7.26) предотвращают потерю корней при преобразованиях.

---

## 4. Методы решения показательных уравнений

### Указание 1. Приведение к одному основанию

$a^{f(x)} = b^{g(x)}$ ($a > 0$, $a \ne 1$, $b > 0$, $b \ne 1$)

Логарифмируем: $f(x)\log a = g(x)\log b$.

### Указание 2. Уравнение $(u(x))^{f(x)} = (u(x))^{g(x)}$

Корнями являются решения системы:
$$\begin{cases} u(x) > 0,\\ u(x) \ne 1,\\ f(x) = g(x) \end{cases}$$
и значения $x$, при которых $u(x) = 1$ (при условии определённости $f(x)$ и $g(x)$).

---

## 5. Методы решения логарифмических уравнений

### Указание 3. $\log_a f(x) = b$

Равносильно: $f(x) = a^b$.

### Указание 4. $\log_a f(x) = \log_a g(x)$

Равносильно каждой из систем:
$$\begin{cases}f(x) > 0,\\ f(x) = g(x)\end{cases} \quad\text{или}\quad \begin{cases}g(x) > 0,\\ f(x) = g(x).\end{cases}$$

### Указание 5. Применение формул (7.4)–(7.6)

$\log_a f(x) + \log_a g(x) = \log_a u(x) \Longleftrightarrow \log_a(f(x) \cdot g(x)) = \log_a u(x)$

> **Осторожно:** при использовании формул (7.4)–(7.6) возможна потеря корней. Для предотвращения используйте формулы (7.24)–(7.26) с модулями.

### Указание 6. Проверка ОДЗ

При решении обязательно проверяйте:
- Аргументы логарифмов должны быть положительны
- Основания логарифмов должны быть положительны и не равны 1
- Проверяйте каждый найденный корень подстановкой в исходное уравнение
