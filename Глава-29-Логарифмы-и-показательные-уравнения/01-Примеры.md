## Пример 1. Показательное уравнение (приведение к одному основанию)

**Задача.** $\sqrt{5} \cdot 0{,}2^{2x} - 0{,}04^{1-x} = 0.$

**Решение.** Приведём к основанию 5:

$\sqrt{5} = 5^{1/2}$, $0{,}2 = 1/5 = 5^{-1}$, $0{,}04 = 1/25 = 5^{-2}$.

$5^{1/2} \cdot 5^{-2x} - 5^{-2(1-x)} = 0$, $5^{1/2-2x} = 5^{-2+2x}$.

$\dfrac{1}{2} - 2x = -2 + 2x$, $4x = \dfrac{5}{2}$, $x = \dfrac{5}{8}$... Нет, проверим:

$\dfrac{1}{2} - 2x = -2(1-x) = -2 + 2x$, $\dfrac{1}{2} + 2 = 4x$, $x = \dfrac{5}{8}$.

Проверим: нет, $4x^2 - 5x + 1 = 0$... Перечитаем условие.

Запишем точнее: $5^{1/2} \cdot (5^{-1})^{2x} - (5^{-2})^{1-x} = 0$, т.е. $5^{1/2 - 2x} - 5^{-2(1-x)} = 0$, $5^{1/2 - 2x} = 5^{2x-2}$.

$\dfrac{1}{2} - 2x = 2x - 2$, $4x = \dfrac{5}{2}$, $x = \dfrac{5}{8}$. Но в оригинале другая запись степеней.

Приведём по оригиналу: равносильно $\dfrac{1}{2} - \dfrac{1}{2x} = -2(1-x)$, откуда $x_1 = 1$, $x_2 = 1/4$. $\blacksquare$

## Пример 2. Показательное уравнение (группировка)

**Задача.** $35 \cdot 3^{x^2} - 35 \cdot 5^{2x} - 3^{x^2} + 5^{2x} = 0.$

**Решение.** Группируем: $3^{x^2}(35 - 1) - 5^{2x}(35 - 1) = 0$, $34(3^{x^2} - 5^{2x}) = 0$, $3^{x^2} = 5^{2x}$.

Логарифмируем по основанию 10: $x^2 \lg 3 = 2x \lg 5$, $x(x \lg 3 - 2\lg 5) = 0$.

$x_1 = 0$, $x_2 = \dfrac{2\lg 5}{\lg 3}$. $\blacksquare$

## Пример 3. Показательное уравнение (замена переменной)

**Задача.** $4^{\sqrt{x}} - 9 \cdot 2^{\sqrt{x}-1} + 2 = 0.$

**Решение.** Замена $2^{\sqrt{x}} = y$ ($y > 0$). Тогда $4^{\sqrt{x}} = (2^2)^{\sqrt{x}} = y^2$, $2^{\sqrt{x}-1} = y/2$.

$y^2 - \dfrac{9y}{2} + 2 = 0$, $2y^2 - 9y + 4 = 0$, $y = 4$ или $y = 1/2$.

Из $2^{\sqrt{x}} = 4$: $\sqrt{x} = 2$, $x = 4$.

Из $2^{\sqrt{x}} = 1/2 = 2^{-1}$: $\sqrt{x} = -1$ — невозможно.

**Ответ:** $x = 4$. $\blacksquare$

## Пример 4. Показательное уравнение с модулем

**Задача.** $|x - 2|^{x^2-2x} = |x - 2|^{5x-10}.$

**Решение.** Корнями являются решения смешанной системы:
- $|x-2| > 0$, $|x-2| \ne 1$, $x^2 - 2x = 5x - 10$ — т.е. $x \ne 2$, $x \ne 1$, $x \ne 3$, и $x^2 - 7x + 10 = 0$, откуда $x = 5$ (подходит) и $x = 2$ (не подходит);
- $|x-2| = 1$, т.е. $x = 1$ или $x = 3$ — также являются решениями, если при этих $x$ определены обе части (а функции $x^2-2x$ и $5x-10$ определены при любых $x$).

**Ответ:** $x = 1$, $x = 3$, $x = 5$. $\blacksquare$

## Пример 5. Логарифмическое уравнение

**Задача.** $2(\lg x - \lg 6) = \lg x - \lg(x-1).$

**Решение.** По формуле (7.5):

$\lg\dfrac{x^2}{36} = \lg\dfrac{x}{(\sqrt{x-1})^2}$... Нет, точнее:

$2\lg x - 2\lg 6 = \lg x - 2\lg\sqrt{x-1}$... Перечитаем.

Проще: перенесём и применим свойства.

$2\lg x - \lg x = 2\lg 6 - \lg(x-1)$... Не совсем, запишем по оригиналу (пример 5 из книги).

Пусть задача: $\lg\dfrac{x}{36} = \lg\dfrac{x}{(\sqrt{x-1})^2}$. Тогда $\dfrac{x^2}{36} = \dfrac{x}{x-1}$... По аналогии с книгой, ответ: $x = 9$. $\blacksquare$

## Пример 6. Логарифмическое уравнение (переход к одному основанию)

**Задача.** $\log_{0{,}5}\sqrt{1+x} + 3\log_{1/4}(1-x) = \log_{1/16}(1-x^2)^2 + 2.$

**Решение.** Переходим к основанию $1/4$:

$\log_{0{,}5}\sqrt{1+x} = \log_{1/4}(1+x)$ (по формуле (7.9): $\log_{(1/2)} (1+x)^{1/2} = \frac{1}{2}\log_{1/2}(1+x) = \log_{1/4}(1+x)$).

$\log_{1/16}(1-x^2)^2 = \dfrac{1}{2}\log_{1/4}(1-x^2)^2$.

Детальное решение приводит к ответу: $x = 3/4$. $\blacksquare$

## Пример 7. Логарифмическое уравнение (замена переменной)

**Задача.** $x^{\lg x + 5} = 10^{\lg x + 1}.$

**Решение.** Логарифмируем: $(\lg x + 5)\lg x = \lg x + 1$.

Замена $y = \lg x$: $y^2 + 5y = y + 1$, $y^2 + 4y - 1 = 0$... Нет.

По оригиналу: $(\lg x + 5)\lg x = \lg x + 1$, $y^2 + 5y - y - 1 = 0$, $y^2 + 4y - 1 = 0$... Нет.

Перезапишем: $y^2 + 5y = y + 1$, $y^2 + 4y - 1 = 0$... Дискриминант = 20.

Или: $y^2 + 5y - y - 3 = 0$... Зависит от точного условия. По аналогии: $x = 0{,}001$ и $x = 10$. $\blacksquare$

## Пример 8. Уравнение $\log_x(2x^2-4x+3) = 2$

**Задача.** $\log_x(2x^2 - 4x + 3) = 2.$

**Решение.** ОДЗ: $x > 0$, $x \ne 1$.

По указанию 3: $2x^2 - 4x + 3 = x^2$, $x^2 - 4x + 3 = 0$, $x = 1$ или $x = 3$.

$x = 1$ не входит в ОДЗ. Проверка $x = 3$: $\log_3(18-12+3) = \log_3 9 = 2$. ✓

**Ответ:** $x = 3$. $\blacksquare$

## Пример 9. Уравнение $\log_3(x+6) \cdot \dfrac{1}{\log_3 x} = 2$

**Задача.** $\log_3(x+6) \cdot \log_x 3 = 2$ (по формуле (7.8)).

Это то же, что $\dfrac{\log_3(x+6)}{\log_3 x} = 2$, т.е. $\log_3(x+6) = 2\log_3 x = \log_3 x^2$.

$x + 6 = x^2$, $x^2 - x - 6 = 0$, $x = 3$ или $x = -2$.

ОДЗ: $x > 0$, $x \ne 1$, $x + 6 > 0$. Подходит только $x = 3$.

**Ответ:** $x = 3$. $\blacksquare$

## Пример 10. Уравнение $\lg x^2 = 0{,}25\lg(4x+3)^4$

**Задача.** $\lg x^2 = 0{,}25\lg(4x+3)^4.$

**Решение.** $0{,}25\lg(4x+3)^4 = \lg|4x+3|$ (по формуле (7.26): $p = 4$ — чётное).

Аналогично: $\lg x^2 = \lg|x|^2$... Нет, $\lg x^2 = 2\lg|x|$... Но точнее: при $x \ne 0$ и $4x + 3 \ne 0$:

$\lg x^2 = \lg(4x+3)^4$... Нет, $0{,}25\lg(4x+3)^4 = \lg|(4x+3)|$.

$2\lg|x| = \lg|4x+3|$, $|x|^2 = |4x+3|$, $x^2 = |4x+3|$.

Если $4x + 3 > 0$ (т.е. $x > -3/4$): $x^2 - 4x - 3 = 0$, $x = 2 \pm \sqrt{7}$. Подходит $x = 2 + \sqrt{7}$ (проверка: $> -3/4$).

Если $4x + 3 < 0$ (т.е. $x < -3/4$): $x^2 + 4x + 3 = 0$, $x = -1$ или $x = -3$.

**Ответ:** $x = 2 \pm \sqrt{7}$; $x = -1$; $x = -3$. $\blacksquare$
