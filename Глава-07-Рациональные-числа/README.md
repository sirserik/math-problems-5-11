# Глава 7. Рациональные числа

## Обзор

Седьмая глава единого сборника задач по математике для 5-6 классов посвящена **рациональным числам** — одной из ключевых тем курса 6 класса. Учащиеся знакомятся с отрицательными числами, учатся выполнять все арифметические действия с числами разных знаков и решать задачи, связанные с реальными ситуациями (температура, высота, долг).

## Класс

**6 класс**

## Количество задач

**150 задач** (нумерация 7.001 – 7.150)

## Структура главы

| Файл | Содержание | Задачи |
|------|-----------|--------|
| `00-Теория.md` | Теоретический материал с мнемониками | — |
| `01-Примеры.md` | 8 разобранных примеров | — |
| `02-А-Задачи.md` | Группа А — базовый уровень | 7.001 – 7.060 (60 задач) |
| `03-Б-Задачи.md` | Группа Б — средний уровень | 7.061 – 7.115 (55 задач) |
| `04-В-Задачи.md` | Группа В — продвинутый уровень | 7.116 – 7.150 (35 задач) |

## Темы

1. Положительные и отрицательные числа
2. Координатная прямая
3. Противоположные числа
4. Модуль числа
5. Сравнение рациональных чисел
6. Сложение рациональных чисел
7. Вычитание рациональных чисел
8. Умножение рациональных чисел
9. Деление рациональных чисел
10. Свойства действий с рациональными числами

## Уровни сложности

- **Группа А** — базовые операции: определение знака, нахождение модуля, сложение и вычитание целых чисел, простые операции с дробями со знаком.
- **Группа Б** — составные выражения, уравнения с отрицательными числами, цепочки сравнений, текстовые задачи на температуру, высоту, финансы.
- **Группа В** — многошаговые вычисления, задачи с модулем, нестандартные и олимпиадные задачи.
