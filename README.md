## Структура каждой главы

```
Глава-XX-Название/
├── README.md              ← Оглавление главы, статистика
├── 00-Теория.md           ← Формулы, правила, «как запомнить»
├── 01-Примеры.md          ← Подробные разборы (5–10 примеров)
├── 02-А-Задачи.md         ← Группа А: базовый уровень
├── 03-Б-Задачи.md         ← Группа Б: повышенный уровень
└── 04-В-Задачи.md         ← Группа В: олимпиадный уровень
```

## Уровни сложности

| Группа | Уровень | Для кого | Рекомендация |
|:------:|---------|----------|--------------|
| **А** | Базовый | Начинающие, закрепление | Обязательно для всех |
| **Б** | Повышенный | Подготовка к контрольным, ЕНТ | После уверенного освоения А |
| **В** | Олимпиадный | Олимпиады, углублённое изучение | Для углублённой подготовки |

## Как пользоваться

1. **Выберите главу** по нужной теме и классу
2. **Прочитайте теорию** (`00-Теория.md`) — убедитесь, что помните формулы
3. **Разберите примеры** (`01-Примеры.md`) — попробуйте решить сами, потом сверьтесь
4. **Решайте задачи** начиная с группы А, переходя к Б и В по мере освоения
