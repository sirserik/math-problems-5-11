## Тематическое распределение

### Группа А — Базовый уровень (35 задач)
Задание множеств перечислением и характеристическим свойством, определение принадлежности элементов, подмножества, пустое множество, простые операции пересечения, объединения и разности.

### Группа Б — Средний уровень (25 задач)
Диаграммы Эйлера-Венна, текстовые задачи на множества с применением формулы $|A \cup B| = |A| + |B| - |A \cap B|$, логические высказывания, операции «и», «или», «не».

### Группа В — Продвинутый уровень (20 задач)
Принцип Дирихле, сложные логические задачи, задачи на рассуждение и доказательство, олимпиадные задачи на множества и логику.
