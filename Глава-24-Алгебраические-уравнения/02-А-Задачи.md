*Задачи на теорему Виета и свойства корней (2.120–2.135):*

**24.120.** Не решая уравнения $ax^2 + bx + c = 0$, найти $x_1^2 + x_2^2$, где $x_1$ и $x_2$ — корни данного уравнения.

> **Подсказка:** $x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1x_2 = \frac{b^2}{a^2} - \frac{2c}{a} = \frac{b^2 - 2ac}{a^2}$.

**24.121.** Составить квадратное уравнение с корнями $\dfrac{1}{x_1}$ и $\dfrac{1}{x_2}$, где $x_1$ и $x_2$ — корни уравнения $ax^2 + bx + c = 0$.

**24.122.** Составить уравнение второй степени, один из корней которого равен сумме, а другой — произведению корней уравнения $ax^2 + bx + c = 0$.

**24.123.** Составить уравнение второй степени, корни которого были бы на единицу больше корней уравнения $ax^2 + bx + c = 0$.

**24.124.** Определить коэффициенты квадратного уравнения $x^2 + px + q = 0$ так, чтобы его корни были равны $p$ и $q$.
