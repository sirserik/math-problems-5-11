## Типы задач

- **Дробно-рациональные уравнения** — ОДЗ, общий знаменатель
- **Квадратные и биквадратные** — дискриминант, теорема Виета
- **Иррациональные** — возведение в квадрат, замена, проверка
- **Уравнения с модулем** — раскрытие по случаям
- **Системы линейных и нелинейных уравнений** — подстановка, сложение
- **Симметрические системы** — замена $s = x+y$, $p = xy$
- **Уравнения высших степеней** — группировка, замена переменной
- **Уравнения с параметрами** — исследование
