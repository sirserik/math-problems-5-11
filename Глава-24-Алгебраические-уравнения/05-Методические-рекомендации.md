### Классификация методов решения уравнений

| Метод | Когда применять | Примеры из главы |
|---|---|---|
| **Разложение на множители** | Когда можно вынести общий множитель или сгруппировать | 2.013, 2.017, 2.136 |
| **Замена переменной** | Уравнения с повторяющимися блоками ($x + 1/x$, $x^2 + ax$) | 2.151, 2.065, 2.066 |
| **Возведение в квадрат** | Иррациональные уравнения (+ обязательная проверка!) | 2.031–2.066 |
| **Деление на степень** | Симметрические уравнения | 2.270, 2.264 |
| **Теорема Виета** | Исследование корней без решения | 2.120–2.135, 2.342–2.370 |
| **Метод интервалов для ОДЗ** | Когда проверка сложна — сужаем ОДЗ заранее | Пример 3 |
| **Подстановка целых делителей** | Уравнения с целыми коэффициентами — ищем рациональные корни | 2.260, 2.270 |

### Типичные ошибки

1. **Забыть проверить корни** после возведения в квадрат
2. **Потерять корни** при делении на выражение с переменной
3. **Не учесть ОДЗ** — корни за пределами области определения
4. **Неправильно раскрыть модуль** — забыть случай отрицательного аргумента
5. **Решить систему подстановкой** и потерять симметричное решение — лучше использовать замены $s = x + y$, $p = xy$

### Порядок изучения

1. Разберите теорию и все 11 примеров — они покрывают основные методы
2. **Группа А:** обязательно решите задачи 2.001–2.030 (алгебраические) и 2.031–2.050 (иррациональные)
3. Системы 2.067–2.100 — критически важны для ЕНТ
4. **Группа Б:** после уверенного освоения группы А
5. **Группа В:** для олимпиад и углублённого изучения
