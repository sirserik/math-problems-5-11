### Классификация методов решения уравнений

| Метод | Когда применять | Примеры из главы |
|---|---|---|
| **Разложение на множители** | Когда можно вынести общий множитель или сгруппировать | 24.013, 24.017, 24.136 |
| **Замена переменной** | Уравнения с повторяющимися блоками ($x + 1/x$, $x^2 + ax$) | 24.151, 24.065, 24.066 |
| **Возведение в квадрат** | Иррациональные уравнения (+ обязательная проверка!) | 24.031–24.066 |
| **Деление на степень** | Симметрические уравнения | 24.270, 24.264 |
| **Теорема Виета** | Исследование корней без решения | 24.120–24.135, 24.342–24.370 |
| **Метод интервалов для ОДЗ** | Когда проверка сложна — сужаем ОДЗ заранее | Пример 3 |
| **Подстановка целых делителей** | Уравнения с целыми коэффициентами — ищем рациональные корни | 24.260, 24.270 |

### Типичные ошибки

1. **Забыть проверить корни** после возведения в квадрат
2. **Потерять корни** при делении на выражение с переменной
3. **Не учесть ОДЗ** — корни за пределами области определения
4. **Неправильно раскрыть модуль** — забыть случай отрицательного аргумента
5. **Решить систему подстановкой** и потерять симметричное решение — лучше использовать замены $s = x + y$, $p = xy$

### Порядок изучения

1. Разберите теорию и все 11 примеров — они покрывают основные методы
2. **Группа А:** обязательно решите задачи 24.001–24.030 (алгебраические) и 24.031–24.050 (иррациональные)
3. Системы 24.067–24.100 — критически важны для ЕНТ
4. **Группа Б:** после уверенного освоения группы А
5. **Группа В:** для олимпиад и углублённого изучения
