## Об этом разделе

Собраны задачи из ведущих международных математических соревнований для учащихся 7–8 классов:

- **AMC 10** (American Mathematics Competition 10) — США, для учащихся до 10 класса
- **Math Kangaroo** (Кенгуру) — международное соревнование, группы 7–8 класс

Задачи AMC 10 — отличная подготовка к олимпиадам для сильных учеников 7–8 классов.
