# Глава 3. Обыкновенные дроби

## Обзор

Третья глава единого сборника задач по математике для 5-6 классов посвящена **обыкновенным дробям** — одной из ключевых тем курса математики 5 класса. Глава охватывает все операции с дробями: от базового понятия дроби до сложных вычислений и олимпиадных задач.

## Структура главы

| Файл | Содержание | Кол-во задач |
|------|-----------|:------------:|
| `00-Теория.md` | Теоретический справочник с советами для запоминания | — |
| `01-Примеры.md` | Разобранные примеры с пошаговыми решениями | 10 |
| `02-А-Задачи.md` | Группа А — базовый уровень (3.001–3.080) | 80 |
| `03-Б-Задачи.md` | Группа Б — средний уровень (3.081–3.150) | 70 |
| `04-В-Задачи.md` | Группа В — продвинутый уровень (3.151–3.200) | 50 |

**Всего задач: 200** (нумерация 3.001–3.200)

## Уровни сложности

- **Группа А** — основные навыки: понятие дроби, сокращение, сравнение, четыре арифметических действия с дробями и смешанными числами.
- **Группа Б** — применение: сложные выражения, нахождение дроби от числа и числа по дроби, задачи на работу, текстовые задачи.
- **Группа В** — углублённый уровень: многоэтажные и цепные дроби, нестандартные задачи, совместная работа повышенной сложности, олимпиадные задачи.

## Соответствие программе

- **Класс**: 5
- **Раздел программы**: Обыкновенные дроби
- **Предварительные знания**: натуральные числа (Глава 1), делимость (Глава 2)
