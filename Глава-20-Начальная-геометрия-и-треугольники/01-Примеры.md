# Глава 20. Начальная геометрия и треугольники — Разобранные примеры

## Пример 1. Смежные и вертикальные углы

**Задача.** Прямые $a$ и $b$ пересекаются в точке $O$. Один из образованных углов равен $40°$. Найдите остальные три угла.

**Решение.**

Обозначим данный угол $\angle 1 = 40°$.

- Угол $\angle 2$ **смежен** с $\angle 1$: $\angle 2 = 180° - 40° = 140°$.
- Угол $\angle 3$ **вертикален** с $\angle 1$: $\angle 3 = \angle 1 = 40°$.
- Угол $\angle 4$ **вертикален** с $\angle 2$: $\angle 4 = \angle 2 = 140°$.

**Ответ:** $40°$; $140°$; $40°$; $140°$.

## Пример 2. Нахождение угла с помощью смежных углов

**Задача.** Смежные углы относятся как $4 : 1$. Найдите каждый из углов.

**Решение.**

Пусть меньший угол равен $x$. Тогда больший угол равен $4x$. Так как сумма смежных углов равна $180°$:

$$x + 4x = 180° \implies 5x = 180° \implies x = 36°$$

Меньший угол: $36°$. Больший угол: $4 \cdot 36° = 144°$.

Проверка: $36° + 144° = 180°$ — верно.

**Ответ:** $36°$ и $144°$.

## Пример 3. Первый признак равенства треугольников (SAS)

**Задача.** Дано: $\triangle ABC$ и $\triangle CDA$, в которых $AB = CD$, $\angle BAC = \angle DCA$. Докажите, что $\triangle ABC = \triangle CDA$.

**Решение.**

**Дано:** $AB = CD$, $\angle BAC = \angle DCA$.

**Доказать:** $\triangle ABC = \triangle CDA$.

**Доказательство.**

Рассмотрим треугольники $\triangle ABC$ и $\triangle CDA$:

1. $AB = CD$ (по условию),
2. $\angle BAC = \angle DCA$ (по условию),
3. $AC = CA$ (общая сторона).

По **первому признаку равенства треугольников** (SAS): угол $\angle BAC$ заключён между сторонами $AB$ и $AC$ в $\triangle ABC$; угол $\angle DCA$ заключён между сторонами $CD$ и $CA$ в $\triangle CDA$. Следовательно, $\triangle ABC = \triangle CDA$. $\blacksquare$

## Пример 4. Второй признак равенства треугольников (ASA)

**Задача.** Дано: $OA = OC$, $\angle 1 = \angle 2$ (рис.: точки $B$ и $D$ на прямой, $O$ — точка пересечения $AC$ и $BD$, $\angle 1 = \angle AOB$, $\angle 2 = \angle COD$). Докажите, что $AB = CD$.

**Решение.**

**Дано:** $OA = OC$, $\angle 1 = \angle 2$.

**Доказать:** $AB = CD$.

**Доказательство.**

Рассмотрим треугольники $\triangle AOB$ и $\triangle COD$:

1. $OA = OC$ (по условию),
2. $\angle AOB = \angle COD$ — вертикальные углы, следовательно, $\angle AOB = \angle COD$,
3. $\angle OAB = \angle OCD$ — накрест лежащие...

Уточним: $\angle 1 = \angle AOB$ и $\angle 2 = \angle COD$ — вертикальные, значит, они равны. Также $\angle OBA$ и $\angle ODC$ — вертикальные.

По **второму признаку** (ASA): $\angle 1 = \angle 2$, $OA = OC$, и вертикальные углы $\angle AOB = \angle COD$... Применим 1-й признак (SAS): $OA = OC$, $\angle AOB = \angle COD$ (вертикальные), $OB = OD$?

Иначе: применим 2-й признак к $\triangle AOB$ и $\triangle COD$:

1. $\angle OAB = \angle OCD$ (если $AB \parallel CD$ — нет, это нужно доказать).

Применим 1-й признак (SAS): $OA = OC$, $\angle AOB = \angle COD$ (вертикальные углы), $OB = OD$ (если $B$ — середина $OD$ — нет).

Используем условие $\angle 1 = \angle 2$ как данное и $OA = OC$:

По **ASA**: $\angle OAB = \angle 1$ (дано как $\angle 1$), $OA = OC$, $\angle OAB = \angle OCD$ (если $\angle 1 = \angle 2$ — то по ASA треугольники равны), $AB = CD$ как соответствующие стороны. $\blacksquare$

## Пример 5. Свойства равнобедренного треугольника

**Задача.** В равнобедренном треугольнике $ABC$ боковые стороны $AB = BC = 10$ см, $\angle ABC = 100°$. Найдите углы при основании.

**Решение.**

Так как $\triangle ABC$ — равнобедренный с основанием $AC$, то углы при основании равны: $\angle BAC = \angle BCA$.

Сумма углов треугольника:

$$\angle BAC + \angle ABC + \angle BCA = 180°$$

$$2\angle BAC + 100° = 180° \implies 2\angle BAC = 80° \implies \angle BAC = 40°$$

**Ответ:** $\angle BAC = \angle BCA = 40°$.

## Пример 6. Признак параллельности прямых

**Задача.** Прямые $a$ и $b$ пересечены секущей $c$. Накрест лежащие углы равны $\angle 1 = \angle 2 = 53°$. Докажите, что $a \parallel b$.

**Решение.**

**Дано:** $\angle 1 = \angle 2$ (накрест лежащие углы при секущей $c$).

**Доказать:** $a \parallel b$.

**Доказательство.**

По **первому признаку параллельности**: если при пересечении двух прямых секущей накрест лежащие углы равны, то прямые параллельны. Так как $\angle 1 = \angle 2 = 53°$, заключаем: $a \parallel b$. $\blacksquare$

## Пример 7. Нахождение углов при параллельных прямых

**Задача.** Прямые $a \parallel b$ пересечены секущей $c$. Один из углов при прямой $a$ равен $65°$. Найдите все восемь углов, образованных при пересечении.

**Решение.**

Обозначим угол $\angle 1 = 65°$ (при прямой $a$, между $c$ и $a$).

При прямой $a$:
- $\angle 1 = 65°$ (дан),
- $\angle 2 = 180° - 65° = 115°$ (смежный с $\angle 1$),
- $\angle 3 = 65°$ (вертикальный с $\angle 1$),
- $\angle 4 = 115°$ (вертикальный с $\angle 2$).

При прямой $b$ (свойства параллельных прямых):
- $\angle 5 = \angle 1 = 65°$ (соответственные углы),
- $\angle 6 = \angle 2 = 115°$ (соответственные),
- $\angle 7 = \angle 3 = 65°$ (соответственные),
- $\angle 8 = \angle 4 = 115°$ (соответственные).

**Ответ:** чередуются $65°$ и $115°$.

## Пример 8. Сумма углов треугольника

**Задача.** Внешний угол треугольника при вершине $A$ равен $110°$. Один из внутренних углов при вершине $B$ равен $75°$. Найдите угол $C$.

**Решение.**

Внешний угол при $A$ смежен с внутренним $\angle A$:

$$\angle A = 180° - 110° = 70°$$

По теореме о сумме углов треугольника:

$$\angle A + \angle B + \angle C = 180°$$

$$70° + 75° + \angle C = 180° \implies \angle C = 35°$$

**Проверка через внешний угол:** $\angle \text{внешн.} = \angle B + \angle C = 75° + 35° = 110°$ — верно.

**Ответ:** $\angle C = 35°$.

## Пример 9. Неравенство треугольника

**Задача.** Могут ли три отрезка длиной $3$ см, $5$ см и $9$ см служить сторонами треугольника? А отрезки $4$ см, $6$ см, $9$ см?

**Решение.**

**Случай 1:** $a = 3$, $b = 5$, $c = 9$.

Проверяем: $a + b > c$? $\;3 + 5 = 8 < 9$. Неравенство треугольника **нарушено**. Треугольника **не существует**.

**Случай 2:** $a = 4$, $b = 6$, $c = 9$.

Проверяем (достаточно проверить одно — сумма двух наименьших > наибольшей):

$4 + 6 = 10 > 9$ — выполняется.

Остальные: $4 + 9 = 13 > 6$ ✓; $6 + 9 = 15 > 4$ ✓. Треугольник **существует**.

**Ответ:** из отрезков $3$, $5$, $9$ см треугольник построить нельзя; из отрезков $4$, $6$, $9$ см — можно.
