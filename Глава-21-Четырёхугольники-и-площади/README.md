## Как пользоваться сборником

1. **Прочитай теорию** — файл `00-Теория.md` содержит все определения, свойства и формулы.
2. **Разбери примеры** — в файле `01-Примеры.md` показаны типовые решения.
3. **Начни с группы А** — базовые задачи на определения и простые вычисления.
4. **Переходи к группе Б** — задачи требуют комбинирования нескольких свойств.
5. **Группа В** — для тех, кто готовится к олимпиадам и экзаменам.
