## Основные формулы

### Соотношения между тригонометрическими функциями одного аргумента

$$\sin^2 x + \cos^2 x = 1 \tag{4.1}$$

$$\operatorname{tg} x = \frac{\sin x}{\cos x}, \quad x \ne \frac{\pi}{2}(2n+1),\ n \in \mathbb{Z} \tag{4.2}$$

$$\operatorname{ctg} x = \frac{\cos x}{\sin x}, \quad x \ne \pi n,\ n \in \mathbb{Z} \tag{4.3}$$

$$\operatorname{tg} x \cdot \operatorname{ctg} x = 1, \quad x \ne \frac{\pi n}{2},\ n \in \mathbb{Z} \tag{4.4}$$

$$1 + \operatorname{tg}^2 x = \frac{1}{\cos^2 x}, \quad x \ne \frac{\pi}{2}(2n+1) \tag{4.5}$$

$$1 + \operatorname{ctg}^2 x = \frac{1}{\sin^2 x}, \quad x \ne \pi n \tag{4.6}$$

> **Как запомнить:** формула (4.1) — это теорема Пифагора для единичной окружности. Формулы (4.5) и (4.6) получаются делением (4.1) на $\cos^2 x$ и $\sin^2 x$ соответственно.

### Формулы сложения

$$\sin(x + y) = \sin x \cos y + \cos x \sin y \tag{4.7}$$
$$\sin(x - y) = \sin x \cos y - \cos x \sin y \tag{4.8}$$
$$\cos(x + y) = \cos x \cos y - \sin x \sin y \tag{4.9}$$
$$\cos(x - y) = \cos x \cos y + \sin x \sin y \tag{4.10}$$

$$\operatorname{tg}(x + y) = \frac{\operatorname{tg} x + \operatorname{tg} y}{1 - \operatorname{tg} x \operatorname{tg} y} \tag{4.11}$$

$$\operatorname{tg}(x - y) = \frac{\operatorname{tg} x - \operatorname{tg} y}{1 + \operatorname{tg} x \operatorname{tg} y} \tag{4.12}$$

> **Мнемоника:** в синусе знаки «совпадают» ($+$ и $+$, $-$ и $-$), в косинусе знак «меняется» ($+$ даёт $-$, $-$ даёт $+$).

### Формулы двойного аргумента

$$\sin 2x = 2 \sin x \cos x \tag{4.13}$$

$$\cos 2x = \cos^2 x - \sin^2 x = 2\cos^2 x - 1 = 1 - 2\sin^2 x \tag{4.14}$$

$$\operatorname{tg} 2x = \frac{2 \operatorname{tg} x}{1 - \operatorname{tg}^2 x} \tag{4.15}$$

> **Как запомнить:** формула (4.13) — это частный случай (4.7) при $y = x$. Три формы (4.14) получаются подстановкой $\sin^2 x = 1 - \cos^2 x$ или $\cos^2 x = 1 - \sin^2 x$.

### Формулы половинного аргумента (понижения степени)

$$\sin^2 \frac{x}{2} = \frac{1 - \cos x}{2} \tag{4.16}$$

$$\cos^2 \frac{x}{2} = \frac{1 + \cos x}{2} \tag{4.17}$$

$$\operatorname{tg} \frac{x}{2} = \frac{\sin x}{1 + \cos x} = \frac{1 - \cos x}{\sin x} \tag{4.18}$$

> **Как запомнить:** в (4.16) стоит $1 - \cos x$ (минус, потому что $\sin 0 = 0$), в (4.17) стоит $1 + \cos x$ (плюс, потому что $\cos 0 = 1$).

### Формулы преобразования суммы в произведение

$$\sin x + \sin y = 2 \sin \frac{x+y}{2} \cos \frac{x-y}{2} \tag{4.19}$$

$$\sin x - \sin y = 2 \cos \frac{x+y}{2} \sin \frac{x-y}{2} \tag{4.20}$$

$$\cos x + \cos y = 2 \cos \frac{x+y}{2} \cos \frac{x-y}{2} \tag{4.21}$$

$$\cos x - \cos y = -2 \sin \frac{x+y}{2} \sin \frac{x-y}{2} \tag{4.22}$$

$$\operatorname{tg} x + \operatorname{tg} y = \frac{\sin(x+y)}{\cos x \cos y} \tag{4.23}$$

$$\operatorname{tg} x - \operatorname{tg} y = \frac{\sin(x-y)}{\cos x \cos y} \tag{4.24}$$

> **Мнемоника:** «сумма синусов — два синуса-косинуса, разность синусов — два косинуса-синуса». Аргументы — полусумма и полуразность.

### Формулы преобразования произведения в сумму

$$\sin x \sin y = \frac{1}{2}(\cos(x-y) - \cos(x+y)) \tag{4.25}$$

$$\cos x \cos y = \frac{1}{2}(\cos(x-y) + \cos(x+y)) \tag{4.26}$$

$$\sin x \cos y = \frac{1}{2}(\sin(x-y) + \sin(x+y)) \tag{4.27}$$

### Соотношения через $\operatorname{tg}(x/2)$ (универсальная подстановка)

$$\sin x = \frac{2 \operatorname{tg} \dfrac{x}{2}}{1 + \operatorname{tg}^2 \dfrac{x}{2}} \tag{4.28}$$

$$\cos x = \frac{1 - \operatorname{tg}^2 \dfrac{x}{2}}{1 + \operatorname{tg}^2 \dfrac{x}{2}} \tag{4.29}$$

> **Когда использовать:** универсальная подстановка $t = \operatorname{tg}(x/2)$ превращает тригонометрическое выражение в алгебраическое. Полезна, когда другие методы не работают.

### Формулы приведения

**Правило:** для аргумента $\frac{\pi}{2} \pm \alpha$ или $\frac{3\pi}{2} \pm \alpha$ функция **меняется на кофункцию** ($\sin \leftrightarrow \cos$, $\operatorname{tg} \leftrightarrow \operatorname{ctg}$). Для $\pi \pm \alpha$ или $2\pi \pm \alpha$ функция **не меняется**. Знак определяется по знаку исходной функции в соответствующей четверти (считая $\alpha$ острым углом).

| | $-\alpha$ | $\pi - \alpha$ | $\pi + \alpha$ | $\frac{\pi}{2} - \alpha$ | $\frac{\pi}{2} + \alpha$ | $\frac{3\pi}{2} - \alpha$ | $\frac{3\pi}{2} + \alpha$ |
|---|---|---|---|---|---|---|---|
| $\sin$ | $-\sin\alpha$ | $\sin\alpha$ | $-\sin\alpha$ | $\cos\alpha$ | $\cos\alpha$ | $-\cos\alpha$ | $-\cos\alpha$ |
| $\cos$ | $\cos\alpha$ | $-\cos\alpha$ | $-\cos\alpha$ | $\sin\alpha$ | $-\sin\alpha$ | $-\sin\alpha$ | $\sin\alpha$ |

> **Мнемоника «Лошадь — осёл»:** если в числителе $\pi$ (горизонтальная ось) → функция та же (лошадь → лошадь), если $\pi/2$ (вертикальная ось) → функция меняется (лошадь → осёл).

## Типичные приёмы решения

1. **Привести к одному аргументу** — используя формулы двойного/половинного угла
2. **Привести к одной функции** — используя основное тождество или формулы приведения
3. **Разложить в произведение** — формулы (4.19)–(4.22) для факторизации
4. **Сгруппировать слагаемые** — часто помогает попарная группировка в числителе и знаменателе
5. **Универсальная подстановка** — когда ничего другое не помогает
