# Глава 4. Разобранные примеры

> 8 подробных примеров, покрывающих основные типы задач главы.

---

## Пример 1. Доказать тождество

$$\operatorname{tg} 2\alpha + \operatorname{ctg} 2\alpha + \operatorname{tg} 6\alpha + \operatorname{ctg} 6\alpha = \frac{8\cos^2 4\alpha}{\sin 12\alpha}.$$

**Решение.** Применяя формулы (4.2), (4.3) и (4.13), преобразуем левую часть:

$$A = \frac{\sin 2\alpha}{\cos 2\alpha} + \frac{\cos 2\alpha}{\sin 2\alpha} + \frac{\sin 6\alpha}{\cos 6\alpha} + \frac{\cos 6\alpha}{\sin 6\alpha}$$

$$= \frac{\sin^2 2\alpha + \cos^2 2\alpha}{\sin 2\alpha \cos 2\alpha} + \frac{\sin^2 6\alpha + \cos^2 6\alpha}{\sin 6\alpha \cos 6\alpha} = \frac{1}{\sin 2\alpha \cos 2\alpha} + \frac{1}{\sin 6\alpha \cos 6\alpha}$$

$$= \frac{2}{\sin 4\alpha} + \frac{2}{\sin 12\alpha} = \frac{2(\sin 12\alpha + \sin 4\alpha)}{\sin 4\alpha \sin 12\alpha}.$$

Преобразуя сумму синусов по формуле (4.19) и учитывая $\sin 8\alpha = 2\sin 4\alpha \cos 4\alpha$:

$$A = \frac{4\sin 8\alpha \cos 4\alpha}{\sin 4\alpha \sin 12\alpha} = \frac{8\cos^2 4\alpha}{\sin 12\alpha}. \quad \blacksquare$$

**Ключевой приём:** привести $\operatorname{tg} + \operatorname{ctg}$ к общему знаменателю, использовать $\sin^2 + \cos^2 = 1$ и $\sin 2x = 2\sin x \cos x$.

## Пример 2. Упростить выражение

$$A = \cos\left(\frac{\pi}{6} - \frac{\alpha}{4}\right)\sin\left(\frac{\pi}{3} - \frac{\alpha}{4}\right)\sin\frac{\alpha}{4}.$$

**Решение.** К произведению первых двух сомножителей применим формулу (4.27):

$$A = \frac{1}{2}\left(\sin\left(\frac{\pi}{2} - \frac{\alpha}{2} - \frac{\pi}{6}\right) + \sin\frac{\pi}{6}\right)\sin\frac{\alpha}{4} = \frac{1}{2}\left(\cos\frac{\alpha}{2} + \frac{1}{2}\right)\sin\frac{\alpha}{4}$$

Снова используя формулу (4.27):

$$A = \frac{1}{4}\left(-\sin\frac{\alpha}{4} + \sin\frac{3\alpha}{4}\right) + \frac{1}{4}\sin\frac{\alpha}{4} = -\frac{1}{4}\sin\frac{\alpha}{4} + \frac{1}{4}\sin\frac{3\alpha}{4} + \frac{1}{4}\sin\frac{\alpha}{4} = \frac{1}{4}\sin\frac{3\alpha}{4}. \quad \blacksquare$$

**Ключевой приём:** последовательное применение формулы произведения в сумму.

## Пример 3. Представить в виде произведения

$$A = 2\cos^2 3\alpha + \sqrt{3}\sin 6\alpha - 1.$$

**Решение.** По формуле (4.14): $2\cos^2 3\alpha - 1 = \cos 6\alpha$. Следовательно:

$$A = \cos 6\alpha + \sqrt{3}\sin 6\alpha = 2\left(\frac{1}{2}\cos 6\alpha + \frac{\sqrt{3}}{2}\sin 6\alpha\right) = 2\left(\cos\frac{\pi}{3}\cos 6\alpha + \sin\frac{\pi}{3}\sin 6\alpha\right).$$

Выражение в скобках — развёрнутая формула (4.10) для косинуса разности:

$$A = 2\cos\left(\frac{\pi}{3} - 6\alpha\right). \quad \blacksquare$$

**Ключевой приём:** вспомогательный угол. Комбинация $a\cos x + b\sin x$ приводится к виду $R\cos(x - \varphi)$, где $R = \sqrt{a^2 + b^2}$.

## Пример 4. Проверить, что $\operatorname{tg} 20° + 4\sin 20° = \sqrt{3}$

**Решение.** Применив формулы (4.2), (4.13) и (4.19):

$$A = \operatorname{tg} 20° + 4\sin 20° = \frac{\sin 20°}{\cos 20°} + 4\sin 20° = \frac{\sin 20° + 4\sin 20°\cos 20°}{\cos 20°}$$

$$= \frac{\sin 20° + 2\sin 40°}{\cos 20°} = \frac{\sin 20° + \sin 40° + \sin 40°}{\cos 20°}.$$

По формуле (4.19): $\sin 20° + \sin 40° = 2\sin 30°\cos 10°= \cos 10°$. Далее:

$$A = \frac{\cos 10° + \sin 40°}{\cos 20°} = \frac{\cos 10° + \cos 50°}{\cos 20°}.$$

Снова по (4.21): $\cos 10° + \cos 50° = 2\cos 30°\cos 20° = \sqrt{3}\cos 20°$.

$$A = \frac{\sqrt{3}\cos 20°}{\cos 20°} = \sqrt{3}. \quad \blacksquare$$

**Ключевой приём:** группировка слагаемых и свёртка суммы в произведение.

## Пример 5. Найти значение $\operatorname{tg}(x/2)$, если $\sin x - \cos x = 1{,}4$

**Решение.** Используем формулы (4.28) и (4.29), полагая $t = \operatorname{tg}(x/2)$:

$$\frac{2t}{1 + t^2} - \frac{1 - t^2}{1 + t^2} = 1{,}4$$

$$\frac{2t - 1 + t^2}{1 + t^2} = 1{,}4$$

Получаем: $t^2 + 2t - 1 = 1{,}4(1 + t^2)$, т.е. $t^2 + 2t - 1 = 1{,}4 + 1{,}4t^2$.

$$0{,}4t^2 - 2t + 2{,}4 = 0 \implies t^2 - 5t + 6 = 0$$

Откуда $t_1 = 2$, $t_2 = 3$. Итак, $\operatorname{tg}(x/2) = 2$ или $\operatorname{tg}(x/2) = 3$.

> **Замечание:** нужно проверить, что $x \ne \pi(2n+1)$, иначе формулы (4.28)–(4.29) неприменимы. Подставив $\sin x - \cos x = 1{,}4 \ne 0 - (-1) = 1$, убеждаемся, что $x \ne \pi + 2\pi n$.

**Ключевой приём:** универсальная подстановка $t = \operatorname{tg}(x/2)$.

## Пример 6. Упростить выражение

$$A = \frac{1}{2}\sin^2\left(2\alpha + \frac{3\pi}{2}\right) - 2(\cos^4\alpha + \sin^4\alpha) + 2(\cos^6\alpha + \sin^6\alpha).$$

**Решение.** По формуле приведения: $\sin^2\left(2\alpha + \frac{3\pi}{2}\right) = \cos^2 2\alpha$.

Преобразуем $\cos^6\alpha + \sin^6\alpha$ как сумму кубов:

$$\cos^6\alpha + \sin^6\alpha = (\cos^2\alpha)^3 + (\sin^2\alpha)^3 = (\cos^2\alpha + \sin^2\alpha)(\cos^4\alpha - \cos^2\alpha\sin^2\alpha + \sin^4\alpha)$$

$$= \cos^4\alpha + \sin^4\alpha - \cos^2\alpha\sin^2\alpha.$$

Используя (4.1) и (4.13): $\cos^4\alpha + \sin^4\alpha = (\cos^2\alpha + \sin^2\alpha)^2 - 2\cos^2\alpha\sin^2\alpha = 1 - \frac{1}{2}\sin^2 2\alpha$.

Подставляя:

$$A = \frac{1}{2}\cos^2 2\alpha - 2\left(1 - \frac{1}{2}\sin^2 2\alpha\right) + 2\left(1 - \frac{1}{2}\sin^2 2\alpha - \frac{1}{4}\sin^2 2\alpha\right)$$

$$= \frac{1}{2}\cos^2 2\alpha - 2 + \sin^2 2\alpha + 2 - \sin^2 2\alpha - \frac{1}{2}\sin^2 2\alpha = \frac{1}{2}\cos^2 2\alpha - \frac{1}{2}\sin^2 2\alpha = \frac{1}{2}\cos 4\alpha. \quad \blacksquare$$

**Ключевой приём:** формула суммы кубов $a^3 + b^3 = (a+b)(a^2 - ab + b^2)$ при $a + b = 1$.

## Пример 7. Упростить выражение, найдя его значение при $\operatorname{tg} 5\alpha = \cos 20°\cos 40°\cos 80°$

$$A = \frac{\sin 8\alpha + \sin 9\alpha + \sin 10\alpha + \sin 11\alpha + \sin 12\alpha}{\cos 8\alpha + \cos 9\alpha + \cos 10\alpha + \cos 11\alpha + \cos 12\alpha}$$

**Решение.** Перегруппируем попарно и воспользуемся формулами (4.19) и (4.21):

Числитель: $(\sin 8\alpha + \sin 12\alpha) + (\sin 9\alpha + \sin 11\alpha) + \sin 10\alpha = 2\sin 10\alpha\cos 2\alpha + 2\sin 10\alpha\cos\alpha + \sin 10\alpha$

$$= \sin 10\alpha(2\cos 2\alpha + 2\cos\alpha + 1).$$

Знаменатель аналогично: $\cos 10\alpha(2\cos 2\alpha + 2\cos\alpha + 1)$.

$$A = \frac{\sin 10\alpha}{\cos 10\alpha} = \operatorname{tg} 10\alpha = \frac{2\operatorname{tg} 5\alpha}{1 - \operatorname{tg}^2 5\alpha}. \quad \blacksquare$$

**Ключевой приём:** попарная группировка «крайних» слагаемых, вынесение общего множителя.

## Пример 8. Доказать формулы суммы косинусов/синусов для $k$ слагаемых

**(а)** $\cos\alpha + \cos 2\alpha + \cos 3\alpha + \ldots + \cos k\alpha = \dfrac{\sin\dfrac{k\alpha}{2}\cos\dfrac{(k+1)\alpha}{2}}{\sin\dfrac{\alpha}{2}}$

**(б)** $\sin\alpha + \sin 2\alpha + \sin 3\alpha + \ldots + \sin k\alpha = \dfrac{\sin\dfrac{k\alpha}{2}\sin\dfrac{(k+1)\alpha}{2}}{\sin\dfrac{\alpha}{2}}$

**Решение (а).** Пусть $S_k(\alpha) = \cos\alpha + \cos 2\alpha + \ldots + \cos k\alpha$. Умножим обе части на $2\sin\frac{\alpha}{2}$:

$$2S_k(\alpha)\sin\frac{\alpha}{2} = 2\sin\frac{\alpha}{2}\cos\alpha + 2\sin\frac{\alpha}{2}\cos 2\alpha + \ldots + 2\sin\frac{\alpha}{2}\cos k\alpha.$$

Применяя формулу (4.27) к каждому слагаемому:

$$2\sin\frac{\alpha}{2}\cos n\alpha = \sin\left(n\alpha + \frac{\alpha}{2}\right) - \sin\left(n\alpha - \frac{\alpha}{2}\right) = \sin\frac{(2n+1)\alpha}{2} - \sin\frac{(2n-1)\alpha}{2}.$$

Сумма телескопическая: каждое слагаемое в скобке уничтожается со следующим. Остаётся:

$$2S_k(\alpha)\sin\frac{\alpha}{2} = \sin\frac{(2k+1)\alpha}{2} - \sin\frac{\alpha}{2}.$$

По формуле (4.20): правая часть $= 2\sin\frac{k\alpha}{2}\cos\frac{(k+1)\alpha}{2}$, что и даёт формулу (а). Аналогично доказывается (б).

**Ключевой приём:** телескопическая сумма. Умножение на «вспомогательный множитель» $2\sin(\alpha/2)$ превращает сумму в «складной».

> **Замечание:** формулы полезны для задач группы В с суммами большого числа тригонометрических функций.
