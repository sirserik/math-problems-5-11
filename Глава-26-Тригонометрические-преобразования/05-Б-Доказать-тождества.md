# Группа Б. Доказать тождества (26.186–26.239)

> **Класс:** 10–11 | **Сложность:** повышенная | **Тип:** сложные тригонометрические тождества

---

## Задачи

**26.186.** $\dfrac{\sin^4\alpha + \cos^4\alpha - 1}{\sin^6\alpha + \cos^6\alpha - 1} = \dfrac{2}{3}.$

**26.187.** $4\cos\left(\dfrac{\pi}{6} - \alpha\right)\sin\left(\dfrac{\pi}{3} - \alpha\right) = \dfrac{\sin 3\alpha}{\sin\alpha}.$

**26.188.** $\sin^{-1} 2\alpha\sin^{-1}(60° - 2\alpha)\sin^{-1}(60° + 2\alpha) = 4\sin^{-1} 6\alpha.$

**26.189.** $\dfrac{\cos 6\alpha - \cos 7\alpha - \cos 8\alpha + \cos 9\alpha}{\sin 6\alpha - \sin 7\alpha - \sin 8\alpha + \sin 9\alpha} = \operatorname{ctg}\dfrac{15\alpha}{2}.$

**26.190.** $\dfrac{\sin^2(3\pi - 4\alpha) + 4\cos^2\left(\dfrac{3\pi}{2} - 2\alpha\right) - 4}{\cos^2\left(\dfrac{\pi}{2} - 4\alpha\right) - 4\cos^2\left(2\alpha - \dfrac{5\pi}{2}\right)} = \operatorname{ctg}^4 2\alpha.$

**26.191.** Сложная дробь — результат: $= \dfrac{1}{8}.$

**26.192.** Сложная дробь — результат: $= \operatorname{tg}\left(\dfrac{\pi}{6} + \alpha\right)\operatorname{tg}\left(\alpha - \dfrac{\pi}{6}\right).$

**26.193.** Сложная дробь — результат: $= \dfrac{\operatorname{tg} 2\alpha}{\sqrt{3}}.$

**26.194.** $\operatorname{tg}\alpha + \cos^{-1}\alpha - 1 = \dfrac{\sqrt{2}\sin\dfrac{\alpha}{2}}{\sin\left(\dfrac{\pi}{4} - \dfrac{\alpha}{2}\right)}.$

**26.195.** $1 + \operatorname{ctg}\alpha + \sin^{-1}\alpha = \dfrac{\sqrt{2}\cos\alpha}{2\sin\dfrac{\alpha}{2}\sin\left(\dfrac{\pi}{4} - \dfrac{\alpha}{2}\right)}.$

**26.196.** $\dfrac{(1 + \sin\alpha)\operatorname{ctg}\left(\dfrac{\alpha}{2} - \dfrac{\pi}{4}\right)}{2\sin\left(\dfrac{7\pi}{4} - \dfrac{\alpha}{2}\right)\cos\left(\dfrac{5\pi}{4} + \dfrac{\alpha}{2}\right)} = -\operatorname{tg}^2\left(\dfrac{\pi}{4} + \dfrac{\alpha}{2}\right).$

**26.197.** $\dfrac{\operatorname{ctg}^2(2\alpha - \pi)}{1 + \operatorname{tg}^2\left(\dfrac{3\pi}{2} - 2\alpha\right)} - 3\cos^2\left(\dfrac{5\pi}{2} - 2\alpha\right) = 4\sin\left(\dfrac{\pi}{6} - 2\alpha\right)\sin\left(\dfrac{\pi}{6} + 2\alpha\right).$

**26.198.** Сложная дробь — результат: $= \operatorname{tg}^4\dfrac{\alpha}{2}.$

**26.199.** $1 - \cos(2\alpha - \pi) + \cos(4\alpha - 2\pi) = 4\cos 2\alpha\cos\left(\dfrac{\pi}{6} + \alpha\right)\cos\left(\dfrac{\pi}{6} - \alpha\right).$

**26.200.** $\sin^2\left(\dfrac{3\pi}{2} - \alpha\right)(\operatorname{tg}^2\alpha - 1)\operatorname{ctg}\left(\alpha - \dfrac{5\pi}{2}\right)\sin^{-2}\left(\dfrac{5\pi}{2} + \alpha\right) = 2.$

**26.201.** $\dfrac{\cos^4(\alpha - \pi)}{\cos^4\left(\alpha - \dfrac{3\pi}{2}\right) + \sin^4\left(\alpha + \dfrac{3\pi}{2}\right) - 1} = -\dfrac{1}{2}\operatorname{ctg}^2\alpha.$

**26.202.** $\operatorname{tg}\left(\alpha - \dfrac{3\pi}{4}\right)(1 - \sin 2\alpha) = \cos 2\alpha.$

**26.203.** $\dfrac{\cos 4\alpha\operatorname{tg} 2\alpha - \sin 4\alpha}{\cos 4\alpha\operatorname{ctg} 2\alpha + \sin 4\alpha} = -\operatorname{tg}^2 2\alpha.$

**26.204.** $\operatorname{ctg}\left(4\alpha - \dfrac{3\pi}{4}\right) + \cos^{-1}(4\alpha - 3\pi) = \operatorname{ctg}\left(2\alpha - \dfrac{\pi}{4}\right).$

**26.205.** $\sin\alpha\sin\beta + \sin\gamma - \sin(\alpha + \beta)\cos\gamma - \cos(\alpha + \beta)\sin\gamma = 4\sin\dfrac{\alpha + \beta}{2}\sin\dfrac{\beta + \gamma}{2}\sin\dfrac{\gamma + \alpha}{2}.$

**26.206.** $\dfrac{2\cos^2 2\alpha + \sqrt{3}\sin 4\alpha - 1}{2\sin^2 2\alpha + \sqrt{3}\sin 4\alpha} = \dfrac{\sin\left(4\alpha + \dfrac{\pi}{6}\right)}{\sin\left(4\alpha - \dfrac{\pi}{6}\right)}.$

**26.207.** $3 - 4\cos(4\alpha - 3\pi) - \cos(5\pi + 8\alpha) = 8\cos^4 2\alpha.$

**26.208.** $\dfrac{1 + \cos(2\alpha + 630°) + \sin(2\alpha + 810°)}{1 - \cos(2\alpha - 630°) + \sin(2\alpha + 630°)} = \operatorname{ctg}\alpha.$

**26.209.** $\dfrac{3 + 4\cos 4\alpha + \cos 8\alpha}{3 - 4\cos 4\alpha + \cos 8\alpha} = \operatorname{ctg}^4 2\alpha.$

**26.210.** $3 + 4\sin\left(4\alpha + \dfrac{3\pi}{2}\right) + \sin\left(8\alpha + \dfrac{5\pi}{2}\right) = 8\sin^4 2\alpha.$

**26.211.** $\cos^{-6}\alpha - \operatorname{tg}^6\alpha = 3\operatorname{tg}^2\alpha\cos^{-2}\alpha + 1.$

**26.212.** $\dfrac{1 - 2\sin^2 2\alpha}{1 - \sin 4\alpha} = \dfrac{1 + \operatorname{tg} 2\alpha}{1 - \operatorname{tg} 2\alpha}.$

**26.213.** $\dfrac{\sin^2(135° - \alpha) - \sin^2(210° - \alpha) - \sin 195°\cos(165° - 2\alpha)}{\cos^2(225° + \alpha) - \cos^2(210° - \alpha) + \sin 15°\sin(75° - 2\alpha)} = -1.$

**26.214.** $\dfrac{\sqrt{\operatorname{ctg}\alpha} + \sqrt{\operatorname{tg}\alpha}}{\sqrt{\operatorname{ctg}\alpha} - \sqrt{\operatorname{tg}\alpha}} = \operatorname{ctg}\left(\dfrac{\pi}{4} - \alpha\right).$

**26.215.** $\operatorname{ctg}^2\alpha + \operatorname{ctg}^2\beta - \dfrac{2\cos(\beta - \alpha)}{\sin\alpha\sin\beta} + 2 = \dfrac{\sin^2(\alpha - \beta)}{\sin^2\alpha\sin^2\beta}.$

**26.216.** $\sin 2\alpha(2\cos 4\alpha + 1)\operatorname{ctg}(30° - 2\alpha)\operatorname{ctg}(30° + 2\alpha) = \sin 6\alpha\operatorname{ctg} 2\alpha\operatorname{tg} 6\alpha.$

**26.217.** $\sin(\pi + \alpha)\sin\left(\dfrac{4\pi}{3} + \alpha\right)\sin\left(\dfrac{2\pi}{3} + \alpha\right) = \dfrac{1}{4}\sin 3\alpha.$

**26.218.** $\dfrac{\sin 6\alpha + \sin 7\alpha + \sin 8\alpha + \sin 9\alpha}{\cos 6\alpha + \cos 7\alpha + \cos 8\alpha + \cos 9\alpha} = \operatorname{tg}\dfrac{15\alpha}{2}.$

**26.219.** Сложная дробь — результат: $= -\operatorname{tg}\dfrac{\alpha}{8}.$

**26.220.** $\dfrac{1 + \cos(2\alpha - 2\pi) + \cos(4\alpha + 2\pi) - \cos(6\alpha - \pi)}{\cos(2\pi - 2\alpha) + 2\cos^2(2\alpha + \pi) - 1} = 2\cos 2\alpha.$

**26.221.** $\operatorname{ctg}\alpha - \operatorname{tg}\alpha - 2\operatorname{tg} 2\alpha - 4\operatorname{tg} 4\alpha = 8\operatorname{ctg} 8\alpha.$

**26.222.** $\operatorname{ctg}\alpha - \operatorname{tg}\alpha - 2\operatorname{tg} 2\alpha = 4\operatorname{ctg} 4\alpha.$

**26.223.** $4\cos\alpha\cos\varphi\cos(\alpha - \varphi) - 2\cos^2(\alpha - \varphi) - \cos 2\varphi = \cos 2\alpha.$

**26.224.** $\sin^2\varphi - \cos^2(\alpha - \varphi) + 2\cos\alpha\cos\varphi\cos(\alpha - \varphi) = \cos^2\alpha.$

**26.225.** $\cos^2\varphi + \cos^2(\alpha - \varphi) - 2\cos\alpha\cos\varphi\cos(\alpha - \varphi) = \sin^2\alpha.$

**26.226.** $\operatorname{tg} 6\beta - \operatorname{tg} 4\beta - \operatorname{tg} 2\beta = \operatorname{tg} 6\beta\operatorname{tg} 4\beta\operatorname{tg} 2\beta.$

**26.227.** Сложная дробь — результат: $= \operatorname{tg} 4\alpha.$

**26.228.** Сложная дробь — результат: $= \operatorname{ctg} 2\alpha.$

**26.229.** Сложная дробь — результат: $= -1.$

**26.230.** $\operatorname{tg} 4\alpha - \cos^{-1} 4\alpha = \dfrac{\sin 2\alpha - \cos 2\alpha}{\sin 2\alpha + \cos 2\alpha}.$

**26.231.** $\cos^6\alpha + \sin^6\alpha = \dfrac{5 + 3\cos 4\alpha}{8}.$

**26.232.** $\cos^8\alpha - \sin^8\alpha = \dfrac{\cos 2\alpha(3 + \cos 4\alpha)}{4}.$

**26.233.** $\operatorname{ctg}(30° - \alpha)\operatorname{ctg}(150° - \alpha)\operatorname{ctg}(270° + \alpha) = \operatorname{tg} 3\alpha.$

**26.234.** $4\sin\left(2\alpha - \dfrac{3\pi}{2}\right)\sin\left(\dfrac{\pi}{6} + 2\alpha\right)\sin\left(\dfrac{\pi}{6} - 2\alpha\right) = \cos 6\alpha.$

**26.235.** $\dfrac{1 - 2\cos^2 2\alpha}{2\operatorname{tg}\left(2\alpha - \dfrac{\pi}{4}\right)\sin^2\left(\dfrac{\pi}{4} + 2\alpha\right)} = 1.$

**26.236.** $16\sin^5\alpha - 20\sin^3\alpha + 5\sin\alpha = \sin 5\alpha.$

**26.237.** $\operatorname{tg}\left(\dfrac{\pi}{4} + \dfrac{\alpha}{2}\right) - \operatorname{tg}\alpha = \dfrac{1}{\cos\alpha}.$

**26.238.** $1 + \sin\left(3\left(\alpha + \dfrac{\pi}{2}\right)\right)\cos 2\alpha + 2\sin 3\alpha\cos(3\pi - \alpha)\sin(\alpha - \pi) = 2\sin^2(5\alpha/2).$

**26.239.** $(\sin\alpha - \sin\beta)(\sin\alpha + \sin\beta) = \sin(\alpha - \beta)\sin(\alpha + \beta).$
