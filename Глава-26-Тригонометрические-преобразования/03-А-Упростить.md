# Группа А. Упростить выражения (26.063–26.113)

> **Класс:** 9–10 | **Сложность:** базовая | **Тип:** упрощение тригонометрических выражений

## Подсказка

Цель — привести выражение к максимально простому виду. Используйте формулы приведения, двойного угла, суммы/разности, основное тождество.

---

## Задачи

**26.063.** $1 - \sin\left(\dfrac{\alpha}{2} - 3\pi\right) - \cos^2\dfrac{\alpha}{4} + \sin^2\dfrac{\alpha}{4}.$

**26.064.** $\dfrac{1 + \sin 2\alpha}{1 + \cos^2\alpha} + \cos^2\alpha.$

**26.065.** $\dfrac{\cos^2\left(\pi + \dfrac{\alpha}{4}\right)\left(1 + \operatorname{tg}^2\left(\dfrac{3\alpha}{4} - \dfrac{3\pi}{2}\right)\right)}{\sin^{-1}\left(\dfrac{9\pi}{2} + \dfrac{\alpha}{2}\right)\left(\operatorname{tg}^2\left(\dfrac{5\pi}{2} - \dfrac{\alpha}{4}\right) - \operatorname{tg}^2\left(\dfrac{3\alpha}{4} - \dfrac{7\pi}{4}\right)\right)}.$

**26.066.** $\dfrac{\sin\left(2\pi + \dfrac{\alpha}{4}\right)\operatorname{ctg}\dfrac{\alpha}{8} - \cos\left(2\pi + \dfrac{\alpha}{4}\right)}{\cos\left(\dfrac{\alpha}{4} - 3\pi\right)\operatorname{ctg}\dfrac{\alpha}{8} + \cos\left(\dfrac{7\pi}{2} - \dfrac{\alpha}{4}\right)}.$

**26.067.** $\cos\alpha(1 + \cos^{-1}\alpha + \operatorname{tg}\alpha)(1 - \cos^{-1}\alpha + \operatorname{tg}\alpha).$

**26.068.** $\sin^2\alpha(1 + \sin^{-1}\alpha + \operatorname{ctg}\alpha)(1 - \sin^{-1}\alpha + \operatorname{ctg}\alpha).$

**26.069.** $\dfrac{1 - \cos(8\alpha - 3\pi)}{\operatorname{tg} 2\alpha - \operatorname{ctg} 2\alpha}.$

**26.070.** $\cos\left(\dfrac{\pi}{6} - \dfrac{\alpha}{2}\right)\sin\left(\dfrac{\pi}{3} - \dfrac{\alpha}{2}\right)\sin\dfrac{\alpha}{2}.$

**26.071.** $\sin^2\left(\dfrac{\alpha}{2} + 2\beta\right) - \sin^2\left(\dfrac{\alpha}{2} - 2\beta\right).$

**26.072.** $\dfrac{\cos^{-1} 2x + \sin 2x\operatorname{tg} 2x}{1 + \cos 4x} + \dfrac{1}{4\sin^2\left(\dfrac{\pi}{4} - x\right)\operatorname{ctg}\left(\dfrac{\pi}{4} - x\right)}.$

**26.073.** $\cos^2(\alpha + 2\beta) + \sin^2(\alpha - 2\beta) - 1.$

**26.074.** $\sin^2(\alpha + 2\beta) + \sin^2(\alpha - 2\beta) - 1.$

**26.075.** $(\cos\alpha - \cos 2\beta)^2 + (\sin\alpha + \sin 2\beta)^2.$

**26.076.** $\dfrac{(1 - \cos 2\alpha)\cos(45° + 2\alpha)}{2\sin^2 2\alpha - \sin 4\alpha}.$

**26.077.** $\cos^2\left(\dfrac{3\pi}{8} - \dfrac{\alpha}{4}\right) - \cos^2\left(\dfrac{11\pi}{8} + \dfrac{\alpha}{4}\right).$

**26.078.** $\operatorname{ctg}\left(45° - \dfrac{\alpha}{2}\right) + \operatorname{ctg}\left(135° - \dfrac{\alpha}{2}\right).$

**26.079.** $\dfrac{1 + \operatorname{ctg} 2\alpha\operatorname{ctg}\alpha}{\operatorname{tg}\alpha + \operatorname{ctg}\alpha}.$

**26.080.** $\dfrac{\cos m\alpha - \cos n\alpha}{\sin n\alpha - \sin m\alpha}.$

**26.081.** $\sin^2\left(\alpha - \dfrac{3\pi}{2}\right)(1 - \operatorname{tg}^2\alpha)\operatorname{tg}\left(\dfrac{\pi}{4} + \alpha\right)\cos^{-2}\left(\dfrac{\pi}{4} - \alpha\right).$

**26.082.** $1 - \dfrac{1}{1 - \sin^{-1}\left(2\alpha + \dfrac{3\pi}{2}\right)}.$

**26.083.** $\dfrac{\cos^{-1}\alpha + \cos^{-1}\beta}{\operatorname{tg}\alpha\cos^{-1}\beta + \operatorname{tg}\beta\cos^{-1}\alpha}.$

**26.084.** $\dfrac{\operatorname{tg}\left(\dfrac{3\pi}{2} - \alpha\right) + \operatorname{tg}^2\left(\dfrac{\pi}{2} + \alpha\right)}{\operatorname{ctg}^3\left(\dfrac{5\pi}{2} - \alpha\right) + \operatorname{ctg}\left(\dfrac{3\pi}{2} + \alpha\right)}.$

**26.085.** $1 - \dfrac{1}{1 - \sin^{-1}\left(\dfrac{\pi}{2} + \alpha\right)}.$

**26.086.** $\dfrac{1 - \operatorname{tg}(\pi - 2\alpha)\operatorname{tg}\alpha}{\operatorname{tg}\left(\dfrac{3\pi}{2} - \alpha\right) + \operatorname{tg}\alpha}.$

**26.087.** $\dfrac{\operatorname{ctg}^2\left(\alpha + \dfrac{\pi}{2}\right)\cos^2\left(\alpha - \dfrac{\pi}{2}\right)}{\operatorname{ctg}^2\left(\alpha - \dfrac{\pi}{2}\right) - \cos^2\left(\alpha + \dfrac{\pi}{2}\right)}.$

**26.088.** $\dfrac{\operatorname{ctg}(270° - \alpha)}{\operatorname{tg}^2(\alpha - 180°)} \cdot \dfrac{\operatorname{ctg}^2(360° - \alpha) - 1}{\operatorname{ctg}(180° + \alpha)}.$

**26.089.** $\dfrac{\cos^2(\alpha - 270°)}{\sin^{-2}(\alpha + 90°) - 1} + \dfrac{\sin^2(\alpha + 270°)}{\cos^{-2}(\alpha - 90°) - 1}.$

**26.090.** $\dfrac{(1 + \operatorname{tg}^2(\alpha - 90°))(\sin^{-2}(\alpha - 270°) - 1)}{(1 + \operatorname{ctg}^2(\alpha + 270°))\cos^{-2}(\alpha + 90°)}.$

**26.091.** $\dfrac{\sin^2\left(\dfrac{\pi}{2} + \alpha\right) - \cos^2\left(\alpha - \dfrac{\pi}{2}\right)}{\operatorname{tg}^2\left(\dfrac{\pi}{2} + \alpha\right) - \operatorname{ctg}^2\left(\alpha - \dfrac{\pi}{2}\right)}.$

**26.092.** $\dfrac{\operatorname{tg}(\alpha/2) - \operatorname{ctg}(\alpha/2)}{\operatorname{tg}(\alpha/2) + \operatorname{ctg}(\alpha/2)}.$

**26.093.** $\dfrac{\cos^2\alpha - \operatorname{ctg}^2\alpha + 1}{\sin^2\alpha + \operatorname{tg}^2\alpha - 1}.$

**26.094.** $\dfrac{\cos^2(2\alpha - 90°) + \operatorname{ctg}^2(90° + 2\alpha) + 1}{\sin^2(2\alpha - 270°) + \operatorname{tg}^2(270° + 2\alpha) + 1}.$

**26.095.** $\dfrac{\sin^2\left(4\alpha - \dfrac{\pi}{2}\right)}{\operatorname{ctg}\left(\dfrac{3\pi}{2} - 2\alpha\right) + \operatorname{tg}\left(\dfrac{3\pi}{2} + 2\alpha\right)}.$

**26.096.** $\dfrac{1}{2\operatorname{tg}\left(\dfrac{3\pi}{2} - \alpha\right)\cos^2\left(\alpha - \dfrac{\pi}{2}\right)} + \dfrac{1 - \cos(4\alpha - \pi)}{\sin^3 2\alpha} + \dfrac{1}{2\operatorname{ctg}\left(\alpha + \dfrac{\pi}{2}\right)\sin^2\left(\alpha - \dfrac{3\pi}{2}\right)}.$

**26.097.** $\dfrac{\cos^2\alpha + 2\sin^2(\alpha - \pi)}{\cos^3(\alpha - 4\pi)} \cdot \dfrac{\cos^3\alpha + 4\sin\alpha + \sin^2(\alpha + \pi)}{\cos\alpha(4\sin\alpha + 1)}.$

**26.098.** $\sin\left(2\alpha - \dfrac{3\pi}{2}\right) + \cos\left(2\alpha - \dfrac{8\pi}{3}\right) + \cos\left(\dfrac{2\pi}{3} + 2\alpha\right).$

**26.099.** $\dfrac{4\sin^2(\alpha - 5\pi) - \sin^2(2\alpha + \pi)}{\cos^2\left(2\alpha - \dfrac{3\pi}{2}\right) - 4 + 4\sin^2\alpha}.$

**26.100.** $\sin^2\left(\dfrac{9\pi}{8} + \alpha\right) - \sin^2\left(\dfrac{17\pi}{8} - \alpha\right).$

**26.101.** $\operatorname{ctg}(4\alpha - \pi)\left(\cos^4\left(\dfrac{5\pi}{4} - 2\alpha\right) - \sin^4\left(\dfrac{9\pi}{4} - 2\alpha\right)\right).$

**26.102.** $\dfrac{\cos^2\left(\dfrac{5\pi}{4} - 2\alpha\right) - \sin^2\left(\dfrac{5\pi}{4} - 2\alpha\right)}{\left(\cos\dfrac{\alpha}{2} + \sin\dfrac{\alpha}{2}\right)\left(\cos\left(2\pi - \dfrac{\alpha}{2}\right) + \cos\left(\dfrac{\pi}{2} + \dfrac{\alpha}{2}\right)\right)\sin\alpha} \cdot \operatorname{tg}\left(\dfrac{5\pi}{2} - \alpha\right)(1 + \sin 2\alpha).$

**26.103.** $\dfrac{\operatorname{tg}\left(\dfrac{5\pi}{2} - \alpha\right)(1 + \sin 2\alpha)}{\cos\left(\dfrac{5\pi}{2} - 2\alpha\right)}.$

**26.104.** $\dfrac{\operatorname{tg} 2\alpha}{\operatorname{tg} 4\alpha - \operatorname{tg} 2\alpha}.$

**26.108.** $\dfrac{4\sin\left(\dfrac{5\pi}{2} + \alpha\right)}{\operatorname{tg}^2\left(\dfrac{7\pi}{2} - \dfrac{\alpha}{2}\right) - \operatorname{ctg}^2\left(\dfrac{3\pi}{2} + \dfrac{\alpha}{2}\right)}.$
