# Группа В. Олимпиадные задачи (4.396–4.430)

> **Класс:** 10–11 | **Сложность:** высокая (олимпиадная) | **Тип:** нестандартные тригонометрические тождества и вычисления

---

## Подсказка

Задачи этой группы требуют владения всем арсеналом тригонометрических формул, а также умения:
- доказывать формулы для сумм $n$ слагаемых (метод телескопирования, метод математической индукции);
- работать с произведениями тригонометрических функций от специальных углов;
- комбинировать несколько приёмов в одной задаче.

---

## Доказать тождества (4.396–4.409)

**26.396.** $\cos\alpha + \cos(\alpha + \beta) + \cos(\alpha + 2\beta) + \ldots + \cos(\alpha + n\beta) = \dfrac{\sin\dfrac{(n+1)\beta}{2}}{\sin\dfrac{\beta}{2}}\cos\left(\alpha + \dfrac{n\beta}{2}\right).$

**26.397.** $\sin\alpha + \sin(\alpha + \beta) + \sin(\alpha + 2\beta) + \ldots + \sin(\alpha + n\beta) = \dfrac{\sin\dfrac{(n+1)\beta}{2}}{\sin\dfrac{\beta}{2}}\sin\left(\alpha + \dfrac{n\beta}{2}\right).$

**26.398.** $\cos\dfrac{\pi}{2n+1}\cos\dfrac{2\pi}{2n+1}\cos\dfrac{4\pi}{2n+1}\cdots\cos\dfrac{2^{n-1}\pi}{2n+1} = \dfrac{1}{2^n}.$

**26.399.** $\sin\alpha\sin 2\alpha\sin 4\alpha\cdots\sin 2^{n-1}\alpha = \dfrac{\sin 2^n\alpha}{2^n}.$

**26.400.** $\cos\alpha\cos 2\alpha\cos 4\alpha\cdots\cos 2^{n-1}\alpha = \dfrac{\sin 2^n\alpha}{2^n\sin\alpha}.$

**26.401.** $\operatorname{tg}\alpha + 2\operatorname{tg} 2\alpha + 4\operatorname{tg} 4\alpha + \ldots + 2^{n-1}\operatorname{tg} 2^{n-1}\alpha = \operatorname{ctg}\alpha - 2^n\operatorname{ctg} 2^n\alpha.$

**26.402.** $\dfrac{1}{\sin 2\alpha} + \dfrac{1}{\sin 4\alpha} + \dfrac{1}{\sin 8\alpha} + \ldots + \dfrac{1}{\sin 2^n\alpha} = \operatorname{ctg}\alpha - \operatorname{ctg} 2^n\alpha.$

**26.403.** $\operatorname{tg}\alpha\operatorname{tg} 2\alpha + \operatorname{tg} 2\alpha\operatorname{tg} 3\alpha + \ldots + \operatorname{tg} n\alpha\operatorname{tg}(n+1)\alpha = \operatorname{ctg}\alpha\operatorname{tg}(n+1)\alpha - (n+1).$

**26.404.** $\dfrac{1}{\cos\alpha\cos 2\alpha} + \dfrac{1}{\cos 2\alpha\cos 3\alpha} + \ldots + \dfrac{1}{\cos n\alpha\cos(n+1)\alpha} = \dfrac{\operatorname{tg}(n+1)\alpha - \operatorname{tg}\alpha}{\sin\alpha}.$

**26.405.** $\sin^2\alpha + \sin^2 2\alpha + \sin^2 3\alpha + \ldots + \sin^2 n\alpha = \dfrac{n}{2} - \dfrac{\sin(n+1)\alpha\sin n\alpha}{2\sin\alpha}.$

**26.406.** $\cos^2\alpha + \cos^2 2\alpha + \cos^2 3\alpha + \ldots + \cos^2 n\alpha = \dfrac{n}{2} + \dfrac{\sin(n+1)\alpha\sin n\alpha}{2\sin\alpha}.$

**26.407.** $\operatorname{ctg}^2\dfrac{\pi}{2n+1} + \operatorname{ctg}^2\dfrac{2\pi}{2n+1} + \ldots + \operatorname{ctg}^2\dfrac{n\pi}{2n+1} = \dfrac{n(2n-1)}{3}.$

**26.408.** $\cos^{-2}\dfrac{\pi}{2n+1} + \cos^{-2}\dfrac{2\pi}{2n+1} + \ldots + \cos^{-2}\dfrac{n\pi}{2n+1} = \dfrac{n(2n+2)}{3}.$

**26.409.** $\sin^{2n}\alpha + \cos^{2n}\alpha \le 1$ для любого натурального $n$ и любого $\alpha$.

## Упростить (4.410–4.412)

**26.410.** $\left(1 + \cos\dfrac{\pi}{8}\right)\left(1 + \cos\dfrac{3\pi}{8}\right)\left(1 + \cos\dfrac{5\pi}{8}\right)\left(1 + \cos\dfrac{7\pi}{8}\right).$

**26.411.** $\cos\dfrac{2\pi}{7} + \cos\dfrac{4\pi}{7} + \cos\dfrac{6\pi}{7}.$

**26.412.** $\cos^2\dfrac{\pi}{5} + \cos^2\dfrac{2\pi}{5}.$

---

## Преобразовать в произведение (4.413–4.415)

**26.413.** $\sin\alpha + \sin\beta + \sin\gamma - \sin(\alpha + \beta + \gamma).$

**26.414.** $\cos\alpha + \cos\beta + \cos\gamma + \cos(\alpha + \beta + \gamma).$

**26.415.** $\operatorname{tg}\alpha + \operatorname{tg}\beta + \operatorname{tg}\gamma - \operatorname{tg}\alpha\operatorname{tg}\beta\operatorname{tg}\gamma$ при условии $\alpha + \beta + \gamma = \pi.$

---

## Доказать справедливость равенств (4.416–4.430)

**26.416.** $\sin^2 20° + \sin^2 40° + \sin^2 80° = \dfrac{3}{2}.$

**26.417.** $\operatorname{tg} 20°\operatorname{tg} 40°\operatorname{tg} 80° = \sqrt{3}.$

**26.418.** $\cos 20°\cos 40°\cos 80° = \dfrac{1}{8}.$

**26.419.** $\operatorname{ctg} 20°\operatorname{ctg} 40°\operatorname{ctg} 80° = \sqrt{3}.$

**26.420.** $\cos\dfrac{\pi}{7}\cos\dfrac{2\pi}{7}\cos\dfrac{3\pi}{7} = \dfrac{1}{8}.$

**26.421.** $\operatorname{tg}\dfrac{\pi}{7}\operatorname{tg}\dfrac{2\pi}{7}\operatorname{tg}\dfrac{3\pi}{7} = \sqrt{7}.$

**26.422.** $\cos\dfrac{\pi}{9}\cos\dfrac{2\pi}{9}\cos\dfrac{4\pi}{9} = \dfrac{1}{8}.$

**26.423.** $\sin 10°\sin 50°\sin 70° = \dfrac{1}{8}.$

**26.424.** $\cos 36° - \cos 72° = \dfrac{1}{2}.$

**26.425.** $\cos 36°\cos 72° = \dfrac{1}{4}.$

**26.426.** $\operatorname{ctg}^2\dfrac{\pi}{7} + \operatorname{ctg}^2\dfrac{2\pi}{7} + \operatorname{ctg}^2\dfrac{3\pi}{7} = 5.$

**26.427.** $\cos^{-2}\dfrac{\pi}{7} + \cos^{-2}\dfrac{2\pi}{7} + \cos^{-2}\dfrac{3\pi}{7} = 8.$

**26.428.** $\cos^2\dfrac{\pi}{7} + \cos^2\dfrac{2\pi}{7} + \cos^2\dfrac{3\pi}{7} = \dfrac{5}{4}.$

**26.429.** $\cos\dfrac{\pi}{11} + \cos\dfrac{3\pi}{11} + \cos\dfrac{5\pi}{11} + \cos\dfrac{7\pi}{11} + \cos\dfrac{9\pi}{11} = \dfrac{1}{2}.$

**26.430.** $\sin\dfrac{\pi}{7} + \sin\dfrac{2\pi}{7} + \sin\dfrac{3\pi}{7} = \dfrac{\sqrt{7}}{2}.$
