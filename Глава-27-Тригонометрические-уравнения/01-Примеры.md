# Примеры решения тригонометрических уравнений

> 9 подробных разборов, охватывающих все основные методы

---

## Пример 1. Простейшее уравнение

**Задача.** $\sin\left(\dfrac{\pi}{6} - 2x\right) = \dfrac{\sqrt{3}}{2}.$

**Решение.**

По формуле $\sin u = a \Rightarrow u = (-1)^n \arcsin a + \pi n$:

$$\frac{\pi}{6} - 2x = (-1)^n \frac{\pi}{3} + \pi n.$$

$$2x = \frac{\pi}{6} - (-1)^n \frac{\pi}{3} - \pi n.$$

$$x = \frac{\pi}{12} - (-1)^n \frac{\pi}{6} - \frac{\pi n}{2}.$$

Выпишем серии:
- При чётных $n$: $x = \dfrac{\pi}{12} - \dfrac{\pi}{6} - \dfrac{\pi n}{2} = -\dfrac{\pi}{12} - \dfrac{\pi n}{2}.$
- При нечётных $n$: $x = \dfrac{\pi}{12} + \dfrac{\pi}{6} - \dfrac{\pi n}{2} = \dfrac{\pi}{4} - \dfrac{\pi n}{2}.$

**Ответ:** $x = (-1)^{n+1}\dfrac{\pi}{6} + \dfrac{\pi}{12} + \dfrac{\pi n}{2},\ n \in \mathbb{Z}.$

## Пример 2. Разложение на множители

**Задача.** $\sin x = \sin 2x\cos 3x.$

**Решение.**

Применяем формулу произведения в сумму: $\sin 2x\cos 3x = \dfrac{1}{2}[\sin 5x + \sin(-x)] = \dfrac{1}{2}(\sin 5x - \sin x).$

Уравнение: $\sin x = \dfrac{1}{2}\sin 5x - \dfrac{1}{2}\sin x.$

$$\frac{3}{2}\sin x - \frac{1}{2}\sin 5x = 0 \quad\Longrightarrow\quad 3\sin x - \sin 5x = 0.$$

Преобразуем сумму синусов. Запишем $\sin 5x = \sin x\cdot(\text{формула через кратные углы})$. Альтернативно:

$3\sin x - \sin 5x = 3\sin x - (16\sin^5 x - 20\sin^3 x + 5\sin x) = -2\sin x - 16\sin^5 x + 20\sin^3 x.$

Проще: перегруппируем.

$\sin x + \sin x + \sin x - \sin 5x = 0.$

Используем $\sin x - \sin 5x = -2\cos 3x\sin 2x$:

$2\sin x - 2\cos 3x\sin 2x = 0$, $2\sin x(1 - 2\cos 3x\cos x) = 0$... — путь усложняется.

Вернёмся к исходному: $\sin x = \sin 2x\cos 3x$, перенесём: $\sin x - \sin 2x\cos 3x = 0.$

Запишем $\sin 2x = 2\sin x\cos x$: $\sin x(1 - 2\cos x\cos 3x) = 0.$

1) $\sin x = 0 \Rightarrow x = \pi n.$

2) $1 - 2\cos x\cos 3x = 0.$ Формула: $2\cos x\cos 3x = \cos 2x + \cos 4x.$ Получаем $\cos 2x + \cos 4x = 1$, т.е. $2\cos 3x\cos x = 1$ — уже имеем.

$\cos 4x + \cos 2x - 1 = 0.$ Замена $\cos 2x = t$: $2t^2 - 1 + t - 1 = 0$, $2t^2 + t - 2 = 0.$

$t = \dfrac{-1 \pm\sqrt{17}}{4}.$ Значение $t_1 = \dfrac{-1+\sqrt{17}}{4} \approx 0{,}78$ — подходит. Значение $t_2 = \dfrac{-1-\sqrt{17}}{4} < -1$ — не подходит.

$\cos 2x = \dfrac{-1+\sqrt{17}}{4}$, $x = \pm\dfrac{1}{2}\arccos\dfrac{\sqrt{17}-1}{4} + \pi k.$

**Ответ:** $x = \pi n;\ x = \pm\dfrac{1}{2}\arccos\dfrac{\sqrt{17}-1}{4} + \pi k,\ n, k \in \mathbb{Z}.$

## Пример 3. Проверка посторонних корней

**Задача.** $(1 - \sin x)(\operatorname{tg}^2 x - 3) = 0.$

**Решение.**

Совокупность: $\sin x = 1$ или $\operatorname{tg}^2 x = 3$.

Из $\sin x = 1$: $x = \pi/2 + 2\pi n$. Но при этих $x$ функция $\operatorname{tg} x$ не определена, и $\operatorname{tg}^2 x$ в исходном уравнении теряет смысл. Поэтому **первая серия — посторонняя**.

Из $\operatorname{tg} x = \pm\sqrt{3}$: $x = \pm\pi/3 + \pi n$.

**Ответ:** $x = \pm\dfrac{\pi}{3} + \pi n,\ n \in \mathbb{Z}.$

## Пример 4. Однородное уравнение

**Задача.** $3\sin^2 x\cos\left(\dfrac{3\pi}{2} + x\right) + 3\sin^2 x\cos x - \sin x\cos^2 x - \cos^3 x = 0.$

**Решение.**

Упростим: $\cos\left(\dfrac{3\pi}{2} + x\right) = \sin x.$

$$3\sin^3 x + 3\sin^2 x\cos x - \sin x\cos^2 x - \cos^3 x = 0.$$

Это однородное уравнение 3-й степени. Проверим $\cos x = 0$: $3\sin^3 x = 0$, т.е. $\sin x = 0$, но тогда $\cos x \ne 0$ — противоречие. Значит $\cos x \ne 0$.

Делим на $\cos^3 x$:

$$3\operatorname{tg}^3 x + 3\operatorname{tg}^2 x - \operatorname{tg} x - 1 = 0.$$

Группируем: $3\operatorname{tg}^2 x(\operatorname{tg} x + 1) - (\operatorname{tg} x + 1) = 0$, $(\operatorname{tg} x + 1)(3\operatorname{tg}^2 x - 1) = 0.$

1) $\operatorname{tg} x = -1 \Rightarrow x = -\pi/4 + \pi n.$

2) $\operatorname{tg} x = \pm\dfrac{1}{\sqrt{3}} \Rightarrow x = \pm\dfrac{\pi}{6} + \pi n.$

**Ответ:** $x = -\dfrac{\pi}{4} + \pi n;\ x = \pm\dfrac{\pi}{6} + \pi n,\ n \in \mathbb{Z}.$

## Пример 5. Умножение обеих частей

**Задача.** $\cos x\cos 2x\cos 4x = \dfrac{1}{8}.$

**Решение.**

Умножим обе части на $8\sin x$:

$$8\sin x\cos x\cos 2x\cos 4x = \sin x.$$

По формуле $2\sin\alpha\cos\alpha = \sin 2\alpha$ применяем трижды:

$8\sin x\cos x = 4\sin 2x$, $4\sin 2x\cos 2x = 2\sin 4x$, $2\sin 4x\cos 4x = \sin 8x.$

Получили: $\sin 8x = \sin x.$

$8x = (-1)^n x + \pi n.$ При $n = 2k$: $7x = 2\pi k$, $x = \dfrac{2\pi k}{7}.$ При $n = 2k+1$: $9x = \pi(2k+1)$, $x = \dfrac{\pi(2k+1)}{9}.$

Исключаем значения, при которых $\sin x = 0$: $x = \pi m$, т.е. $k$ кратно 7 (первая серия) или $2k+1$ кратно 9 (вторая).

**Ответ:** $x = \dfrac{2\pi k}{7}$ ($k \ne 7m$) и $x = \dfrac{\pi(2k+1)}{9}$ ($2k+1 \ne 9m$), $k, m \in \mathbb{Z}.$

## Пример 6. Универсальная подстановка

**Задача.** $3\cos^2\left(\dfrac{x - \pi/2}{2}\right) - 2\cos x = 4.$

**Решение.**

Раскроем: $\cos^2\left(\dfrac{x}{2} - \dfrac{\pi}{4}\right) = \dfrac{1 + \cos(x - \pi/2)}{2} = \dfrac{1 + \sin x}{2}.$

Уравнение: $3 \cdot \dfrac{1 + \sin x}{2} - 2\cos x = 4$, т.е. $3 + 3\sin x - 4\cos x = 8$, $3\sin x - 4\cos x = 5.$

Метод вспомогательного угла: $\sqrt{9 + 16} = 5$, значит $5\sin(x + \varphi) = 5$, $\sin(x + \varphi) = 1$, где $\cos\varphi = 3/5$, $\sin\varphi = -4/5$.

$x + \varphi = \pi/2 + 2\pi n$, $x = \pi/2 - \varphi + 2\pi n$.

Так как $\varphi = -\arcsin(4/5) = \operatorname{arctg}(-4/3)$, получаем $x = \pi/2 + \arcsin(4/5) + 2\pi n$.

Можно также: воспользуемся подстановкой $\operatorname{tg}(x/2) = t$.

$\sin x = \dfrac{2t}{1+t^2}$, $\cos x = \dfrac{1-t^2}{1+t^2}.$

$\dfrac{6t}{1+t^2} - \dfrac{4(1-t^2)}{1+t^2} = 5$, $6t - 4 + 4t^2 = 5 + 5t^2$, $t^2 - 6t + 9 = 0$, $(t-3)^2 = 0$, $t = 3.$

$\operatorname{tg}(x/2) = 3$, $x = 2\operatorname{arctg} 3 + 2\pi n.$

Проверим $x = \pi + 2\pi n$ (точка, где $\operatorname{tg}(x/2)$ не определён): $3(-1) - 4(-1) = 1 \ne 5$ — не подходит.

**Ответ:** $x = 2\operatorname{arctg} 3 + 2\pi n,\ n \in \mathbb{Z}.$

## Пример 7. Уравнение $\sin^6 x + \cos^6 x = a$

**Задача.** Решить уравнение $\sin^6 x + \cos^6 x = a$ ($a$ — заданное число).

**Решение.**

По формуле суммы кубов:

$\sin^6 x + \cos^6 x = (\sin^2 x + \cos^2 x)^3 - 3\sin^2 x\cos^2 x(\sin^2 x + \cos^2 x) = 1 - 3\sin^2 x\cos^2 x.$

Так как $\sin x\cos x = \dfrac{1}{2}\sin 2x$:

$1 - \dfrac{3}{4}\sin^2 2x = a$, $\sin^2 2x = \dfrac{4(1-a)}{3}.$

Уравнение имеет решение при $0 \le \dfrac{4(1-a)}{3} \le 1$, т.е. $\dfrac{1}{4} \le a \le 1$.

$\sin 2x = \pm\sqrt{\dfrac{4(1-a)}{3}}$,

$x = \pm\dfrac{1}{2}\arcsin\sqrt{\dfrac{4(1-a)}{3}} + \dfrac{\pi n}{2},\ n \in \mathbb{Z}.$

В частности, при $a = 1$: $x = \pi n/2$.

## Пример 8. Уравнение $\sin^{2k+2} x + \cos^{2k+2} x = 1$

**Задача.** Решить уравнение $\sin^{2k+2} x + \cos^{2k+2} x = 1$ ($k \in \mathbb{N}$).

**Решение.**

Числа $x = \pi n/2$ ($n \in \mathbb{Z}$) являются решениями (один из синуса/косинуса равен $\pm 1$, другой 0).

Покажем, что других решений нет. Если $\sin x \ne 0$ и $\cos x \ne 0$, то $|\sin x| < 1$ и $|\cos x| < 1$. Тогда:

$\sin^{2k+2} x < \sin^2 x$ и $\cos^{2k+2} x < \cos^2 x,$

поэтому $\sin^{2k+2} x + \cos^{2k+2} x < \sin^2 x + \cos^2 x = 1.$

**Ответ:** $x = \dfrac{\pi n}{2},\ n \in \mathbb{Z}.$

## Пример 9. Уравнение $\sin(\pi\cos 2x) = 1$

**Задача.** $\sin(\pi\cos 2x) = 1.$

**Решение.**

$\pi\cos 2x = \dfrac{\pi}{2} + 2\pi k$, т.е. $\cos 2x = \dfrac{1}{2} + 2k$, $k \in \mathbb{Z}$.

Так как $|\cos 2x| \le 1$, единственное допустимое значение: $k = 0$, $\cos 2x = 1/2$.

$2x = \pm\dfrac{\pi}{3} + 2\pi n$, $x = \pm\dfrac{\pi}{6} + \pi n.$

**Ответ:** $x = \pm\dfrac{\pi}{6} + \pi n,\ n \in \mathbb{Z}.$
