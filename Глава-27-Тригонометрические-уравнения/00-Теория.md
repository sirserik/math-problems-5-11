# Глава 5. Тригонометрические уравнения — Теория

> **Классы:** 9–11 | Основные методы решения тригонометрических уравнений

---

## 1. Простейшие тригонометрические уравнения

### Общие формулы

| Уравнение | Решение | Условие |
|-----------|---------|---------|
| $\sin x = a$ | $x = (-1)^n \arcsin a + \pi n,\ n \in \mathbb{Z}$ | $\|a\| \le 1$ |
| $\cos x = a$ | $x = \pm \arccos a + 2\pi n,\ n \in \mathbb{Z}$ | $\|a\| \le 1$ |
| $\operatorname{tg} x = a$ | $x = \operatorname{arctg} a + \pi n,\ n \in \mathbb{Z}$ | $a \in \mathbb{R}$ |
| $\operatorname{ctg} x = a$ | $x = \operatorname{arcctg} a + \pi n,\ n \in \mathbb{Z}$ | $a \in \mathbb{R}$ |

### Частные случаи

| Уравнение | Решение |
|-----------|---------|
| $\sin x = 0$ | $x = \pi n$ |
| $\sin x = 1$ | $x = \pi/2 + 2\pi n$ |
| $\sin x = -1$ | $x = -\pi/2 + 2\pi n$ |
| $\cos x = 0$ | $x = \pi/2 + \pi n$ |
| $\cos x = 1$ | $x = 2\pi n$ |
| $\cos x = -1$ | $x = \pi + 2\pi n$ |
| $\operatorname{tg} x = 0$ | $x = \pi n$ |
| $\operatorname{ctg} x = 0$ | $x = \pi/2 + \pi n$ |

> Во всех формулах $n \in \mathbb{Z}$.

## 2. Основные методы решения

### Метод 1. Сведение к простейшему

Уравнения вида $\sin(\omega x + \varphi) = a$, $\cos(\omega x + \varphi) = a$, $\operatorname{tg}(\omega x + \varphi) = b$, $\operatorname{ctg}(\omega x + \varphi) = b$ решаются непосредственно по формулам, заменив $x$ на $\omega x + \varphi$.

**Пример.** $\sin\left(\dfrac{\pi}{6} - 2x\right) = \dfrac{\sqrt{3}}{2}.$

Решение: $\dfrac{\pi}{6} - 2x = (-1)^n \arcsin\dfrac{\sqrt{3}}{2} + \pi n = (-1)^n \dfrac{\pi}{3} + \pi n.$

Так как $\arcsin\dfrac{\sqrt{3}}{2} = \dfrac{\pi}{3}$, получаем $x = (-1)^{n+1}\dfrac{\pi}{6} + \dfrac{\pi}{12} + \dfrac{\pi n}{2}.$

---

### Метод 2. Разложение на множители

Если уравнение можно представить в виде произведения, равного нулю, то оно равносильно совокупности уравнений.

**Важно:** при разложении на множители возможно появление посторонних корней. Необходимо проверять, что полученные значения не обращают в ноль знаменатели или не выводят из области определения.

**Пример.** $(1 - \sin x)(\operatorname{tg}^2 x - 3) = 0.$

Совокупность: $\sin x = 1$ или $\operatorname{tg} x = \pm\sqrt{3}$.

Но при $\sin x = 1$ (т.е. $x = \pi/2 + 2\pi n$) исходное уравнение не имеет смысла, если в нём присутствует $\operatorname{tg} x$. Поэтому первая серия — посторонняя.

---

### Метод 3. Однородные уравнения

**Первой степени:** $a\sin kx + b\cos kx = 0.$

Делим на $\cos kx \ne 0$: $a\operatorname{tg} kx + b = 0$, $\operatorname{tg} kx = -b/a$.

**Второй степени:** $a\sin^2 kx + b\sin kx\cos kx + c\cos^2 kx = 0.$

Делим на $\cos^2 kx$: $a\operatorname{tg}^2 kx + b\operatorname{tg} kx + c = 0$.

> **Проверка:** значения $x$, при которых $\cos kx = 0$, нужно проверить отдельно подстановкой в исходное уравнение.

**Третьей степени:** $a\sin^3 kx + b\sin^2 kx\cos kx + c\sin kx\cos^2 kx + d\cos^3 kx = 0.$

Делим на $\cos^3 kx$: получаем кубическое уравнение относительно $\operatorname{tg} kx$.

**Приводимое к однородному:** уравнение вида

$$a\sin^2 kx + b\sin kx\cos kx + c\cos^2 kx = d$$

при $d \ne 0$ можно привести к однородному, заменив $d$ на $d(\sin^2 kx + \cos^2 kx)$.

---

### Метод 4. Понижение степени

Используются формулы:
- $\sin^2 x = \dfrac{1 - \cos 2x}{2}$, $\cos^2 x = \dfrac{1 + \cos 2x}{2}$
- $\sin x\cos x = \dfrac{\sin 2x}{2}$

Полезны для уравнений с чётными степенями тригонометрических функций.

---

### Метод 5. Универсальная подстановка

Замена $\operatorname{tg}(x/2) = t$ приводит к:

$$\sin x = \frac{2t}{1 + t^2},\quad \cos x = \frac{1 - t^2}{1 + t^2},\quad \operatorname{tg} x = \frac{2t}{1 - t^2}.$$

> **Ограничение:** замена $\operatorname{tg}(x/2) = t$ справедлива для всех $x \ne \pi + 2\pi n$. Значения $x = \pi + 2\pi n$ нужно проверять отдельно подстановкой в исходное уравнение.

---

### Метод 6. Введение вспомогательного угла

Уравнение $a\sin x + b\cos x = c$ приводится к виду:

$$\sqrt{a^2 + b^2}\sin(x + \varphi) = c,$$

где $\sin\varphi = \dfrac{b}{\sqrt{a^2+b^2}}$, $\cos\varphi = \dfrac{a}{\sqrt{a^2+b^2}}$.

Уравнение имеет решение тогда и только тогда, когда $c^2 \le a^2 + b^2$.

---

### Метод 7. Преобразование суммы в произведение

Формулы:
- $\sin\alpha + \sin\beta = 2\sin\dfrac{\alpha+\beta}{2}\cos\dfrac{\alpha-\beta}{2}$
- $\sin\alpha - \sin\beta = 2\cos\dfrac{\alpha+\beta}{2}\sin\dfrac{\alpha-\beta}{2}$
- $\cos\alpha + \cos\beta = 2\cos\dfrac{\alpha+\beta}{2}\cos\dfrac{\alpha-\beta}{2}$
- $\cos\alpha - \cos\beta = -2\sin\dfrac{\alpha+\beta}{2}\sin\dfrac{\alpha-\beta}{2}$

Позволяют свести уравнение к произведению и применить метод 2.

---

### Метод 8. Преобразование произведения в сумму

Формулы:
- $2\sin\alpha\cos\beta = \sin(\alpha+\beta) + \sin(\alpha-\beta)$
- $2\cos\alpha\cos\beta = \cos(\alpha-\beta) + \cos(\alpha+\beta)$
- $2\sin\alpha\sin\beta = \cos(\alpha-\beta) - \cos(\alpha+\beta)$

---

## 3. Типичные ошибки

1. **Деление на выражение, обращающееся в ноль.** При делении обеих частей на $\sin x$, $\cos x$ и т.п. можно потерять корни. Нужно проверить отдельно случай, когда делитель равен нулю.

2. **Посторонние корни при умножении.** Умножение обеих частей на выражение (например, на $\sin x$) может внести посторонние корни. Необходима проверка.

3. **Неправильная запись ответа.** Общая формула должна содержать все серии решений. Не забывайте объединять серии, если это возможно.

4. **Потеря корней при извлечении корня.** $\sin^2 x = a$ даёт $\sin x = \pm\sqrt{a}$, а не только $\sin x = \sqrt{a}$.
