## Методы решения

- **Сведение к простейшему** — прямое применение формул корней
- **Разложение на множители** — с проверкой посторонних корней
- **Однородные уравнения** — деление на $\cos^n x$
- **Понижение степени** — формулы двойного угла
- **Универсальная подстановка** — $\operatorname{tg}(x/2) = t$
- **Вспомогательный угол** — $a\sin x + b\cos x = c$
- **Преобразование сумм** — в произведения и обратно
- **Специальные приёмы** — бесконечные ряды, корни, логарифмы, параметры
