# Группа А. Уравнения со степенями и кратными углами (27.051–27.100)

> **Класс:** 9–10 | **Сложность:** базовая–средняя | **Тип:** уравнения со степенями тригонометрических функций, кратными углами

## Подсказка

Используйте формулы понижения степени, формулы кратного угла, основное тригонометрическое тождество. Часто помогает замена $\cos 2x = 1 - 2\sin^2 x$ или $\cos 2x = 2\cos^2 x - 1$.

---

## Задачи

**27.051.** $\sin 3x + \sin 5x = 2\,(\cos^2 2x - \sin^2 3x).$

**27.052.** $0{,}5\,(\cos 5x + \cos 7x) - \cos^2 2x + \sin^2 3x = 0.$

**27.053.** $1 + \sin x - \cos 5x - \sin 7x = 2\cos^2\dfrac{3x}{2}.$

**27.054.** $\cos 2x + \cos 6x + 2\sin^2 x = 1.$

**27.055.** $\cos^2 3x + \cos^2 4x + \cos^2 5x = 1{,}5.$

**27.056.** $\cos^2\dfrac{x}{2} + \cos^2\dfrac{3x}{2} - \sin^2 2x - \sin^2 4x = 0.$

**27.057.** $\cos^2 x + \cos^2 2x - \cos^2 3x - \cos^2 4x = 0.$

**27.058.** $\sin^2 3x + \sin^2 4x = \sin^2 5x + \sin^2 6x.$

**27.059.** $\sin^2 2x + \sin^2 3x + \sin^2 4x + \sin^2 5x = 2.$

**27.060.** $\sin 7x + \sin 9x = 2\left(\cos^2\left(\dfrac{\pi}{4} - x\right) - \cos^2\left(\dfrac{\pi}{4} + 2x\right)\right).$

**27.061.** $8\cos x\cos(60° - x)\cos(60° + x) + 1 = 0.$

**27.062.** $\sin x\sin(60° - x)\sin(60° + x) = 1/8.$

**27.063.** $\sin 2x\sin 6x = \cos x\cos 3x.$

**27.064.** $\sin x\sin 3x + \sin 4x\sin 8x = 0.$

**27.065.** $\sin 3x\cos 3x = \sin 2x.$

**27.066.** $\cos 3x\cos 6x = \cos 4x\cos 7x.$

**27.067.** $\cos(x + 1)\sin 2(x + 1) = \cos 3(x + 1)\sin 4(x + 1).$

**27.068.** $\operatorname{tg} 3t - \operatorname{tg} t - 4\sin t = 0.$

**27.069.** $\sqrt{3} - \operatorname{tg} x = \operatorname{tg}\left(\dfrac{\pi}{3} - x\right).$

**27.070.** $(\cos 6x - 1)\operatorname{ctg} 3x = \sin 3x.$

**27.071.** $\cos^{-1} x + \operatorname{ctg} 3x = \operatorname{ctg}\dfrac{3x}{2}.$

**27.072.** $\cos x\cos 2x = \sin\left(\dfrac{\pi}{4} + x\right)\sin\left(\dfrac{\pi}{4} + 4x\right) + \sin\left(\dfrac{3\pi}{4} + 4x\right)\cos\left(\dfrac{7\pi}{4} - 5x\right).$

**27.073.** $\sin 3x - 4\sin x\cos 2x = 0.$

**27.074.** $\sin\dfrac{7x}{2}\cos\dfrac{3x}{2} + \sin\dfrac{x}{2}\cos\dfrac{5x}{2} + \sin 2x\cos 7x = 0.$

**27.075.** $\cos\dfrac{x}{2}\cos\dfrac{3x}{2} - \sin x\sin 3x - \sin 2x\sin 3x = 0.$

**27.076.** $\sin\left(\dfrac{\pi}{4} + 5x\right)\cos\left(\dfrac{\pi}{4} + 2x\right) = \sin\left(\dfrac{\pi}{4} + x\right)\sin\left(\dfrac{\pi}{4} - 6x\right).$

**27.077.** $\sin x\cos 2x + \cos x\cos 4x = \sin\left(\dfrac{\pi}{4} + 2x\right)\sin\left(\dfrac{\pi}{4} - 3x\right).$

**27.078.** $\operatorname{ctg} t - \sin t = 2\sin^2\dfrac{t}{2}.$

**27.079.** $2\left(\operatorname{tg}\dfrac{t}{2} - 1\right) = \cos t.$

**27.080.** $1 + \sin 2x = \sin x + \cos x.$

**27.081.** $\cos 2x = \sqrt{2}\,(\cos x - \sin x).$

**27.082.** $2\operatorname{ctg}^2 x\cos^2 x + 4\cos^2 x - \operatorname{ctg}^2 x - 2 = 0.$

**27.083.** $\sin^2 x\cos^{-4} x - 4\operatorname{tg}^2 x + 3\cos^{-2} x - 12 = 0.$

**27.084.** $\cos^3 x + \cos^2 x - 4\cos^2\dfrac{x}{2} = 0.$

**27.085.** $2\operatorname{tg}^3 x - 2\operatorname{tg}^2 x + 3\operatorname{tg} x - 3 = 0.$

**27.086.** $\cos 4x + 2\sin^2 x = 0.$

**27.087.** $3\sin^2 2x + 7\cos 2x - 3 = 0.$

**27.088.** $6\operatorname{ctg}^2 x - 2\cos^2 x = 3.$

**27.089.** $\cos 2x - 5\sin x - 3 = 0.$

**27.090.** $\cos(4x + 2) + 3\sin(2x + 1) = 2.$

**27.091.** $2\cos^2 x + 5\sin x - 4 = 0.$

**27.092.** $6\sin^2 x + 2\sin^2 2x = 5.$

**27.093.** $25\sin^2 x + 100\cos x = 89.$

**27.094.** $\cos 2x + \sin^2 x + \sin x = 0{,}25.$

**27.095.** $\cos 2x + 3\sin x = 2.$

**27.096.** $2\operatorname{tg}^4 3x - 3\operatorname{tg}^2 3x + 1 = 0.$

**27.097.** $\cos^4 2x + 6\cos^2 2x = 25/16.$

**27.098.** $\operatorname{ctg}^4 2x + \sin^{-4} 2x = 25.$

**27.099.** $2\sin^2 x + \operatorname{tg}^2 x = 2.$

**27.100.** $2 + \operatorname{tg} x\operatorname{ctg}\dfrac{x}{2} + \operatorname{ctg} x\operatorname{tg}\dfrac{x}{2} = 0.$
