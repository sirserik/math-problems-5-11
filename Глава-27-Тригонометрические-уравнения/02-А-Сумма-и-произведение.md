# Группа А. Уравнения на сумму/произведение тригонометрических функций (27.001–27.050)

> **Класс:** 9–10 | **Сложность:** базовая | **Тип:** преобразование сумм в произведения и наоборот

## Подсказка

Большинство уравнений этой группы решаются преобразованием суммы/разности тригонометрических функций в произведение (или наоборот) с последующим разложением на множители.

---

## Задачи

**27.001.** $\cos 3x - \sin x = \sqrt{3}\,(\cos x - \sin 3x).$

**27.002.** $\sin 3x - \cos 3x = \sqrt{3}/2.$

**27.003.** $\sin 2x + \cos 2x = \sqrt{2}\sin 3x.$

**27.004.** $2\,(\cos 4x - \sin x\cos 3x) = \sin 4x + \sin 2x.$

**27.005.** $\sin 2x\sin 6x - \cos 2x\cos 6x = \sqrt{2}\sin 3x\cos 8x.$

**27.006.** $\sin\dfrac{x}{2}\cos\dfrac{3x}{2} - \dfrac{1}{\sqrt{3}}\sin 2x = \sin\dfrac{3x}{2}\cos\dfrac{x}{2}.$

**27.007.** $\operatorname{tg} 2x\cos 3x + \sin 3x + \sqrt{2}\sin 5x = 0.$

**27.008.** $\cos t\sin\left(\dfrac{\pi}{2} + 6t\right) + \cos\left(\dfrac{\pi}{2} - t\right)\sin 6t = \cos 6t + \cos 4t.$

**27.009.** $\sin\left(\dfrac{\pi}{2} + 2x\right)\operatorname{ctg} 3x + \sin(\pi + 2x) - \sqrt{2}\cos 5x = 0.$

**27.010.** $\sin 3x + \dfrac{\sqrt{3}}{2}\sin 5x + \dfrac{1}{2}\cos 5x = 0.$

**27.011.** $\cos 4x\cos(\pi + 2x) - \sin 2x\cos\left(\dfrac{\pi}{2} - 4x\right) = \dfrac{\sqrt{2}}{2}\sin 4x.$

**27.012.** $(\sin 2x + \sqrt{3}\cos 2x)^2 = 2 - 2\cos\left(\dfrac{2\pi}{3} - x\right).$

**27.013.** $\sin 3x + \sin 5x = \sin 4x.$

**27.014.** $\cos x - \cos 3x = \sin 2x.$

**27.015.** $\cos x - \cos 2x = \sin 3x.$

**27.016.** $\cos 5x + \cos 7x = \cos(\pi + 6x).$

**27.017.** $\cos\left(\dfrac{\pi}{2} + 5x\right) + \sin x = 2\cos 3x.$

**27.018.** $\sin x + \sin\dfrac{1}{x} = \sin\left(x + \dfrac{1}{x}\right).$

**27.019.** $\cos x - \sqrt{3}\sin x = \cos 3x.$

**27.020.** $\sqrt{3}\sin 2x + \cos 5x - \cos 9x = 0.$

**27.021.** $\sqrt{2}\cos x + \cos 2x + \cos 4x = 0.$

**27.022.** $\sin 3x - \sin 7x = \sqrt{3}\sin 2x.$

**27.023.** $\sin 2x + \sin(\pi - 8x) = \sqrt{2}\cos 3x.$

**27.024.** $\sin\left(\dfrac{\pi}{2} + 3x\right) - \sin(\pi - 5x) = \sqrt{3}\,(\cos 5x - \sin 3x).$

**27.025.** $\sin(15° + x) + \sin(45° - x) = 1.$

**27.026.** $\cos(20° + x) + \cos(100° - x) = 1/2.$

**27.027.** $\sin(15° + x) + \cos(45° + x) + 0{,}5 = 0.$

**27.028.** $\sin 5x = \cos 4x.$

**27.029.** $\operatorname{tg}(70° + x) + \operatorname{tg}(20° - x) = 2.$

**27.030.** $\sin 6x + \sin 2x = 0{,}5\operatorname{tg} 2x.$

**27.031.** $\operatorname{tg} x + \operatorname{tg} 2x - \operatorname{tg} 3x = 0.$

**27.032.** $\sin 9x = 2\sin 3x.$

**27.033.** $\cos 3x = 2\sin\left(\dfrac{3\pi}{2} + x\right).$

**27.034.** $\sin 3x = 2\cos\left(\dfrac{\pi}{2} - x\right).$

**27.035.** $\cos 6x = 2\sin\left(\dfrac{3\pi}{2} + 2x\right).$

**27.036.** $\sin x + \sin 2x + \sin 3x = \cos x + \cos 2x + \cos 3x.$

**27.037.** $\cos 9x - \cos 7x + \cos 3x - \cos x = 0.$

**27.038.** $\sin x - \sin 2x + \sin 5x + \sin 8x = 0.$

**27.039.** $\cos 7x + \sin 8x = \cos 3x - \sin 2x.$

**27.040.** $\sin x - \sin 3x - \sin 5x + \sin 7x = 0.$

**27.041.** $\sin x + \sin 7x - \cos 5x + \cos(3x - 2\pi) = 0.$

**27.042.** $\sin 2x - \sin 3x + \sin 8x = \cos\left(7x + \dfrac{3\pi}{2}\right).$

**27.043.** $1 - \cos 6x = \operatorname{tg} 3x.$

**27.044.** $(1 + \cos 4x)\sin 2x = \cos^2 2x.$

**27.045.** $1 - \cos(\pi + x) - \sin\dfrac{3\pi + x}{2} = 0.$

**27.046.** $\cos 4x + 2\cos^2 x = 1.$

**27.047.** $1 + \cos t + \cos 2t + \cos 3t = 0.$

**27.048.** $1 + \sin 2x = (\cos 3x + \sin 3x)^2.$

**27.049.** $1 - \sin 3x = \left(\sin\dfrac{x}{2} - \cos\dfrac{x}{2}\right)^2.$

**27.050.** $1 + \cos 7x = \left(\sin\dfrac{3x}{2} - \cos\dfrac{3x}{2}\right)^2.$
