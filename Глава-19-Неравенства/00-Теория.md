# Теория. Неравенства

---

## 1. Числовые неравенства и их свойства

**Число $a$ больше числа $b$**, если разность $a - b$ — положительное число: $a > b \Leftrightarrow a - b > 0$.

**Число $a$ меньше числа $b$**, если разность $a - b$ — отрицательное число: $a < b \Leftrightarrow a - b < 0$.

### Основные свойства неравенств

| № | Свойство | Формула |
|---|----------|---------|
| 1 | **Транзитивность** | Если $a > b$ и $b > c$, то $a > c$ |
| 2 | **Прибавление числа** | Если $a > b$, то $a + c > b + c$ для любого $c \in \mathbb{R}$ |
| 3 | **Умножение на положительное** | Если $a > b$ и $c > 0$, то $ac > bc$ |
| 4 | **Умножение на отрицательное** | Если $a > b$ и $c < 0$, то $ac < bc$ (знак меняется!) |
| 5 | **Сложение неравенств** | Если $a > b$ и $c > d$, то $a + c > b + d$ |
| 6 | **Умножение неравенств** | Если $a > b > 0$ и $c > d > 0$, то $ac > bd$ |
| 7 | **Возведение в степень** | Если $a > b > 0$ и $n \in \mathbb{N}$, то $a^n > b^n$ |

> **Главное правило:** при умножении (делении) обеих частей неравенства на **отрицательное** число знак неравенства **меняется на противоположный**.

**Пример сложения неравенств:**
$$\text{Если } a > b \text{ и } c > d, \text{ то } a + c > b + d$$
$$\text{Если } 5 > 3 \text{ и } 7 > 2, \text{ то } 5 + 7 > 3 + 2, \text{ т.е. } 12 > 5 \quad \checkmark$$

## 2. Линейные неравенства с одной переменной

**Линейное неравенство** с одной переменной — неравенство вида:
$$ax + b > 0, \quad ax + b < 0, \quad ax + b \geq 0, \quad ax + b \leq 0$$

**Алгоритм решения линейного неравенства:**

1. Перенести все слагаемые с $x$ в левую часть, числа — в правую.
2. Привести подобные слагаемые.
3. Разделить обе части на коэффициент при $x$.
4. **Если делим на отрицательное число — знак неравенства меняется!**

**Пример:** Решить $3(x + 2) - 5 > 4 - 2x$

$$3x + 6 - 5 > 4 - 2x$$
$$3x + 2x > 4 - 1$$
$$5x > 3$$
$$x > \frac{3}{5}$$

**Ответ:** $x \in \left(\dfrac{3}{5};\, +\infty\right)$

## 3. Числовые промежутки

| Обозначение | Название | Запись неравенством | Геометрически |
|-------------|----------|---------------------|---------------|
| $(a;\, b)$ | Интервал | $a < x < b$ | Без концов |
| $[a;\, b]$ | Отрезок | $a \leq x \leq b$ | С обоими концами |
| $[a;\, b)$ | Полуинтервал | $a \leq x < b$ | С левым концом |
| $(a;\, b]$ | Полуинтервал | $a < x \leq b$ | С правым концом |
| $(a;\, +\infty)$ | Луч | $x > a$ | Без конца вправо |
| $[a;\, +\infty)$ | Луч | $x \geq a$ | С точкой $a$ |
| $(-\infty;\, b)$ | Луч | $x < b$ | Без конца влево |
| $(-\infty;\, b]$ | Луч | $x \leq b$ | С точкой $b$ |

> **Как запомнить:** Круглая скобка — конец **не включён** (рисуем пустой кружок), квадратная — **включён** (закрашенный кружок).

## 4. Системы линейных неравенств

**Система неравенств** — задача найти все значения переменной, удовлетворяющие каждому из неравенств одновременно.

$$\begin{cases} f_1(x) > 0 \\ f_2(x) < 0 \end{cases}$$

**Решение системы** — пересечение множеств решений каждого неравенства.

**Алгоритм:**
1. Решить каждое неравенство отдельно.
2. Изобразить решения на числовой прямой.
3. Найти пересечение (общую часть).

**Пример:** $\begin{cases} 2x - 5 > 1 \\ 3x + 1 \leq 13 \end{cases}$

- Из первого: $2x > 6 \Rightarrow x > 3$, т.е. $x \in (3;\, +\infty)$
- Из второго: $3x \leq 12 \Rightarrow x \leq 4$, т.е. $x \in (-\infty;\, 4]$
- Пересечение: $x \in (3;\, 4]$

**Двойное неравенство** $a < f(x) < b$ — это система:
$$\begin{cases} f(x) > a \\ f(x) < b \end{cases}$$

Решается одновременным выполнением преобразований над всеми тремя частями.

**Пример:** $-3 < 2x + 1 \leq 7$

Вычитаем 1 из всех частей: $-4 < 2x \leq 6$

Делим на 2: $-2 < x \leq 3$

**Ответ:** $x \in (-2;\, 3]$

## 5. Совокупность неравенств

**Совокупность неравенств** — задача найти все значения переменной, удовлетворяющие хотя бы одному из неравенств.

Решение совокупности — **объединение** множеств решений:

$$\left[\begin{array}{l} f_1(x) > 0 \\ f_2(x) < 0 \end{array}\right. \quad \Rightarrow \quad \text{решение} = A_1 \cup A_2$$

**Пример:** $\left[\begin{array}{l} x < -1 \\ x > 4 \end{array}\right.$

**Ответ:** $x \in (-\infty;\, -1) \cup (4;\, +\infty)$

## 6. Степень с целым показателем

### Определение

Для любого $a \neq 0$ и натурального $n$:

$$a^{-n} = \frac{1}{a^n}$$

$$a^0 = 1$$

**Примеры:**
$$2^{-3} = \frac{1}{2^3} = \frac{1}{8}, \quad \left(\frac{1}{3}\right)^{-2} = 3^2 = 9, \quad 5^0 = 1$$

### Свойства степени с целым показателем

Для любых целых $m$, $n$ и $a \neq 0$, $b \neq 0$:

$$a^m \cdot a^n = a^{m+n}$$

$$a^m : a^n = a^{m-n}$$

$$(a^m)^n = a^{mn}$$

$$(ab)^n = a^n \cdot b^n$$

$$\left(\frac{a}{b}\right)^n = \frac{a^n}{b^n}$$

> **Как запомнить:** Все свойства степени с натуральным показателем сохраняются для целого показателя. Новое только одно: $a^{-n} = \dfrac{1}{a^n}$ — «минус в показателе — это дробь».

**Пример упрощения:**
$$\frac{a^{-3} \cdot b^2}{a^2 \cdot b^{-5}} = a^{-3-2} \cdot b^{2-(-5)} = a^{-5} \cdot b^7 = \frac{b^7}{a^5}$$

## 7. Стандартный вид числа

Любое число можно записать в виде:

$$a \times 10^n, \quad \text{где } 1 \leq a < 10, \quad n \in \mathbb{Z}$$

Такая запись называется **стандартным видом** числа.

| Число | Стандартный вид |
|-------|-----------------|
| $5\,300\,000$ | $5{,}3 \times 10^6$ |
| $0{,}00042$ | $4{,}2 \times 10^{-4}$ |
| $210\,000\,000$ | $2{,}1 \times 10^8$ |
| $0{,}00016$ | $1{,}6 \times 10^{-4}$ |

> **Правило:** Считай, на сколько позиций сдвигается запятая. Если число больше 10 — показатель положительный, если меньше 1 — отрицательный.

## Краткая шпаргалка

| Тема | Ключевое правило |
|------|-----------------|
| Умножение на $c > 0$ | Знак неравенства сохраняется |
| Умножение на $c < 0$ | Знак неравенства **меняется** |
| Сложение неравенств | Знаки одинаковы: складывать можно |
| Система неравенств | Пересечение решений ($\cap$) |
| Совокупность неравенств | Объединение решений ($\cup$) |
| Двойное неравенство | Преобразовывать все три части одновременно |
| $a^{-n}$ | $= \dfrac{1}{a^n}$, при $a \neq 0$ |
| $a^0$ | $= 1$, при $a \neq 0$ |
| Стандартный вид | $a \times 10^n$, где $1 \leq a < 10$ |
