## Как пользоваться сборником

1. **Прочитай теорию** — файл `00-Теория.md` содержит все определения, свойства и формулы.
2. **Разбери примеры** — в файле `01-Примеры.md` показаны типичные задачи с подробными решениями.
3. **Начни с группы А** — убедись, что умеешь решать линейные неравенства и системы.
4. **Переходи к группе Б** — здесь нужно уверенно работать с параметрами, двойными неравенствами и степенями с целым показателем.
5. **Группа В** — для тех, кто готовится к олимпиадам: доказательства неравенств методом AM-GM и нестандартные системы.
