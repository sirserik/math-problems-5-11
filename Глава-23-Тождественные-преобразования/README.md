## Порядок изучения

1. **Начните с теории** — убедитесь, что помните все формулы наизусть
2. **Разберите примеры** — не просто читайте, а пробуйте решить самостоятельно, затем сверяйтесь
3. **Группа А** — решите минимум 20–30 задач, добейтесь уверенности
4. **Группа Б** — переходите только после освоения группы А
5. **Группа В** — для олимпиадной подготовки и углублённого изучения
