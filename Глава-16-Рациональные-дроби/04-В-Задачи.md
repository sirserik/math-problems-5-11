## Блок 2. Многоэтажные дроби и сложные выражения

**16.113.** Упростите выражение:
$$\left(\frac{x}{x^2+2x+4} + \frac{x^2+8}{x^3-8} - \frac{1}{x-2}\right) \cdot \left(\frac{x^2-4}{2-x} - \frac{2}{x-2}\right)$$

**16.114.** Упростите выражение:
$$\left(\frac{1}{2-a} + \frac{6a-4-a^2}{a^3-8} - \frac{2-a}{a^2+2a+4}\right) \cdot \frac{a^3+4a^2+8a+8}{4-4a+a^2}$$

**16.115.** Упростите выражение:
$$\left(\frac{x}{x^2+2x+4} + \frac{x^2+8}{x^3-8} - \frac{1}{x-2}\right) : \left(\frac{x^2-4}{x^3-8} + \frac{x}{x^2-4x+4}\right)$$

**16.116.** Вычислите значение выражения при $a = 1{,}2$, $b = 0{,}5$, $c = 1{,}3$:
$$\frac{\dfrac{1}{a} - \dfrac{1}{b-c}}{\dfrac{1}{a+b-c}} \cdot \left(1 - \frac{b^2+c^2-a^2}{2bc}\right) : \frac{b-a-c}{abc}$$

**16.117.** Упростите выражение:
$$\left(\frac{x}{x^2+2x+4} + \frac{x^2+8}{x^3-8} - \frac{1}{x-2}\right)\left(\frac{x^2-4}{2-x} - \frac{2}{x-2}\right)$$

**16.118.** Докажите, что при любых допустимых значениях переменных значение выражения
$$(b^2+b+ab+a)(2a-b-b^2+2ab)\left(\frac{a-b}{2a-b} - \frac{a^2+b^2+a+5}{2a^2+ab-b^2}\right) : (2b^2+a+5)$$
неположительно и не зависит от переменной $a$.

## Блок 3. Доказательства

**16.119.** Докажите, что при всех допустимых значениях переменных значение выражения
$$\frac{2}{(a+b)^3}\left(\frac{1}{a} + \frac{1}{b}\right) + \frac{1}{a^2+b^2+2ab}\left(\frac{1}{a^2} + \frac{1}{b^2}\right)$$
положительно.

**16.120.** Докажите, что при всех допустимых значениях переменных значение выражения
$$\left(\frac{a^2}{2b^3} + \frac{a}{b}\right) : \left(2b^3 - \frac{b}{2} + \frac{a}{2}\right) : \frac{(a-2b)^2+8ab}{4+\dfrac{2a}{b}}$$
положительно.

**16.121.** Докажите, что при любом значении $a > 1$ значение выражения
$$\frac{1}{1-a} + \frac{1}{1+a} + \frac{2}{1+a^2} + \frac{4}{1+a^4} + \frac{8}{1+a^8} + \frac{16}{1+a^{16}} + \frac{32}{1+a^{32}}$$
отрицательно.

**16.122.** Докажите, что значение выражения
$$\frac{4}{1-a}\left(\frac{2a+2}{3-a}\right)^2 \cdot \left(\frac{a+9}{a^2+2a+1} + \frac{2a}{1-a}\right)$$
не зависит от значения переменной при всех допустимых значениях.

**16.123.** Докажите, что при любых допустимых значениях переменной $n$:
$$\left(\frac{2}{n-4m^2} + \frac{6n}{n^2-4m^2} - \frac{4}{n^2-4m^2}\right) : \left(1 + \frac{4m^2+n^2}{4m^2-n^2}\right)$$
не зависит от $n$.
