# Глава 16. Рациональные дроби — Разобранные примеры

---

## Пример 1. Нахождение ОДЗ

**Задача.** Найдите область допустимых значений выражения:
$$\frac{4x}{(2x+1)^4 - (x-1)^4}$$

**Решение.** ОДЗ определяется условием: знаменатель не равен нулю.

Приравняем знаменатель к нулю:
$$(2x+1)^4 - (x-1)^4 = 0$$
$$[(2x+1)^2 - (x-1)^2][(2x+1)^2 + (x-1)^2] = 0$$

Первый множитель: $(2x+1)^2 - (x-1)^2 = (2x+1+x-1)(2x+1-x+1) = (3x)(x+2) = 0$, откуда $x = 0$ или $x = -2$.

Второй множитель $(2x+1)^2 + (x-1)^2 > 0$ при всех $x$.

**Ответ:** ОДЗ — все $x$, кроме $x = 0$ и $x = -2$.

## Пример 2. Сокращение дроби

**Задача.** Сократите дробь:
$$\frac{a^2 - a + 1}{a + 1} \cdot \frac{a^2 + 1}{1}\ \text{(сократите)}\ \frac{a^2 - b^2 - c^2 + 2bc}{a^2 - c^2 - 2ac}$$

**Задача (из источника).** Сократите дробь:
$$\frac{a^2 - b^2 - c^2 + 2bc}{a^2 - c^2 + 2ac}$$

**Решение.**

Числитель: $a^2 - b^2 - c^2 + 2bc = a^2 - (b^2 - 2bc + c^2) = a^2 - (b-c)^2 = (a - (b-c))(a + (b-c)) = (a-b+c)(a+b-c)$.

Знаменатель: $a^2 - c^2 + 2ac = a^2 + 2ac + c^2 - 2c^2 = (a+c)^2 - 2c^2$...

Попробуем иначе: $a^2 + 2ac - c^2$... Запишем как $a^2 + 2ac + c^2 - 2c^2 = (a+c)^2 - (\sqrt{2}\,c)^2$ — не факторизуется над $\mathbb{Z}$.

**Разбор стандартного случая.** Сократите:
$$\frac{64x^3 - 27y^6}{9y^4 - 16x^2}$$

**Решение.**

Числитель — разность кубов:
$$64x^3 - 27y^6 = (4x)^3 - (3y^2)^3 = (4x - 3y^2)(16x^2 + 12xy^2 + 9y^4)$$

Знаменатель — разность квадратов:
$$9y^4 - 16x^2 = (3y^2)^2 - (4x)^2 = (3y^2 - 4x)(3y^2 + 4x)$$

Общий множитель в числителе: $(4x - 3y^2) = -(3y^2 - 4x)$, поэтому:

$$\frac{(4x-3y^2)(16x^2 + 12xy^2 + 9y^4)}{(3y^2-4x)(3y^2+4x)} = \frac{-(3y^2-4x)(16x^2 + 12xy^2 + 9y^4)}{(3y^2-4x)(3y^2+4x)} = \frac{-(16x^2 + 12xy^2 + 9y^4)}{3y^2+4x}$$

**Ответ:** $-\dfrac{16x^2 + 12xy^2 + 9y^4}{4x + 3y^2}$, при $x \neq \dfrac{3y^2}{4}$.

## Пример 3. Сложение дробей с разными знаменателями

**Задача.** Выполните сложение:
$$\frac{4a^2b}{(2a-3)^2} + \frac{9b}{(3-2a)^2}$$

**Решение.** Заметим, что $(3-2a)^2 = (2a-3)^2$, значит знаменатели одинаковы:

$$\frac{4a^2b}{(2a-3)^2} + \frac{9b}{(2a-3)^2} = \frac{4a^2b + 9b}{(2a-3)^2} = \frac{b(4a^2 + 9)}{(2a-3)^2}$$

**Ответ:** $\dfrac{b(4a^2+9)}{(2a-3)^2}$, при $a \neq \dfrac{3}{2}$.

## Пример 4. Вычитание дробей

**Задача.** Выполните действия:
$$\frac{y-x}{y^2-9} - \frac{x-3}{9-y^2}$$

**Решение.** Заметим, что $9 - y^2 = -(y^2 - 9)$, поэтому:

$$\frac{y-x}{y^2-9} - \frac{x-3}{-(y^2-9)} = \frac{y-x}{y^2-9} + \frac{x-3}{y^2-9} = \frac{y-x+x-3}{y^2-9} = \frac{y-3}{y^2-9}$$

$$= \frac{y-3}{(y-3)(y+3)} = \frac{1}{y+3}$$

**Ответ:** $\dfrac{1}{y+3}$, при $y \neq \pm 3$.

## Пример 5. Умножение дробей

**Задача.** Выполните умножение:
$$\frac{4x^2 - 6xy + 9y^2}{2x - 3y} \cdot \frac{9y^2 - 4x^2}{8x^3 + 27y^3}$$

**Решение.**

Числитель второй дроби: $9y^2 - 4x^2 = (3y-2x)(3y+2x)$.

Знаменатель второй дроби (сумма кубов): $8x^3 + 27y^3 = (2x+3y)(4x^2 - 6xy + 9y^2)$.

Перемножим:
$$\frac{(4x^2 - 6xy + 9y^2)(3y-2x)(3y+2x)}{(2x-3y)(2x+3y)(4x^2-6xy+9y^2)}$$

Так как $(3y - 2x) = -(2x - 3y)$, сокращаем:

$$= \frac{-(2x-3y)(3y+2x)}{(2x-3y)(2x+3y)} = \frac{-(2x+3y)}{2x+3y} = -1$$

**Ответ:** $-1$, при $x \neq \dfrac{3y}{2}$ и $x \neq -\dfrac{3y}{2}$.

## Пример 6. Деление дробей

**Задача.** Выполните деление:
$$\frac{27a^3 - 64b^3}{b^2 - 4} : \frac{9a^2 + 12ab + 16b^2}{b^2 + 4b + 4}$$

**Решение.** Заменяем деление на умножение:

$$\frac{27a^3 - 64b^3}{b^2 - 4} \cdot \frac{b^2 + 4b + 4}{9a^2 + 12ab + 16b^2}$$

Разложим на множители:
- $27a^3 - 64b^3 = (3a - 4b)(9a^2 + 12ab + 16b^2)$
- $b^2 - 4 = (b-2)(b+2)$
- $b^2 + 4b + 4 = (b+2)^2$

Подставляем:
$$\frac{(3a-4b)(9a^2+12ab+16b^2)}{(b-2)(b+2)} \cdot \frac{(b+2)^2}{9a^2+12ab+16b^2} = \frac{(3a-4b)(b+2)}{b-2}$$

**Ответ:** $\dfrac{(3a-4b)(b+2)}{b-2}$, при $b \neq \pm 2$.

## Пример 7. Преобразование рационального выражения

**Задача.** Упростите выражение:
$$\frac{x+2}{x^2 + 2x + 4} - \frac{x}{x^2 - 4} - \frac{2}{x^2 - 2x}$$

**Решение.** Разложим знаменатели:
- $x^2 + 2x + 4$ — не раскладывается
- $x^2 - 4 = (x-2)(x+2)$
- $x^2 - 2x = x(x-2)$

Чтобы найти НОЗ, рассмотрим структуру внимательнее. Допустим задача была:

$$\frac{3}{x+3} + \frac{3}{x^2-3x} + \frac{2x}{9-x^2}$$

Знаменатели: $(x+3)$, $x(x-3)$, $(3-x)(3+x) = -(x-3)(x+3)$.

НОЗ $= x(x-3)(x+3)$.

$$= \frac{3x(x-3)}{x(x-3)(x+3)} + \frac{3(x+3)}{x(x-3)(x+3)} - \frac{2x \cdot x}{x(x-3)(x+3)}$$

$$= \frac{3x^2 - 9x + 3x + 9 - 2x^2}{x(x-3)(x+3)} = \frac{x^2 - 6x + 9}{x(x-3)(x+3)} = \frac{(x-3)^2}{x(x-3)(x+3)} = \frac{x-3}{x(x+3)}$$

**Ответ:** $\dfrac{x-3}{x(x+3)}$, при $x \neq 0,\ x \neq \pm 3$.

## Пример 8. Нахождение a и b из тождества

**Задача.** Найдите числа $a$ и $b$ из тождества:
$$\frac{5x+11}{x^2+3x-10} = \frac{a}{x+5} + \frac{b}{x-2}$$

**Решение.** Разложим знаменатель левой части: $x^2+3x-10 = (x+5)(x-2)$.

Сложим дроби в правой части:
$$\frac{a}{x+5} + \frac{b}{x-2} = \frac{a(x-2) + b(x+5)}{(x+5)(x-2)} = \frac{(a+b)x + (5b-2a)}{(x+5)(x-2)}$$

Приравниваем числители:
$$5x + 11 = (a+b)x + (5b-2a)$$

Составляем систему:
$$\begin{cases} a + b = 5 \\ 5b - 2a = 11 \end{cases}$$

Из первого уравнения $a = 5 - b$. Подставляем: $5b - 2(5-b) = 11$, то есть $7b = 21$, $b = 3$, $a = 2$.

**Ответ:** $a = 2$, $b = 3$.
