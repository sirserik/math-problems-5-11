## Темы главы

1. Рациональные выражения. Область допустимых значений (ОДЗ)
2. Основное свойство дроби. Сокращение дробей
3. Сложение и вычитание дробей
4. Умножение и деление дробей. Возведение в степень
5. Преобразование рациональных выражений
6. Разложение на множители. Деление многочленов

## Источники

## Распределение задач по уровням

- **Группа А (базовый):** нахождение ОДЗ, вычисление значений дробей, простое сокращение, сложение/вычитание с одинаковыми знаменателями
- **Группа Б (средний):** сокращение сложных дробей, действия с разными знаменателями, умножение и деление дробей, представление в виде дроби
- **Группа В (повышенный):** упрощение многоэтажных дробей, доказательства, пропорции, задачи на нахождение целочисленных значений, деление многочленов
