**22.100.** Описанный четырёхугольник: $AB + CD = BC + AD$. $AB : CD = 2 : 3$, $AD : BC = 2 : 1$. Пусть $AB = 2a$, $CD = 3a$, $AD = 2b$, $BC = b$. $2a + 3a = b + 2b$, $5a = 3b$, $b = \dfrac{5a}{3}$. $S = r(AB+BC+CD+AD)/2 = r \cdot (5a + 3b)/2 = r \cdot 5a$... Нет: $S = r \cdot p$, где $p$ — полупериметр. $p = \dfrac{5a+3b}{2} = \dfrac{5a+5a}{2} = 5a$. $S = 5ar$. Также $S$ задана. Стороны: $AB = 2a$, $CD = 3a$, $BC = \dfrac{5a}{3}$, $AD = \dfrac{10a}{3}$. $S = 5ar$ $\Rightarrow a = \dfrac{S}{5r}$. Стороны выражаются через $S$ и $r$.
