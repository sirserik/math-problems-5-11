**15.006.** Решите уравнение $(2 + 3x) - (4x - 7) = 10$.

**15.009.** Решите уравнение $3x = -7$.

**15.010.** Решите уравнение $3x - 8 = 5 - 7x$.

**15.011.** Решите уравнение $\dfrac{8x}{3} = \dfrac{1}{3} - \dfrac{5x + 2}{11}$.

**15.012.** Решите уравнение $\dfrac{3x + 11}{5} = \dfrac{5x - 7}{7}$.

**15.013.** Решите уравнение $\dfrac{3x + 1}{3} = \dfrac{5x - 1}{7}$.

**15.014.** Решите уравнение $\dfrac{7x - 2}{111} = \dfrac{8x + 5}{37}$.

**15.015.** Решите уравнение $\dfrac{3 + 7x}{8} = \dfrac{8 + 8x}{8}$.

**15.017.** Решите уравнение $\dfrac{x + 3}{6} + \dfrac{x - 7}{6} = \dfrac{x + 11}{2}$.

**15.018.** Решите уравнение $(3x - 1)^2 + (4x + 5)^2 = (5x - 7)^2$.

**15.019.** Решите уравнение $(2x + 7)(3x - 1) - (5x - 1)(x + 3) = (x + 1)^2$.

**15.020.** Решите уравнение $(x - 2)^3 + (x + 2)^2 = 2(x - 3)(x^2 + 3x + 9)$.

## Раздел 3. Функция и её значения (15.031 — 15.040)

**15.031.** Функция задана формулой $y = 2x - 3$.

а) Найдите $y$ при $x = 4$;
б) найдите $x$, при котором $y = 1$;
в) принадлежит ли графику функции точка $A(-1;\, -5)$?

**15.032.** Функция задана формулой $y = 3x + 2$.

а) Найдите $y$ при $x = -1$;
б) найдите $x$, при котором $y = 8$;
в) принадлежит ли графику точка $B(2;\, 0)$?

**15.033.** Функция задана формулой $y = -x + 3$.

а) Найдите значение функции при $x = -2$;
б) найдите значение аргумента, при котором значение функции равно $4$;
в) какие из точек $A(0;\, 3)$, $B(2{,}5;\, -0{,}5)$, $C\!\left(5\tfrac{1}{3};\, 2\tfrac{1}{3}\right)$, $D(-4;\, 7)$ принадлежат графику функции?

**15.034.** Функция задана формулой $y = 4 - 3x$.

а) Найдите значение функции при $x = -4$;
б) найдите значение аргумента, при котором значение функции равно $1$;
в) какие из точек $A(0;\, 4)$, $B(-1{,}5;\, 8{,}5)$, $C\!\left(\tfrac{1}{3};\, 1\right)$, $D(-2;\, 2)$ принадлежат графику?

**15.035.** Функция задана формулой $y = \dfrac{1}{3}x - 6$.

а) Найдите значение функции при $x = 6$;
б) найдите значение аргумента, при котором значение функции равно $\dfrac{2}{3}$;
в) какая точка графика этой функции имеет абсциссу, равную ординате?

**15.036.** Функция задана формулой $y = \dfrac{1}{2}x + 4$.

а) Найдите значение функции при $x = -8$;
б) найдите значение аргумента, при котором значение функции равно $-0{,}5$;
в) какая точка графика этой функции имеет абсциссу, равную ординате?

**15.037.** Одна из сторон прямоугольника равна $x$ см, другая — $5$ см. Запишите формулой зависимость площади прямоугольника $S$ от $x$.

**15.038.** Расстояние между пунктами $A$ и $B$ равно $60$ км. Из пункта $A$ в пункт $B$ выехал велосипедист. Первые $t$ часов он ехал со скоростью $12$ км/ч. Выразите зависимость оставшегося пути $s$ от $t$.

**15.039.** Выразите из формулы $\rho = \dfrac{m}{V}$ переменную $m$. Найдите $m$, если $\rho = 7{,}8$ г/см$^3$ и $V = 5$ см$^3$.

**15.040.** Выразите из формулы $s = s_0 + vt$ переменную $v$. Найдите $v$, если $s = 100$, $s_0 = 10$, $t = 5$.
