## Типы задач

- **Нахождение членов и сумм** — по заданным условиям (сумма, произведение, разность)
- **Текстовые задачи** — задачи на применение прогрессий в реальных ситуациях
- **Бесконечные ГП** — суммы, уравнения с бесконечными рядами
- **Комбинированные** — задачи, связывающие АП и ГП
- **Доказательства** — свойства прогрессий, делимость, неравенства
- **Нестандартные** — задачи с цифрами чисел, геометрические приложения, суммирование рядов
