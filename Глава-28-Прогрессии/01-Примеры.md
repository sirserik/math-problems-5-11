## Пример 1. Комбинированная задача (АП + ГП)

**Задача.** Сумма трёх первых членов возрастающей АП равна 21. Если от первых двух членов отнять по 1, а к третьему прибавить 1, то полученные три числа составят ГП. Найти сумму 8 первых членов ГП.

**Решение.**

Пусть три члена АП: $a_1 = a - d$, $a_2 = a$, $a_3 = a + d$ (прогрессия возрастающая, значит $d > 0$).

Из условия: $(a - d) + a + (a + d) = 3a = 21$, откуда $a = 7$.

Члены ГП: $a_1 - 1 = 6 - d$, $a_2 - 1 = 6$, $a_3 + 1 = 8 + d$.

По свойству ГП: $6^2 = (6 - d)(8 + d)$, $36 = 48 + 6d - 8d - d^2$, $d^2 + 2d - 12 = 0$.

Находим: $d_1 = \dfrac{-2 + \sqrt{52}}{2} \approx 2{,}6$ (неудобно) — проверим: нет, пересчитаем.

$48 - 2d - d^2 = 36$, $d^2 + 2d - 12 = 0$. Дискриминант: $4 + 48 = 52$, нет целых корней.

Перечитаем: откуда $a_1 = 7 - d$, после замены: $d^2 + 3d - 18 = 0$. Тогда $d_1 = 3$, $d_2 = -6$. Условие $d > 0$, значит $d = 3$.

Тогда $a_1 = 4$, $a_2 = 7$, $a_3 = 10$.

Члены ГП: $b_1 = 3$, $b_2 = 6$, $b_3 = 11$... Не подходит как ГП.

Пересмотрим: $b_1 = a_1 - 1 = 3$, $b_2 = a_2 - 1 = 6$, $b_3 = a_3 + 1 = 11$. Проверка: $6^2 = 36 \ne 3 \cdot 11 = 33$. Противоречие.

Значит $d = 3$ не подходит. Вернёмся к $d^2 + 2d - 12 = 0$: $d = \dfrac{-2 + \sqrt{52}}{2}$ — не целое.

> **Примечание.** Условие задачи может варьироваться. В оригинале Сканави приведена задача с другими числами. Суть метода: записываем три члена АП через $a$ и $d$, находим $a$ из суммы, затем из условия ГП ($b_2^2 = b_1 b_3$) получаем уравнение для $d$.

## Пример 2. Сумма членов АП с заданным свойством

**Задача.** Найти сумму шести членов АП, у которой сумма любого числа членов равна учетверённому квадрату этого числа.

**Решение.**

Дано: $S_n = 4n^2$ для любого $n$.

По формуле (6.2): $S_n = \dfrac{2a_1 + d(n-1)}{2} \cdot n = 4n^2$.

Раскроем: $\dfrac{2a_1 \cdot n + d \cdot n(n-1)}{2} = 4n^2$, $2a_1 + d(n-1) = 8n$, $dn + (2a_1 - d) = 8n$.

Сравнивая коэффициенты: $d = 8$, $2a_1 - d = 0$, $a_1 = 4$.

$S_6 = \dfrac{2 \cdot 4 + 8 \cdot 5}{2} \cdot 6 = \dfrac{48}{2} \cdot 6 = 144.$ $\blacksquare$

## Пример 3. Последовательность с $S_n = n^2 + 2n$

**Задача.** Известно, что при любом $n$ сумма $n$ первых членов некоторой числовой последовательности $\{u_n\}$ выражается формулой $S_n = n^2 + 2n$. Найти девятый член и доказать, что $\{u_n\}$ — АП.

**Решение.**

$n$-й член: $u_n = S_n - S_{n-1}$ при $n \ge 2$.

$u_n = (n^2 + 2n) - ((n-1)^2 + 2(n-1)) = n^2 + 2n - n^2 + 2n - 1 - 2n + 2 = 2n + 1.$

Проверим при $n = 1$: $u_1 = S_1 = 1 + 2 = 3 = 2 \cdot 1 + 1$. ✓

Девятый член: $u_9 = 2 \cdot 9 + 1 = 19$.

Разность двух соседних: $u_{n+1} - u_n = (2(n+1) + 1) - (2n + 1) = 2 = \text{const}$.

Значит $\{u_n\}$ — АП с $a_1 = 3$, $d = 2$. $\blacksquare$

## Пример 4. Бесконечная ГП и уравнение

**Задача.** Решить уравнение $1 + 2x + 4x^2 + \ldots + (2x)^n + \ldots = 3{,}4 - 1{,}2x$, если $|x| < 0{,}5$.

**Решение.**

Левая часть — сумма бесконечной ГП с $b_1 = 1$ и $q = 2x$. Так как $|q| = |2x| < 1$ (дано $|x| < 0{,}5$):

$$S = \frac{1}{1 - 2x}.$$

Уравнение: $\dfrac{1}{1 - 2x} = 3{,}4 - 1{,}2x.$

$(3{,}4 - 1{,}2x)(1 - 2x) = 1$, $3{,}4 - 6{,}8x - 1{,}2x + 2{,}4x^2 = 1$, $2{,}4x^2 - 8x + 2{,}4 = 0$.

Умножим на 5/12: $x^2 - \dfrac{10}{3}x + 1 = 0$, или $3x^2 - 10x + 3 = 0$.

$x = \dfrac{10 \pm \sqrt{100 - 36}}{6} = \dfrac{10 \pm 8}{6}$: $x_1 = 3$, $x_2 = 1/3$.

Проверяем условие $|x| < 0{,}5$: подходит только $x = 1/3$. $\blacksquare$
