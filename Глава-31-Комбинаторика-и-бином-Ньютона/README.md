## Типы задач

- **Уравнения с $C_n^m$, $A_n^m$, $P_n$** — применение формул, решение относительно $n$ и $m$
- **Тождества** — доказательства свойств сочетаний и перестановок
- **Бином Ньютона** — нахождение отдельных членов, коэффициентов, наибольшего члена
- **Комбинаторные задачи** — правило умножения/сложения, подсчёт вариантов
- **Задачи на размещение** — расстановка предметов с ограничениями
- **Доказательства** — неравенства с биномиальными коэффициентами
