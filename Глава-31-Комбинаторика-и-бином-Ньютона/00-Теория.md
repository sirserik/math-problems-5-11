# Теория. Комбинаторика и бином Ньютона

> **Класс:** 9–11 | Перестановки, сочетания, размещения, формула бинома

---

## 1. Перестановки

**Перестановка** из $n$ элементов — это упорядоченный набор всех $n$ элементов.

$$P_n = n! = 1 \cdot 2 \cdot 3 \cdots n$$

| $n$ | $P_n = n!$ |
|-----|-----------|
| 1 | 1 |
| 2 | 2 |
| 3 | 6 |
| 4 | 24 |
| 5 | 120 |
| 6 | 720 |
| 7 | 5040 |
| 8 | 40320 |
| 9 | 362880 |
| 10 | 3628800 |

> **Как запомнить:** $n!$ — это количество способов расставить $n$ различных предметов в ряд.

По определению: $0! = 1$.

---

## 2. Сочетания

**Сочетание** из $n$ элементов по $m$ — это неупорядоченный выбор $m$ элементов из $n$.

$$C_n^m = \binom{n}{m} = \frac{n!}{m!\,(n-m)!}$$

Частные случаи: $C_n^0 = 1$, $\; C_n^n = 1$.

### Свойства сочетаний

| Свойство | Формула |
|----------|---------|
| Симметрия | $C_n^m = C_n^{n-m}$ |
| Сложение (треугольник Паскаля) | $C_n^m + C_n^{m+1} = C_{n+1}^{m+1}$ |
| Сумма всех | $C_n^0 + C_n^1 + \ldots + C_n^n = 2^n$ |

> **Треугольник Паскаля** — наглядный способ вычислять $C_n^m$: каждый элемент равен сумме двух стоящих над ним.

---

## 3. Размещения

**Размещение** из $n$ элементов по $m$ — это упорядоченный выбор $m$ элементов из $n$.

$$A_n^m = P_m \cdot C_n^m = \frac{n!}{(n-m)!}$$

> **Связь:** $A_n^m = m! \cdot C_n^m$. Размещение — это сочетание, в котором важен порядок.

---

## 4. Формула бинома Ньютона

$$(a + b)^n = \sum_{k=0}^{n} C_n^k\, a^{n-k}\, b^k = C_n^0 a^n + C_n^1 a^{n-1}b + C_n^2 a^{n-2}b^2 + \ldots + C_n^n b^n$$

### $(k+1)$-й член разложения (общий член)

$$T_{k+1} = C_n^k\, a^{n-k}\, b^k \qquad (k = 0, 1, 2, \ldots, n)$$

### Ключевые следствия

| Подстановка | Результат |
|-------------|-----------|
| $a = b = 1$ | $C_n^0 + C_n^1 + \ldots + C_n^n = 2^n$ |
| $a = 1,\; b = -1$ | $C_n^0 - C_n^1 + C_n^2 - \ldots = 0$ |

Отсюда:
- **Сумма коэффициентов** при чётных степенях $= 2^{n-1}$
- **Сумма коэффициентов** при нечётных степенях $= 2^{n-1}$

---

## 5. Свойства биномиальных коэффициентов

1. $C_n^k = C_n^{n-k}$ — симметрия.
2. Наибольший коэффициент:
   - При чётном $n$: $C_n^{n/2}$
   - При нечётном $n$: $C_n^{(n-1)/2} = C_n^{(n+1)/2}$
3. Сумма биномиальных коэффициентов $(a + b)^n$ равна $2^n$.

---

## 6. Основные комбинаторные правила

### Правило умножения
Если первое действие можно выполнить $m$ способами, а второе — $n$ способами, то оба действия вместе можно выполнить $m \cdot n$ способами.

### Правило сложения
Если одно действие можно выполнить $m$ способами, а другое (несовместное с первым) — $n$ способами, то выполнить одно из этих действий можно $m + n$ способами.

---

## Типичные ошибки

| Ошибка | Правильно |
|--------|-----------|
| Путать $A_n^m$ и $C_n^m$ | $A$ — важен порядок, $C$ — не важен |
| Забыть, что $0! = 1$ | $0! = 1$ по определению |
| Неправильно определять номер члена в биноме | $T_{k+1}$ — это $(k+1)$-й член, $k$ считается от 0 |
| Не учитывать ограничения ($m \leqslant n$, $m \geqslant 0$) | Проверять область определения $C_n^m$ |
